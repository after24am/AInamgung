# 21화. 카운트다운

---

남궁혁 방문까지 20일.

창궁무애검법 제1초식의 완성도는 58%였다. 리팩토링의 효과가 컸다. 곡선 전환점에 기를 집중하는 방식을 태을이 정밀하게 다듬어주니, 몸이 따라오기 시작했다.

하지만 58%는 부족했다. 남궁혁 앞에서 보여주려면 최소 70%. 누가 봐도 "검법"이라고 인정할 수 있는 수준.

남궁선은 낮과 밤의 수련을 바꿨다.

낮 연무장에서는 기초검식만. 여전히 중간 자리. 눈에 띄지 않게. 하지만 밤에는 창궁무애검법에 모든 시간을 쏟았다.

```
[수련 적립: 일일 평균 +0.35토큰.]
[내공 수치: 0.7. 검 증폭 포함 체감 내공: 1.1.]
[내공 1.0을 넘었습니다. 유의미한 기준선입니다.]
```

내공 1.0. 태을이 말한 "일반 하수는 이기고, 문파 제자에게는 쳐맞는" 수준에서 한 단계 올라선 것이다. 검 증폭까지 합치면 — 문파 하급 제자와 겨룰 수 있다.

형과의 대련에서도 변화가 왔다. 42초. 그리고 — 처음으로 형의 빈틈에 목검을 갖다 댔다.

맞추지는 못했다. 남궁재가 반사적으로 쳐냈다. 하지만 형의 얼굴에 처음 보는 표정이 떠올랐다.

놀람이 아니었다. 경계였다.

"...한 번 더."

형이 먼저 대련을 요청한 건 그날이 처음이었다.

---

남궁혁 방문까지 15일.

연무장에서 사건이 터졌다.

본가 위탁생 다섯 중 맏이인 소년이 — 남궁선에게 대련을 신청했다.

남궁유(南宮裕). 열일곱. 본가 직계의 막내아들이었다. 5화에서 "방계 망나니가 검을 잡으니까 어색하네"라고 말했던 바로 그놈이었다.

"남궁선. 너 요즘 실력이 늘었다며? 한 수 겨루자."

연무장이 조용해졌다. 제자들의 시선이 쏠렸다. 조 사범은 바깥에 나가 있었다. 교관이 없는 틈.

남궁선은 남궁유를 바라보았다.

서른여섯의 눈이 읽었다. 이 소년의 의도를.

실력이 궁금한 게 아니다. 방계 출신 망나니가 실력이 느는 게 — 거슬리는 것이다. 본가 위탁생이 방계 제자한테 밀리면 체면이 선다. 지금 눌러놓고 싶은 거다.

```
[남궁유. 내공 추정: 0.8~1.0. 기초검식 숙련도 상위.]
[플레이어와 정면 대결 시 — 50:50.]
[리팩토링된 검식을 쓰면 유리하지만, 공개 대련에서 쓰면 노출됩니다.]
```

남궁선의 잔머리가 회전했다.

이길 수 있다. 리팩토링 검식을 쓰면. 하지만 남궁혁이 15일 뒤에 온다. 지금 리팩토링 검식을 연무장에서 보여주면 — 소문이 본가까지 올라가고, 남궁혁이 "정기 점검"이 아니라 "조사"를 하러 온다. 상황이 달라진다.

반대로 — 기초검식 정통으로만 싸우면? 50:50. 질 수도 있다.

지면 어떻게 되지?

남궁선은 생각했다. 서른여섯의 계산.

지면 — "역시 망나니"로 끝난다. 위탁생이 방계 제자를 이긴 거니까 뉴스가 안 된다. 소문이 안 퍼진다. 남궁혁의 관심을 끌지 않는다.

이기면 — "망나니가 본가 위탁생을 이겼다"는 소문이 퍼진다. 남궁혁이 더 날카로운 눈으로 본다.

```
[플레이어의 심박수가 상승했습니다.]
[계산 중이시죠?]
```

'...응.'

```
[공유해주시겠습니까?]
```

'지는 게 이득이야.'

```
[......처음 듣는 전략입니다.]
```

남궁선은 목검을 들었다. "좋습니다."

남궁유가 웃었다. 자신 있는 웃음이었다.

두 사람이 마주 섰다. 제자들이 둘러섰다.

남궁유가 먼저 움직였다. 기초검식 제3초식. 빠르고 정확했다. 본가에서 배운 정통. 교과서 그 자체.

남궁선은 정통 기초검식으로 받았다. 리팩토링 없이. 날것 그대로의, 예비 동작이 포함된 원래 검식.

느렸다. 리팩토링에 익숙해진 몸이 오히려 어색해했다. 불필요한 동작을 넣어야 하는 게 — 일부러 서투르게 하는 것보다 어려웠다.

탁, 탁. 목검이 부딪혔다. 밀렸다. 일부러 밀리는 게 아니라, 정통 검식으로는 진짜 밀렸다. 남궁유의 정통 숙련도가 높았다.

10초. 15초. 남궁선은 버텼다. 체력과 내공은 남궁유와 비슷했으니까.

하지만 20초를 넘기지 못했다. 남궁유의 제5초식이 목검을 쳐올렸고, 남궁선의 목검이 날아갔다.

패배.

남궁유가 목검을 거두며 웃었다. "역시 망나니는 망나니네."

남궁선은 목검을 주우며 고개를 숙였다. "졌습니다."

제자들이 흩어졌다. "그래봤자지." "역시 방계는." 수군거림이 멀어졌다.

```
[대련 결과: 패배. 교전 시간 22초.]
[패배 원인: 리팩토링 미적용에 의한 의도적 약화.]
[실제 전력으로 대련했을 경우 예상 결과: 승리 확률 67%.]
[......일부러 진 겁니까?]
```

'응.'

```
[이유를 들어도 됩니까?]
```

'15일 뒤에 남궁혁이 온다. 그 전에 "망나니가 본가 위탁생을 이겼다"는 소문이 돌면 안 돼. 남궁혁이 주의 깊게 보기 시작하거든.'

```
[그래서 — 소문이 안 나게 졌다.]
```

'이기는 건 나중에 해도 돼. 지금은 눈에 안 띄는 게 이기는 거야.'

```
[......]
[효율 점수 9.1/10.]
```

'점수 매기지 말라니까.'

```
[이번엔 진심이었습니다.]
```

남궁선은 연무장을 나오면서 남궁유의 검을 떠올렸다. 22초 동안 봤다. 정통 기초검식의 완성형. 빈틈이 적었다. 하지만 — 제5초식에서 왼쪽 손목이 열렸다. 리팩토링 검식이라면 거기를 찔렀을 것이다.

```
[남궁유 전투 데이터 수집 완료.]
[다음 대련 시 활용 가능.]
```

남궁선은 웃지 않았다. 대신 걸었다. 서별채로. 밤수련을 하러.

15일. 아직 시간이 있다.
