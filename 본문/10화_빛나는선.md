# 10화. 빛나는 선

---

수련 한 달째.

내공 순환이 자리를 잡아가고 있었다. 처음에는 열 번 호흡에 한 번 찰나의 흐름이 왔지만, 한 달이 지나자 세 번에 한 번은 단전에서 감각이 올라왔다. 수로가 젖기 시작한 것이다.

```
[내공 수치: 0.3. 일반인 기준 상위, 무인 기준 최하위.]
[일반 하수와 겨루면 이기고, 정식 문파 제자에게는 집니다.]
[한 달 전에 비하면 0에서 0.3. 무한대 성장률입니다.]
```

'무한대라는 건 뭔데.'

```
[0에서 시작하면 어떤 수를 곱해도 무한히 커 보인다는 뜻입니다.]
[실제로는 여전히 약합니다.]
```

'칭찬인 줄 알았는데.'

```
[팩트입니다.]
```

연무장에서의 위치는 끝자리에서 뒤에서 셋째로 올라왔다. 조 사범이 한 번 고개를 갸웃거렸다. "자세가 나아졌군." 그 한마디가 전부였지만, 남궁선에게는 충분했다.

형 남궁재와의 밤 대련은 열두 번째를 넘겼다. 최근 교전 시간은 19초. 여전히 졌지만, 형의 제3초식까지 보게 되었다.

```
[남궁재 전투 데이터 축적률: 38%.]
[제5초식까지 수집되면 패턴 분석이 가능합니다.]
```

모든 것이 천천히, 하지만 확실하게 움직이고 있었다.

그날도 밤수련이었다.

---

기초검식을 서른 번 반복한 뒤, 남궁선은 숨을 골랐다. 한 달 전이라면 열 번에 쓰러졌을 것이다. 체력도 검술도 자라고 있었다. 느리지만 — 자라고 있었다.

'태을아.'

```
[네.]
```

'궤적 가이드라는 게 있잖아. 적의 공격을 미리 보여주는 거.'

```
[실시간 궤적 가이드. 유료 기능입니다.]
```

'한 번 해볼 수 있어?'

```
[가능합니다. 단순 시연 모드 비용: 0.1토큰.]
[실전에서는 상대 수준에 따라 1~3토큰이 소모됩니다.]
[시연 모드는 가상의 공격 궤적을 투사하여 체험하는 것입니다.]
[때리는 사람이 없으니 맞을 걱정은 없습니다.]
```

'해.'

```
[결제 완료. 0.1토큰 소모. 잔여: 16.4.]
[궤적 가이드 시연 모드 실행.]
```

남궁선의 시야가 바뀌었다.

달빛 아래의 공터는 그대로였다. 하지만 그 위에 — 빛나는 선이 떠올랐다.

붉은 선 하나가 허공에 그려졌다. 왼쪽 어깨를 향해 비스듬히 내려오는 궤적. 검이 지나갈 길이었다. 가상의 적이 제1초식을 내리치는 경로.

그리고 그 옆에 — 푸른 선.

남궁선의 몸이 어떻게 움직여야 하는지를 보여주는 궤적이었다. 오른발을 반 보 빼고, 허리를 비틀고, 목검으로 붉은 선의 아래를 치면 — 빈틈.

빈틈이 붉은 점으로 빛나고 있었다.

남궁선은 숨이 멎었다.

'이게... 이게 보이는 거야?'

```
[네. 이것이 궤적 가이드입니다.]
[적의 공격 경로(붉은 선), 최적 대응 경로(푸른 선), 빈틈(붉은 점).]
[실전에서는 이것이 실시간으로 업데이트됩니다.]
```

남궁선은 푸른 선을 따라 움직였다. 오른발을 빼고, 허리를 비틀고, 검을 올렸다.

목검 끝이 붉은 점에 닿았다.

정확히 빈틈 위에.

```
[명중. 가상 대상 제압.]
[반응 시간: 1.2초. 실전 기준으로는 느리지만, 첫 체험치고는 양호합니다.]
```

남궁선은 목검을 든 채 서 있었다. 손이 떨리고 있었다. 흥분이었다.

이걸로 싸울 수 있다. 내공이 모자라도, 검술이 서툴러도 — 이 선이 보이면 싸울 수 있다.

```
[흥분하신 것은 이해하지만, 현실을 말씀드리겠습니다.]
[시연 모드는 정지된 가상 궤적입니다. 실전에서는 상대가 움직입니다.]
[궤적 가이드의 품질은 상대의 수준에 비례하여 토큰을 소모합니다.]
[창검당 제자 수준: 0.5토큰. 방계 교관 수준: 1.0. 본가 제자: 2~3.]
[남궁재 수준: 현재 연산 불가. 데이터 부족.]
```

남궁선의 흥분이 가라앉았다. 역시 공짜는 없다. 이 빛나는 선은 값을 먹는 선이었다.

'형한테 쓰면 몇이야.'

```
[추정 불가. 최소 3토큰 이상.]
[현 잔여 토큰으로 5회 이내. 비추입니다.]
```

남궁선은 목검을 내렸다. 아쉽지만 맞는 판단이었다. 궤적 가이드는 진짜 필요할 때만 쓰는 것이다. 수련은 몸으로 해야 한다.

```
[시연 모드 종료.]
[조언: 궤적 가이드 없이도 상대의 패턴을 읽는 연습을 하십시오.]
[가이드는 보조 장치입니다. 눈을 대신하는 것이 아닙니다.]
[무료 조언입니다.]
```

'그건 맞는 말이네.'

```
[가끔 맞는 말도 합니다.]
```

남궁선은 검을 들었다. 빛나는 선 없이, 자기 눈으로. 정통 제1초식의 궤적을 머릿속에 떠올렸다. 앞줄 제자들의 검, 조 사범의 검, 형의 검. 태을이 아닌 자기 기억에서 꺼낸 궤적.

거기에 빈틈이 보일까.

아직은 보이지 않았다. 하지만 방향은 알겠다.

---

다음 날 아침.

어머니의 기침이 심했다.

부엌에서 죽을 끓이다가 세 번 연속으로 기침을 했고, 세 번째에 몸이 기울었다. 남궁선이 달려가 부축했다. 어머니의 몸이 가벼웠다. 너무 가벼웠다.

"괜찮아, 선아. 원래 아침에 좀 그래."

남궁선은 어머니를 앉히고 대신 죽을 저었다. 손이 떨렸지만 티를 내지 않았다.

```
[긴급 센서 데이터.]
[유설화. 기침 강도 전주 대비 18% 상승. 폐부 손상 진행 가속 감지.]
[기존 예측: 비가역 단계 도달 4년 2개월 후.]
[수정 예측: 3년 10개월 후.]
[4개월 앞당겨졌습니다.]
```

남궁선의 숟가락이 솥 안에서 멈췄다.

'왜. 왜 빨라지는 거야.'

```
[원인 추정: 계절 변화에 의한 기온 저하, 영양 부족, 폐부의 기저 손상이 자체 악화 중.]
[정밀 스캔을 실행하면 정확한 원인과 진행 속도를 파악할 수 있습니다.]
[비용: 0.5토큰.]
```

'해.'

망설이지 않았다.

```
[결제 완료. 0.5토큰 소모. 잔여: 15.9.]
[유설화 정밀 신체 스캔 실행 중...]
```

남궁선의 눈앞에 어머니의 신체가 푸른 윤곽으로 떠올랐다. 투과되듯 내부가 보였다. 폐. 왼쪽 폐의 하단에 검은 그림자가 있었다. 오른쪽에도 작은 점이 있었다.

```
[스캔 완료.]
[진단: 기맥 3곳 막힘. 폐부 좌하엽 손상 진행 중. 우측은 초기.]
[현재 단계: 중등도. 치료하지 않으면 비가역 진행.]
[치료 방안 1: 천기 연단술 (100토큰). 완치 가능.]
[치료 방안 2: 영약 투여 + 기맥 소통. 악화 정지 및 부분 회복 가능.]
[방안 2에 필요한 영약: '청심초(淸心草)'. 폐부 손상 억제 효능.]
```

'청심초. 어디 있어?'

```
[회귀 기억 기반 위치 추정.]
[스캔 중... 기억 신뢰도 41%.]
[위치: 창검당 영지 북쪽, 금수계곡 상류. 절벽 아래 그늘진 바위틈.]
```

41%. 열에 넷. 반도 안 되는 확률.

```
[신뢰도 향상을 위한 정밀 기억 스캔: 5.0토큰.]
[실행하시겠습니까?]
```

남궁선은 고개를 저었다. 5.0이면 잔여의 삼분의 일이다. 쓸 수 없다.

'41%로 간다.'

```
[절반도 안 됩니다.]
```

'없는 것보다는 낫잖아.'

```
[......이전에도 비슷한 말을 하신 적이 있는 것 같습니다.]
[동의합니다.]
```

남궁선은 죽을 그릇에 담았다. 어머니에게 건네고, 자기 몫도 떠서 먹었다.

싱거웠다. 언제나 싱거웠다. 하지만 따뜻했다.

'가야 해.'

하지만 지금은 아니다. 금수계곡은 반나절 거리이고, 절벽 아래라면 위험할 수 있다. 체력을 더 길러야 하고, 내공도 더 필요하다.

```
[권장: 최소 내공 0.5 이상에서 출발.]
[현재 내공: 0.3.]
[예상 도달 시점: 10일 후.]
[급하지만 무리하면 빈손으로 돌아옵니다.]
```

열흘. 남궁선은 그 숫자를 가슴에 새겼다.

열흘 뒤에 간다.
