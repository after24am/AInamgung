# 28화. 3회전

---

3회전 직전.

남궁선은 눈을 뜨고 비무대를 바라보았다. 직전 경기가 끝나고 있었다. 승자가 비무대에서 내려왔다. 다음이 남궁선 차례였다.

```
[3회전 사전 브리핑.]
[상대: 오천(吳天). 입문 5년차. 중급 제자.]
[내공 추정: 1.5~1.8. 플레이어(0.9) 대비 확실한 상위.]
[검식: 기초검식 상급 + 본가 중급 검법 일부.]
[약점: 왼손잡이 교정 습관. 급할 때 왼손이 나옴.]
[추가 약점: 공격 후 수비 전환 시 0.3초 공백.]
[토큰 예산: 3.0.]
[승률: 41%.]
```

41%. 열에 넷. 남궁선은 일어서며 손목을 돌렸다. 두 경기를 치렀다. 체력은 괜찮았다. 1회전에서 아꼈고, 2회전에서도 직접 맞은 건 없었다.

문제는 내공이었다. 0.9. 오천의 1.5~1.8에 비하면 반밖에 안 된다. 검이 부딪히면 밀린다. 2회전의 유광은 보법이 느려서 피할 수 있었지만, 오천은 — 빠르다.

서른여섯의 잔머리가 마지막 전략을 짰다.

정면으로 안 된다. 기동전도 어렵다. 오천의 보법은 유광과 다르다. 돌아다니면 쫓아온다. 시간을 끌면 내공 차이로 먼저 지친다.

답은 하나. 짧게 끝낸다.

'태을아. 오천을 30초 안에 이기는 방법이 있어?'

```
[30초 컷은 — 조건부로 가능합니다.]
[조건: 오천이 왼손 습관을 노출하는 순간을 포착하고,]
[창궁무애검법 제1초식으로 왼쪽 빈틈을 찌르는 것.]
[문제: 오천이 왼손 습관을 쓰려면 — 오천이 "급해야" 합니다.]
[오천을 급하게 만드는 방법이 필요합니다.]
```

오천을 급하게 만드는 방법. 남궁선은 생각했다.

급하다는 건 — 예상 밖의 상황에 처한다는 뜻이다. 오천이 예상하지 못한 것. 오천이 1, 2회전을 봤다면 남궁선에 대해 알고 있는 건 — 기동전, 카운터, 리팩토링 속도.

그러면 오천이 예상하지 못한 건 — 남궁선이 정면으로 오는 것이다.

'...아까 유광한테는 먼저 가서 먹혔지. 같은 수를 쓰면 안 돼.'

```
[맞습니다. 유광전의 "선제 공격"은 이미 노출되었습니다.]
[하지만 — 선제 공격의 종류가 다르면?]
```

남궁선의 눈이 빛났다.

1회전: 기다리는 검. 정통.
2회전: 먼저 가는 검. 리팩토링.
3회전: ?

매 경기마다 다른 검을 보여줬다. 오천은 둘 다 대비할 것이다. 기다려도, 먼저 와도 대응하겠다고.

그렇다면 — 세 번째 종류.

기다리다가 가는 검. 처음엔 지는 척, 나중에 뒤집기.

어젯밤 비무대에서 태을이 말한 그것.

```
[비무대 전술 — "초반 후퇴, 중반 탈환" 시나리오.]
[오천에게 맞춤 조정:]
[1단계(0~15초): 수비. 밀린다. 오천이 우세를 확신하게 만듦.]
[2단계(15~20초): 급격한 반격. 오천의 리듬을 깨뜨림. "급한" 상황 유발.]
[3단계(20~30초): 왼손 습관 노출 시 — 창궁무애검법 제1초식.]
[토큰 소모 예상: 2.5~3.0.]
```

남궁선은 고개를 끄덕였다.

"8번 남궁선, 3번 오천. 비무대로."

---

오천(吳天). 스물둘. 단정한 얼굴에 차가운 눈. 비무대 위에서 남궁선을 바라보는 시선이 — 1, 2회전 상대와 질이 달랐다.

얕보지 않았다. 남궁선이 두 경기를 이긴 것을 봤다. 방계 출신이라고 무시할 수준이 아니라는 걸 알고 있었다. 경계하는 눈이었다.

```
[오천의 태도: 경계 + 자신감.]
["이길 수 있지만 방심하지 않겠다"는 태도.]
[이런 상대가 가장 다루기 어렵습니다.]
```

남궁선은 목검을 잡았다. 천천히 숨을 내쉬었다.

관중석이 조용해졌다. 3회전부터 수준이 달라진다는 걸 모두 알고 있었다. 방계 출신이 여기까지 온 것만으로 화제였다.

남궁혁의 시선이 — 이번에는 확실히 비무대 위에 고정되어 있었다.

"시작."

오천이 움직였다. 빨랐다. 기초검식 제3초식 — 하지만 남궁선이 아는 기초검식이 아니었다. 본가의 검. 같은 초식인데 궤적이 더 날카롭고, 힘 전달이 더 효율적이었다.

본가는 기초검식부터 다르다. 남궁선은 이를 악물고 받았다.

탁.

목검이 부딪혔다. 충격이 손목을 타고, 팔꿈치를 타고, 어깨까지 울렸다. 내공 차이. 한 수 받았을 뿐인데 팔이 저렸다.

오천의 두 번째 수가 곧바로 왔다. 쉼 없는 연격. 세 번째 수. 네 번째 수.

남궁선은 받고, 흘리고, 빠지고, 또 받았다. 리팩토링 검식의 속도로 간신히 따라갔지만 — 밀렸다. 비무대 가장자리로.

밧줄이 등 뒤에서 느껴졌다. 한 발만 더 빠지면 장외패.

관중석에서 탄식이 나왔다. "거기까지인가." 누군가 말했다.

```
[현재 상황: 비무대 가장자리. 밧줄까지 1보.]
[오천: 비무대 중앙. 리듬 안정. 우세 확신.]
[1단계 완료. 오천은 "곧 끝난다"고 생각하고 있습니다.]
[2단계 진입 권장. 지금.]
```

남궁선의 눈이 달라졌다.

밀린 것이 아니었다. 밀려준 것이었다.

15초 동안 남궁선은 오천의 검을 전부 받았다. 전부 봤다. 오천의 리듬, 연격의 패턴, 수비와 공격의 전환점, 힘을 싣는 타이밍. 서른여섯의 눈이 15초 동안 읽은 데이터.

```
[궤적 가이드 — 풀 가동. 1.5토큰.]
[오천 패턴 분석 완료.]
[연격 후 5번째 수에서 0.3초 공백 — 공격→수비 전환점.]
[다음 5번째 수 예측 시점: 2초 후.]
[궤적 표시.]
```

오천의 다섯 번째 수가 왔다. 대각선 내려베기. 강했다. 마무리 일격처럼 내려쳤다.

남궁선은 받지 않았다. 밧줄 앞에서 — 왼쪽으로 돌았다. 오천의 검이 빈 공기를 갈랐다. 마무리라고 생각한 일격이 빗나갔다.

0.3초. 오천의 검이 아래로 내려간 순간. 공격이 끝나고 수비가 시작되기 전의 공백.

남궁선이 치고 들어갔다.

리팩토링 제2초식. 예비 동작 없는 횡베기. 오천의 오른쪽 옆구리를 향해.

오천이 반응했다. 빨랐다. 검을 올려 막았다. — 막았다. 중급 제자는 역시 달랐다.

하지만 남궁선은 첫 수로 끝낼 생각이 아니었다. 리듬을 깨뜨리는 것이 목적이었다.

이어서 제4초식. 찌르기. 오천의 왼쪽 어깨.

오천이 검을 돌려 막았다. 또 막았다. 하지만 리듬이 달라졌다. 공격하던 사람이 갑자기 수비로 돌아선 것이다. 오천의 표정이 변했다.

"급해졌습니다."가 아니라 — 아직은 "당황"에 가까웠다.

남궁선은 세 번째 수를 넣었다. 리팩토링 제7초식. 아래에서 올려치기. 오천의 손목을 노렸다.

오천이 뒤로 빠졌다. 처음으로 오천이 빠졌다. 비무대 중앙을 — 남궁선에게 내줬다.

관중석이 술렁였다. 밀리던 방계 제자가 중앙을 점거했다.

오천이 멈춰 섰다. 숨을 고르고 — 다시 왔다. 이번에는 달랐다. 기초검식이 아니었다. 본가 중급 검법. 궤적이 복잡해졌다. 곡선과 직선이 섞인, 남궁선이 처음 보는 검.

```
[본가 중급 검법 — 청풍검식(淸風劍式) 추정.]
[데이터 없음. 궤적 예측 불가.]
[센서 분석으로 실시간 대응 시도. 0.5토큰 추가.]
```

남궁선은 이를 악물었다. 처음 보는 검. 예측이 안 된다. 받아야 한다.

탁. 탁. 탁.

세 수를 받았다. 세 수 모두 — 겨우 받았다. 목검이 흔들렸다. 내공 차이가 본가 검법과 합쳐지니 벽이었다.

네 번째 수에서 남궁선의 목검이 밀렸다. 오천의 검이 남궁선의 어깨를 스쳤다. 도복이 찢어졌다.

"유효타." 심판의 목소리.

남궁선은 한 발 물러섰다. 어깨가 얼얼했다. 목검인데도 — 내공이 실리면 이 정도였다.

```
[유효타 1회 허용. 3회 허용 시 패배.]
[어깨 충격: 경미. 전투 지속 가능.]
[토큰 잔여 예산: 1.0.]
[시간: 교전 32초.]
[판단: 장기전 불리. 끝내야 합니다.]
```

끝내야 한다. 남궁선은 숨을 내쉬고 — 계산을 끝냈다.

지금까지 오천은 세 가지를 보여줬다. 기초검식, 연격 패턴, 청풍검식. 세 가지 모두 오른손 검이었다. 교정된, 깔끔한, 정통 오른손 검.

왼손 습관은 아직 안 나왔다. 급하지 않았으니까. 오천이 우세했으니까.

오천을 급하게 만들어야 한다.

남궁선은 검을 바꿨다. 리팩토링 검식이 아니라 — 창궁무애검법.

제1초식.

직선에서 곡선으로. 곡선에서 다시 직선으로. 전환점에 기를 집중하여 궤적을 비틀었다.

오천의 눈이 커졌다. 이 검은 — 아까와 다르다. 완전히.

창궁무애검법의 곡선이 오천의 왼쪽을 파고들었다. 왼쪽. 의도적이었다. 왼손잡이의 교정된 오른손은 — 왼쪽 방어가 미세하게 늦다.

오천이 막았다. 막았지만 — 0.1초 늦었다. 곡선의 궤적이 오천의 예측을 벗어났다. 처음 보는 검법. 처음 보는 궤적.

남궁선은 이어서 제2초식을 넣었다. 곡선이 반대로 흘렀다. 오른쪽에서 왼쪽으로. 다시 왼쪽으로.

오천이 뒤로 빠지며 막았다. 두 번째로 빠졌다. 오천의 얼굴에 처음으로 — 초조함이 떠올랐다.

```
[오천 스트레스 반응 감지.]
[호흡 빨라짐. 검 쥔 손에 힘 증가.]
[왼손 — 움직임 감지.]
```

왔다.

오천이 반격에 나섰다. 청풍검식의 연격. 빠르고 강했다. 하지만 — 세 번째 수에서.

오천의 왼손이 검집 쪽으로 갔다. 0.2초. 무의식적 동작. 급해지니 왼손 습관이 나온 것이다.

그 0.2초 동안 오천의 오른손 검이 — 흔들렸다.

남궁선은 그 흔들림을 놓치지 않았다.

```
[궤적 가이드 — 최종. 0.5토큰.]
[왼쪽 갈비뼈. 수비 공백 0.2초. 지금.]
```

창궁무애검법 제1초식. 곡선 전환. 오천의 왼쪽 갈비뼈를 향해.

오천이 반응했다. 왼손이 먼저 움직였다 — 검을 잡으려고. 하지만 검은 오른손에 있었다. 0.1초의 혼란. 왼손은 허공을 잡았고, 오른손은 방향 전환이 늦었다.

남궁선의 목검이 오천의 왼쪽 갈비뼈에 닿았다.

정확했다. 깊지 않았다. 하지만 — 확실했다.

"유효타."

오천이 한 발 물러섰다. 얼굴이 굳어 있었다. 지금 뭐가 일어난 건지 — 본인도 정확히 모르는 표정이었다.

1:1. 유효타 동점.

오천의 눈이 달라졌다. 경계가 아니었다. 경각심이었다. "질 수도 있다"는 눈.

두 사람이 다시 마주 섰다. 비무대 중앙에서.

오천이 먼저 움직였다. 빨랐다. 이전보다 빨랐다. 본가 중급 검법의 연격이 쏟아졌다. 전력이었다.

남궁선은 받았다. 창궁무애검법 제3초식의 곡선 흘리기로 힘을 분산시키며 — 버텼다.

하지만 내공 차이는 기술로 완전히 메울 수 없었다. 한 수. 두 수. 세 수. 네 수째에 남궁선의 팔이 밀렸다.

다섯 수째. 오천의 검이 남궁선의 손목을 쳤다.

탁.

목검이 — 떨어지지 않았다. 남궁선이 이를 악물고 잡았다. 하지만 손목이 저렸다. 감각이 반쯤 날아갔다.

"유효타." 심판의 목소리.

2:1. 오천 우세.

```
[유효타 2회 허용. 1회 더 허용 시 패배.]
[오른손 손목 충격: 중간. 악력 40% 저하.]
[토큰 잔여: 0.3.]
[시간: 교전 51초.]
```

남궁선은 손목을 꽉 쥐었다. 목검을 바꿔 잡았다. 오른손에서 — 왼손으로 넘기지 않았다. 그건 남궁선의 카드가 아니었다. 오른손 그대로. 약해진 악력 그대로.

오천이 달려왔다. 마지막이라고 판단한 것이다. 빠르고, 강하고, 정확한 일격.

남궁선은 — 빠지지 않았다.

정면으로 갔다.

목검이 부딪혔다. 탁. 남궁선의 팔이 튕겨졌다. 악력이 부족했다. 목검이 흔들렸다.

하지만 남궁선은 튕겨지는 힘을 — 돌렸다. 튕겨진 검이 원을 그리며, 곡선을 그리며, 아래에서 위로 올라왔다.

창궁무애검법 제2초식의 변형. 튕겨지는 힘을 곡선 전환의 동력으로 쓴 것이다.
