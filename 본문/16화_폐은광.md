# 16화. 폐은광

---

다음 날 밤. 달이 구름에 반쯤 가려져 있었다.

남궁선과 남궁재는 금수계곡을 올라갔다. 형의 걸음은 빠르고 조용했다. 남궁선은 따라가기 바빴지만, 한 달 반의 수련이 발목을 붙잡지는 않았다.

계곡 중턱에서 남궁선이 손을 들었다.

"여기서부터 보초 범위입니다."

남궁재가 고개를 끄덕였다. 검을 잡은 손에 힘이 들어갔다.

```
[센서: 전방 45보. 보초 1명. 위치 — 전과 동일. 바위 위.]
[심박수: 안정. 경계 수준 낮음.]
[추가: 동굴 방향에서 인물 복수 감지. 정확한 수는 거리 한계로 불명.]
```

남궁선은 형에게 손짓으로 보초의 위치를 알려주었다. 남궁재는 한 번 보고 고개를 끄덕였다.

"내가 간다. 넌 여기서 기다려."

예상한 대로였다. 형은 동생을 앞에 세울 사람이 아니다.

남궁선은 순순히 고개를 끄덕였다. "예."

남궁재가 바위 그림자를 타고 앞으로 나아갔다. 소리가 없었다. 남궁선보다 열 배는 능숙한 움직임이었다.

```
[남궁재가 보초에 접근 중. 거리 20보... 10보...]
```

남궁선은 기다리지 않았다.

형이 보초에 집중하는 사이, 남궁선은 계곡 왼쪽 사면을 탔다. 지난번에 우회했던 길이었다. 보초를 지나, 동굴 쪽으로.

```
[플레이어가 대기 명령을 무시하고 이동 중입니다.]
[예상된 행동입니다.]
```

'이러려고 형한테 정면을 맡긴 거야.'

```
[알고 있었습니다. 작전 공유 요청 시 이 부분만 빠져 있었으니까.]
```

남궁선은 바위 사이를 기어갔다. 형이 보초를 처리하는 동안 동굴까지 갈 수 있다. 형은 보초를 조용히 제압하고 안쪽으로 올 것이다. 그 사이에 남궁선은 동굴 안에서 저들이 뭘 파는지 확인한다.

사면을 타고 올라가는 도중에 — 뒤쪽에서 짧은 소리가 났다. 목검이 사람을 때리는 둔탁한 소리. 형이 보초를 쓰러뜨린 것이다.

조용했다. 비명은 없었다. 한 방에 끝났다는 뜻이다.

남궁선은 걸음을 빨리했다.

---

동굴 입구가 보였다.

모닥불이 꺼져 있었다. 불씨만 남아 희미하게 빛나고 있었다. 바깥에 사람이 없었다. 전원 안에 들어간 것이다.

안쪽에서 곡괭이 소리가 들렸다. 그리고 — 말소리.

"...독기가 옅어졌다. 거의 다 뚫렸어."

"주인께 전서를 보내야 합니까?"

"아직이다. 실물을 확인한 뒤에."

남궁선은 동굴 입구 옆 바위에 몸을 붙이고 귀를 세웠다. 태을의 센서보다 자기 귀를 먼저 쓰는 습관이 들어 있었다. 값이 안 드니까.

"실물"이라는 단어가 걸렸다. 은이 아니다. 특정한 무언가를 찾고 있다.

```
[청각 센서 보조. 무료.]
[내부 인원: 5명. 전원 동굴 심부에 집중.]
[입구에서 심부까지 추정 거리: 30보.]
```

남궁선은 생각했다. 들어갈까. 안에 5명. 들어가면 갇힌다. 하지만 밖에서는 "실물"이 뭔지 알 수 없다.

형이 올 때까지 기다리면 — 형이 오면 안으로 들어가지 못하게 할 것이다.

남궁선은 동굴 안으로 들어갔다.

```
[......매번 위험한 선택을 하시는 건 성격입니까.]
```

'아니. 계산이야. 5명이 전부 30보 안쪽에 있어. 입구에는 아무도 없어. 소리 없이 10보만 들어가면 안이 보여.'

```
[10보 이내에서 발각되면 탈출 경로가 없습니다.]
```

'형이 뒤에서 온다. 형이 입구를 막으면 저쪽이 갇히는 거야.'

```
[......그 계산은 맞습니다. 인정합니다.]
```

동굴 안은 축축했다. 벽에 곡괭이 자국이 선명했다. 오래된 갱도. 은광의 흔적이 남아 있었다. 천장이 낮아 허리를 숙여야 했다.

5보. 어둠에 눈이 적응했다.

10보. 안쪽에서 불빛이 보였다. 횃불. 그리고 사람들의 그림자.

남궁선은 벽에 붙어 멈췄다. 앞을 내다보았다.

갱도가 넓어지는 곳에 다섯 명이 있었다. 셋은 곡괭이를 들고 바닥을 파고 있었고, 둘은 팔짱을 끼고 서 있었다. 검은 옷. 등에 검. 파는 놈과 지키는 놈이 구분되어 있었다.

그리고 — 바닥에 구멍이 뚫려 있었다. 사람 하나가 들어갈 크기. 거기서 기묘한 기운이 올라오고 있었다.

독기가 아니었다. 남궁선의 단전이 반응했다. 미세하게, 하지만 확실하게. 기(氣)였다. 순수한 기운이 바닥에서 올라오고 있었다.

```
[경고: 미확인 기류 감지.]
[분석 — 지하에서 올라오는 기류는 독기가 아닙니다.]
[기의 밀도가 비정상적으로 높습니다.]
[추정: 지하에 기를 품은 물체 또는 공간이 존재합니다.]
[정밀 분석: 1.0토큰.]
```

남궁선은 눈을 좁혔다. 기를 품은 무언가가 이 밑에 있다. 그걸 찾으려고 이 사람들이 파고 있는 것이다.

1.0값은 아깝지만 — 써야 했다.

'해.'

```
[결제 완료. 1.0토큰 소모. 잔여: 15.95.]
[정밀 분석 실행...]
[지하 약 5장(丈) 깊이에 밀폐된 공간 감지.]
[내부에 고농도 기류 순환 중.]
[형태: 인공적. 자연 동굴이 아닙니다.]
[추정: 누군가가 만든 밀실. 내부에 무언가를 봉인한 것으로 보입니다.]
```

봉인. 50년 전 독기 때문에 폐쇄한 게 아니었다. 독기는 — 봉인의 부산물이었을 수 있다.

남궁선의 서른여섯이 머릿속에서 소리쳤다. 여기서 더 들어가면 안 된다. 정보는 충분하다. 돌아가야 한다.

돌아서는 순간, 발밑의 돌이 굴렀다.

작은 소리였다. 하지만 동굴 안에서는 — 충분히 큰 소리였다.

"누구냐!"

안쪽에서 검을 지키던 둘 중 하나가 고개를 돌렸다. 횃불빛이 남궁선이 있는 방향을 비추었다.

남궁선은 이미 뛰고 있었다.

몸을 돌려 갱도를 달렸다. 허리를 숙인 채, 벽에 어깨를 부딪히면서. 뒤에서 발소리가 쫓아왔다. 두 명. 빠르다. 교관급의 보법이었다.

```
[추적자 2명. 거리 15보. 접근 속도 — 플레이어의 1.5배.]
[10초 내에 따라잡힙니다.]
```

남궁선은 이를 악물었다. 10초. 입구까지 20보. 부족하다.

입구가 보였다. 밤하늘. 그리고 — 입구에 서 있는 그림자.

남궁재였다.

형이 입구를 막고 서 있었다. 검을 들고. 달빛을 등진 채.

남궁선이 형 옆을 스쳐 지나갔다. "둘이요!"

남궁재가 한 발 앞으로 나섰다. 추적자 두 명이 갱도에서 뛰어나왔다. 검을 뽑은 상태.

남궁재의 검이 움직였다.

남궁선은 보지 못했다. 소리만 들었다. 금속이 부딪히는 소리 두 번, 사람이 벽에 부딪히는 소리 한 번. 3초도 안 걸렸다.

돌아보니 추적자 두 명이 갱도 안에 쓰러져 있었다. 기절. 남궁재의 검에 피는 묻어 있지 않았다.

```
[교전 시간: 2.7초. 남궁재의 전투력 — 교관급 상위.]
[플레이어의 판단이 맞았습니다. 형을 데려온 것은 정답이었습니다.]
```

남궁재가 돌아보았다. 형의 눈이 서늘했다.

"기다리라고 했다."

"죄송합니다."

"뭘 봤느냐."

남궁선은 빠르게 말했다. "밑에 봉인된 공간이 있습니다. 기를 품은 뭔가가 있어요. 이 사람들이 그걸 파내려고 하는 겁니다."

남궁재의 눈이 미세하게 떨렸다. 형은 그 말의 무게를 이해했다.

"안에 셋이 더 있다."

"알고 있습니다."

남궁재는 갱도 안을 한 번 보았다. 그리고 — 돌아섰다.

"돌아간다."

"형님?"

"셋이 더 있고, '주인'이라는 놈의 수준도 모른다. 여기서 끝낼 수 있는 판이 아니다."

남궁선은 입을 다물었다. 형이 맞았다. 안에 셋을 치러 들어가면 — 좁은 갱도에서 난투가 된다. 형이 강해도 셋을 상대하며 동생까지 지킬 수는 없다.

"아버지에게 알려야 한다."

남궁재의 말에 남궁선은 고개를 끄덕였다. 이 단계까지 왔으면 당주에게 보고하는 게 맞다. 정보는 충분히 모았다.

두 사람은 계곡을 내려갔다. 빠르게. 쓰러진 보초와 추적자 둘이 깨어나기 전에 거리를 벌려야 했다.

달리면서 남궁선은 생각했다.

봉인된 공간. 기를 품은 무언가. 50년 전 폐쇄된 은광 아래에.

그리고 그것을 "주인"에게 가져다주려는 자들.

```
[사건 규모가 창검당 차원을 넘어서고 있습니다.]
[조언: 이 정보를 본가에도 알려야 할 수 있습니다.]
```

남궁선은 대답하지 않았다. 본가. 본가에 알리면 — 본가가 가져간다. 봉인된 것이 무엇이든, 방계의 영지에 있어도 본가가 손을 뻗으면 방계 몫은 없다.

잔머리가 또 돌기 시작했다.

'...아버지에게 먼저. 본가는 나중에.'

```
[정치적 판단입니다.]
```

'생존이야.'
