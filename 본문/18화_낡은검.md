# 18화. 낡은 검

---

폐은광 사건 이후, 창검당은 긴장 속에 며칠을 보냈다.

남궁성은 영지 북쪽 경계에 교대 순찰을 세웠다. 조 사범과 부사범이 번갈아 돌았다. 포박한 마교 잔당 넷은 본가에 넘겼다. 남궁성은 고민 끝에 본가에 알렸다. 다만 — 남궁선이 짠 대로 알렸다.

"영지 순찰 중 수상한 자들을 발견하여 제압했습니다."

폐은광의 봉인도, 고대 검도 언급하지 않았다. 마교 잔당이 영지에 침입했다는 사실만 보고.

이것은 남궁선의 잔머리였다.

'본가에 "마교 잔당을 잡았다"고만 알리면요. 본가는 창검당이 공을 세웠다고 생각합니다. 우리한테 빚이 생기는 거죠.'

남궁성은 그 말을 듣고 한참 아들을 바라보았다. 곧은 사람이 아들의 꾀를 받아들이는 건 쉬운 일이 아니었다. 하지만 — 받아들였다.

본가에서 답신이 왔다. "수고했다. 잔당의 심문은 본가에서 처리하겠다." 담담한 한 줄. 하지만 남궁선은 그 한 줄 뒤에 깔린 것을 읽었다.

방계가 일을 했다. 본가가 인정했다. 작은 빚.

```
[정치적 자산 +1. 수치화가 어렵지만, 있는 것과 없는 것은 다릅니다.]
[무료 분석입니다.]
```

'빚은 모아두면 쓸 데가 있어.'

```
[고리대금 사고방식이지만, 효과적입니다.]
```

---

고대 검은 남궁성의 집무실에 보관되었다.

사건 후 사흘째 아침, 남궁성이 남궁선을 불렀다.

집무실에 들어서자, 탁자 위에 검이 놓여 있었다. 삭은 검집에서 꺼내진 상태. 검신이 드러나 있었다.

녹이 있을 줄 알았다. 없었다.

검신은 맑은 은빛이었다. 100년 이상 땅속에 묻혀 있었는데, 녹 한 점 없이 빛나고 있었다. 그리고 검신 위에 글자가 새겨져 있었다. 너무 작아서 눈을 가까이 대야 보였다.

```
[글자 판독. 무료 범위.]
[검신 각인: '창(蒼)'.]
[한 글자만 새겨져 있습니다.]
```

창. 푸를 창.

남궁선은 아버지를 보았다. 남궁성의 표정이 복잡했다.

"아버지. 이 글자..."

"남궁세가의 검에 새기는 글자다."

남궁선의 눈이 커졌다. 남궁세가의 검.

"본가의 검이 왜 여기에?"

남궁성이 한숨을 내쉬었다. "모르겠다. 하지만 영지에 묻혀 있었다는 건 — 옛날에 누군가가 의도적으로 숨긴 거다."

남궁성이 검을 들어 남궁선에게 내밀었다.

"가져라."

"...예?"

"본가에 알리지 않은 이상, 이건 우리 것이다. 그리고 나는 이미 검이 있다."

남궁성이 허리의 장검을 두드렸다. 삼십 년 쓴 검.

"네가 수련하는 건 안다. 목검으로는 한계가 있지."

남궁선은 검을 받아들었다. 생각보다 가벼웠다. 그리고 — 손에 닿는 순간, 단전이 반응했다. 검에서 흘러나오는 기가 남궁선의 기와 공명하듯 진동했다.

```
[경고: 검에서 유입되는 기류 감지.]
[검신의 잔류 기가 플레이어의 기맥과 동조하고 있습니다.]
[이 현상은 — 검과 사용자의 기맥 구조가 유사할 때 발생합니다.]
[정밀 분석: 3.0토큰.]
```

남궁선은 잠시 고민했다. 3.0. 크다. 하지만 이 검은 — 분명 보통이 아니었다.

'...아끼지 말아야 할 때가 있지.'

```
[드문 판단입니다. 결제하시겠습니까?]
```

'해.'

```
[결제 완료. 3.0토큰 소모. 잔여: 16.95.]
[정밀 분석 실행...]
```

남궁선의 눈앞에 검의 내부 구조가 떠올랐다. 검신 안에 기의 흐름이 보였다. 맥처럼 갈라지는 선들. 사람의 혈맥과 비슷한 구조였다.

```
[분석 완료.]
[검명: 불명. 검신 각인 '창(蒼)'만 확인.]
[재질: 한철(寒鐵) 합금. 일반 철의 3배 강도. 기 전도율 극상.]
[잔류 기: 검 제작자의 기가 검신에 봉인되어 있음.]
[특수 효과: 사용자의 기를 증폭합니다. 현재 내공 0.5 기준 — 체감 내공 약 0.8로 상승.]
[부가 정보: 검신의 기맥 구조가 남궁세가 혈통의 기맥과 유사합니다.]
[이 검은 남궁세가 사람을 위해 만들어졌습니다.]
```

남궁선은 검을 쥔 채 멍하니 서 있었다.

내공 0.5가 0.8이 된다. 열에 여섯이 늘어나는 셈이다. 이건 — 엄청난 것이었다.

'이 검 이름이 뭐야?'

```
[불명입니다. 검신에 '창' 한 글자뿐.]
[이름을 지어주시겠습니까?]
```

남궁선은 검을 들어 올렸다. 은빛 검신에 달빛이 반사되었다.

'...필요 없어. 창이면 됐지.'

낡은 검집에 검을 넣었다. 검집은 새로 만들어야 한다. 하지만 그건 나중 일이다.

남궁성이 한 가지 더 내밀었다. 천으로 감싼 얇은 책.

"이것도 가져가라."

남궁선이 펼쳤다. 낡은 종이에 빼곡한 글씨. 검식의 초식도가 그려져 있었다.

"이건..."

"창검당의 비장 검결이다. 《창궁무애검법(蒼穹無礙劍法)》. 대대로 당주만 보관하는 것인데 — 익히는 자가 없어서 장롱에 묻혀 있었지."

남궁선은 검결을 내려다보았다. 기초검식보다 복잡했다. 초식도의 궤적이 자유롭고, 변화가 많았다. 기초검식의 직선적인 움직임과는 차원이 달랐다.

```
[비급 감지.]
[《창궁무애검법》— 초식 12수. 기초검식의 상위 검법.]
[무료 범위 판독: 기본 구조 확인 가능. 초식 해석은 불가.]
[정밀 해독: 5.0토큰.]
```

남궁선은 고개를 저었다. 5.0은 안 쓴다. 아까 3.0 썼으니까 오늘은 이만.

'해독은 내가 할게. 눈으로 읽고 몸으로 익히면 되지.'

```
[시간이 걸립니다.]
```

'시간은 있어.'

```
[4년 2개월. 엄밀히는 3년 10개월.]
```

남궁선은 대답하지 않았다. 어머니의 기한. 잊은 적 없다.

검결을 품에 넣고 집무실을 나섰다. 허리에 낡은 검이 걸려 있었다. 목검 대신 진검. 무게가 달랐다. 걸음이 달라졌다.

```
[장비 변경: 목검 → 창(蒼).]
[체감 내공: 0.5 → 0.8.]
[전투력 변동: 측정 필요.]
[하지만 확실히 — 어제보다 강합니다.]
```

남궁선은 서별채로 걸어가면서 검결의 첫 장을 머릿속으로 되새겼다. 제1초식의 궤적. 기초검식과 시작은 같지만 — 중간에 갈라진다. 직선이 곡선이 되고, 곡선이 다시 직선으로 돌아온다.

서른여섯의 잔머리가 말했다. 이건 혼자 풀 수 있다. 태을의 5.0값짜리 해독 대신, 기초검식 리팩토링 때처럼 하면 된다. 초식을 보고, 따라 하고, 틀리면 고치고.

느리지만 공짜다.

'궁갑이의 방식이지.'

```
[구두쇠의 방식이라고도 합니다.]
```

'같은 말이야.'

```
[이번엔 동의합니다.]
```
