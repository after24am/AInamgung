# 8화. 굳은살

---

보름이 흘렀다.

남궁선의 하루는 정해져 있었다. 해 뜨면 연무장, 해 지면 서별채 뒤편. 사이사이에 어머니의 죽을 먹고, 아버지의 등을 보고, 잠이 들었다.

기초검식 8초식의 리팩토링은 열흘째에 끝났다. 총 투자 0.8값. 태을이 내놓은 성적표는 이랬다.

```
[기초검식 전 초식 리팩토링 완료.]
[평균 발동 속도: 정통 대비 21% 향상.]
[평균 체력 소모: 정통 대비 26% 절감.]
[투자 0.8토큰 → 누적 초과 적립 2.1토큰.]
[수익률 263%.]
```

'수익... 그건 됐고.'

```
[관심 없는 척하시지만 방금 입꼬리가 올라갔습니다. 무료 센서 데이터.]
```

남궁선은 입꼬리를 내렸다.

연무장에서의 위치는 여전히 끝자리였다. 조 사범의 눈에 남궁선은 "그래도 나오긴 하는 놈" 정도였다. 제자들 사이에서 망나니라는 시선이 바뀌지는 않았다. 남궁선이 일부러 그랬다. 리팩토링된 검식을 연무장에서 쓰지 않았다. 정통 동작으로만 수련했다.

```
[실력 은폐 전략. 효율 점수 7.2/10.]
```

'점수 매기지 말라니까.'

```
[습관입니다.]
```

밤수련 때만 깎인 검을 썼다. 형 남궁재는 그 뒤로 두 번 더 나타났다. 한 번은 멀찍이 서서 지켜보다 갔고, 한 번은 다시 대련을 신청했다. 결과는 7초 만에 패배.

닷새 전보다 3초 늘었다.

```
[교전 시간 75% 증가. 유의미한 성장입니다.]
[이 속도라면 30초 버티기까지 약 40일.]
```

'30초면 뭐가 달라지는데.'

```
[아무것도요. 여전히 집니다.]
```

'...'

```
[하지만 30초 버티면 상대의 패턴을 읽을 시간이 생깁니다.]
[그때부터가 진짜 대련입니다.]
```

남궁선은 고개를 끄덕였다. 지금은 형의 첫 수에 밀려나지만, 30초를 버티면 형의 검을 볼 수 있다. 보면 배울 수 있고, 배우면 따라잡을 수 있다.

멀지만 보이는 길이었다.

---

아침이었다.

부엌에서 어머니가 상을 차리고 있었다. 죽과 나물. 늘 같은 아침이었다.

남궁선이 앉았다. 어머니가 밥그릇을 내밀었고, 남궁선이 받아들었다. 그 순간 — 어머니의 손이 멈췄다.

유설화가 아들의 손을 보고 있었다.

남궁선의 손바닥. 목검을 쥐던 자리에 굳은살이 올라 있었다. 보름 전에는 없던 것이었다. 부드럽기만 하던 망나니의 손에, 검을 잡은 흔적이 박혀 있었다.

"선아."

"네?"

"이 손이..."

유설화의 눈이 젖어들었다. 남궁선은 손을 슬쩍 뒤로 뺐지만, 어머니가 먼저 잡았다. 마른 손이 아들의 굳은살 위를 더듬었다.

"수련... 하고 있는 거야?"

남궁선은 어머니의 얼굴을 보았다. 서른여섯의 기억 속 어머니. 기침에 피가 섞이던 날에도, "괜찮다"고 웃던 사람. 스물넷 봄에 눈을 감은 사람.

"네. 하고 있어요."

유설화의 입술이 떨렸다. 무언가를 말하려다 — 그냥 웃었다. 눈물이 한 줄 흘렀지만 웃고 있었다.

```
[센서 데이터: 유설화. 심박수 상승. 호흡 불안정. 울음 판정.]
[감정 분류: 기쁨 + 안도. 복합 반응.]
```

남궁선은 죽을 떠먹었다. 한 숟갈도 남기지 않았다.

어머니가 부엌을 나간 뒤, 남궁선은 잠시 앉아 있었다. 빈 그릇을 내려다보았다.

'태을아.'

```
[네.]
```

'어머니 상태. 지금 속도면 어떻게 돼?'

```
[현재 진행 속도 유지 시, 폐부 손상이 비가역 단계에 도달하는 시점: 약 4년 2개월 후.]
[스물넷이 아니라 스물셋에 위험 구간 진입.]
[기존 기억보다 1년 빠릅니다.]
```

남궁선의 숟가락이 멈췄다. 1년 빠르다?

'왜?'

```
[알 수 없습니다. 회귀에 의한 인과 변동 가능성, 또는 플레이어의 기억 왜곡.]
[확실한 것은 — 영약이 필요하다는 점은 변함없습니다.]
```

남궁선은 일어섰다.

부엌 문을 나서려는데 — 마당에 사람이 서 있었다. 아버지였다.

남궁성이 문 앞에 서 있었다. 언제부터 있었는지 모르겠다.

"아버지."

남궁성은 아들을 내려다보았다. 넓은 어깨, 깊은 주름, 피곤한 눈. 당주의 얼굴이었다. 하지만 그 눈이 — 남궁선의 손으로 내려갔다.

굳은살.

남궁성의 시선이 거기 머물렀다.

아무 말이 없었다. 남궁선도 말하지 않았다. 부자가 마당에서 잠시 마주 선 채, 아침 햇살이 둘 사이를 비추었다.

남궁성이 돌아섰다. "저녁에 고기를 사 와라. 허 어르신이 외상을 내줄 게다."

그리고 걸어갔다.

남궁선은 아버지의 등을 보았다. 넓고, 곧고, 무거운 등. 아들에게 "수련하느냐"고 묻지 않았다. 대신 고기를 먹이려 했다. 그게 이 사람의 방식이었다.

```
[대상 '남궁성'의 심박 데이터를 감지했습니다.]
[평소 대비 심박수 12% 상승. 감정 분류: 불확실.]
[추정 가능 범위 내에서 — 기쁨, 또는 걱정.]
```

남궁선은 대답하지 않았다. 둘 다일 것이다.

---

저녁, 남궁선은 허 노인을 찾아갔다.

허 노인은 총무실에 앉아 있었다. 마르고 깐깐한 노인이 장부를 들여다보고 있었다. 금고 사건 이후, 허 노인의 눈 밑에는 그림자가 깊어져 있었다. 삼십 년 지킨 금고가 뚫렸다. 그 자존심이 아직 회복되지 않은 얼굴이었다.

"어르신. 아버지가 고기 외상 된다고 하셨는데요."

허 노인이 장부에서 눈을 떼지 않고 말했다.

"알고 있다. 벌써 돈을 빼놨다."

남궁선은 잠시 서 있었다. 허 노인이 고개를 들었다.

"뭐냐. 갈 거면 가라."

"어르신. 저 수련하고 있습니다."

허 노인의 붓이 멈췄다.

"들었다."

"...벌써요?"

"이 좁은 창검당에서 비밀이 어딨느냐. 조 사범이 진작 떠들고 다녔다."

허 노인의 눈이 남궁선을 올려다보았다. 예전처럼 독이 오른 눈은 아니었다. 금고 사건에서 남궁선이 범인을 잡아준 뒤로, 무언가가 달라져 있었다. 완전히 풀린 건 아니지만 — 문이 열린 정도.

"...오래 해라."

그 한마디가 전부였다. 허 노인은 다시 장부를 들여다보았다. 남궁선은 고기값을 받아들고 돌아섰다.

```
[호감도 변동: 허 노인 +4. 누적 호감도 — 측정 불가.]
[호감도 시스템은 탑재되어 있지 않습니다. 농담입니다.]
```

'그런 농담은 공짜여도 필요 없어.'

```
[유머 모듈 개선 중입니다.]
```

'개선하지 마.'

남궁선은 고기를 사 왔다. 저녁 상에 고기가 올라왔다. 어머니가 놀랐고, 아버지는 아무 말 없이 먹었다.

남궁선은 고기를 씹으며 생각했다. 보름 전과 달라진 것. 손에 굳은살이 생겼고, 연무장에 자리가 생겼고, 형이 지켜보고 있고, 허 어르신이 문을 열었다.

작은 것들이었다. 하지만 서른여섯의 기억에는 없던 것들이었다.

```
[현황 정리.]
[잔여 토큰: 15.3.]
[수련 일수: 15일.]
[기초검식 리팩토링: 8/8 완료.]
[내공: 미미. 그래도 0은 아닙니다.]
[다음 목표 제안: 첫 내공 순환 시도.]
```

남궁선은 고기를 한 점 더 집었다.

'내일 하자.'

```
[내일이라고 하고 실제로 하는 사람은 열에 셋입니다.]
```

'내가 그 셋이야.'

```
[기대하겠습니다.]
```
