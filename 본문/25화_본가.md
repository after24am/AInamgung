# 25화. 본가

---

비무 전날.

남궁선은 본가에 발을 디뎠다.

창검당에서 본가까지 반나절 거리. 아버지 남궁성과 함께 걸었다. 형 남궁재는 "볼 일이 있다"며 먼저 갔다. 무슨 볼 일인지 묻지 않았다. 형에게는 형의 방법이 있었다.

본가의 첫인상은 — 컸다.

창검당이 마을이라면, 본가는 성이었다. 높은 담장, 넓은 마당, 겹겹이 늘어선 전각. 연무장만 해도 창검당의 세 배는 됐다. 제자들이 수십 명씩 무리 지어 수련하고 있었고, 그 뒤로 더 넓은 연무장이 또 있었다.

```
[환경 스캔. 무료.]
[본가 연무장 면적: 창검당 대비 약 3.4배.]
[감지된 인원: 반경 내 47명. 대다수 내공 보유.]
[평균 내공 추정: 1.2~1.8. 플레이어 기준 상위.]
```

남궁선은 숫자를 씹었다. 평균이 1.2~1.8이면 — 남궁선의 0.9는 하위권이다. 검 증폭을 합쳐도 1.4. 평균에 간신히 닿는 수준.

'내가 여기서 제일 약해?'

```
[아닙니다. 비무 참가자 16명 기준으로는 중하위입니다.]
[하위가 아니라 중하위. 약간의 위안이 되시길.]
```

'안 돼.'

남궁성이 본가 관리인에게 안내를 받았다. 비무 참가자를 위한 객실이 배정되었다. 좁았다. 본가 제자들은 각자 방이 있지만, 외부 참가자는 한 칸에 몰렸다.

남궁선은 방에 짐을 풀었다. 창(蒼)을 벽에 기대세우고, 검결을 품에 넣었다.

```
[비무 전날 점검.]
[내공: 0.9 (검 증폭 1.4).]
[토큰 잔여: 17.8.]
[기초검식 8초식: 리팩토링 완료.]
[창궁무애검법 제1초식 81%, 제2초식 68%, 제3초식 51%.]
[전투 가용 패턴: 11종.]
[신체 상태: 양호. 근육 피로 없음. 수면 권장.]
```

남궁선은 잠을 자야 했다. 하지만 잠이 오지 않았다.

방을 나왔다. 밤공기가 차가웠다. 본가의 밤은 창검당보다 밝았다. 곳곳에 등불이 켜져 있었고, 순찰이 돌고 있었다.

남궁선은 연무장 방향으로 걸었다. 구경이 목적이 아니었다. 서른여섯의 잔머리가 말했다. 내일 싸울 판을 미리 봐야 한다.

비무가 열리는 연무장은 본가 중앙에 있었다. 낮에 스쳐 지나갔을 때는 사람이 많아서 잘 못 봤다. 밤에는 비어 있었다.

사각형 비무대가 중앙에 놓여 있었다. 높이 두 자, 가로세로 열 보. 밧줄로 경계가 쳐져 있었다.

남궁선은 비무대 위에 올라섰다.

열 보 사방. 좁았다. 돌아다니며 치는 기동전을 하려면 — 공간이 제한된다. 뒤로 빠지면 밧줄에 닿고, 옆으로 돌면 모서리에 몰린다.

```
[비무대 분석. 무료.]
[면적: 10보 × 10보.]
[소재: 석판. 미끄러움 없음.]
[높이: 2자. 떨어지면 장외패.]
[전술 제한: 기동 범위 협소. 장거리 보법 불가.]
[유리한 전투 스타일: 근접 교환. 정면 승부.]
```

정면 승부가 유리한 판. 남궁선에게 불리했다. 남궁선의 장기는 관찰과 타이밍이지, 순수 근접 교환이 아니었다.

'좁은 판에서 이기려면 뭐가 필요해?'

```
[좁은 판에서의 핵심: 선제 타격과 중심 장악.]
[먼저 치고, 상대를 밀어내고, 비무대 중앙을 점거하는 자가 유리합니다.]
[플레이어의 강점(관찰, 패턴 읽기)을 살리려면 — 초반에 후퇴하며 패턴을 읽되,]
[중반 이후 중앙을 탈환하는 전개가 필요합니다.]
[요약: 처음엔 지는 척, 나중에 뒤집기.]
```

남궁선은 피식 웃었다. 처음엔 지는 척, 나중에 뒤집기. 내 장기 아닌가.

비무대에서 내려와 객실로 돌아가는 길에 — 사람과 마주쳤다.

복도 모퉁이에서 나온 사내. 남궁선보다 한 살쯤 많아 보였다. 키가 컸고, 어깨가 넓었다. 도복이 깔끔했고, 허리에 찬 검이 — 좋았다. 검집에 은사(銀絲)가 감겨 있었다.

두 사람이 멈춰 섰다.

사내의 눈이 남궁선을 훑었다. 위에서 아래로. 그리고 허리의 창(蒼)에서 멈췄다. 낡은 검집. 삭은 가죽. 은사 검과 비교하면 거지의 물건이었다.

사내가 말했다.

"방계?"

한마디였다. 하지만 그 한마디에 든 것들 — 시선의 방향, 목소리의 높이, 입꼬리의 각도. 서른여섯의 눈이 전부 읽었다.

멸시가 아니었다. 관심도 아니었다. 확인이었다. 내일 비무대에서 만날 수 있는 상대를 확인하는 눈.

```
[대상 분석. 무료 센서.]
[내공: 측정 범위 상한 초과. 최소 3.0 이상.]
[보행 안정성: 극상. 숨쉬는 것처럼 자연스러운 내공 운용.]
[추정: 본가 상위 제자.]
```

남궁선은 대답했다. "예."

사내가 고개를 끄덕이고 지나갔다. 뒤돌아보지 않았다.

남궁선은 사내의 등을 보았다. 넓고, 곧고, 흔들림이 없는 등. 아버지의 등과 비슷했지만 — 질이 달랐다. 아버지의 등은 무거웠고, 이 사내의 등은 가벼웠다. 무거울 필요가 없는 사람의 등.

'태을아. 저 사람 누구야.'

```
[데이터 부족. 직접적 판별 불가.]
[단, 내공 3.0 이상, 본가 상위 제자, 나이 20세 전후.]
[회귀 기억과 교차 검증 시도 중...]
[일치 항목: 1건.]
[남궁천(南宮天). 본가 최고의 천재.]
[신뢰도: 82%.]
```

남궁선의 발이 멈췄다.

남궁천.

한줄플롯의 마지막 상대. 형이 "나보다 강해"라고 말한 사람. 내공 3.0 이상.

남궁선의 1.4와 남궁천의 3.0. 배가 넘는 차이.

```
[정면 대결 시 승리 확률: 8%.]
```

남궁선은 숫자를 보고 — 웃었다.

'열에 하나도 안 되네.'

```
[그렇습니다.]
```

'근데 거기까지 가야 하잖아. 먼저 1회전부터.'

```
[맞습니다. 한 걸음씩.]
```

남궁선은 객실로 돌아가 누웠다. 눈을 감았다.

내일, 처음으로 본가의 비무대에 선다.
