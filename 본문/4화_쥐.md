# 4화. 쥐

---

담장 바깥은 뒷골목이었다.

방계 별채와 외곽 민가 사이에 낀, 사람 하나 겨우 지나갈 폭의 좁은 길. 벽에 이끼가 끼어 있었고, 바닥에는 마른 흙과 낙엽이 깔려 있었다. 거기에 발자국이 찍혀 있었다. 선명하지는 않았지만, 새벽 이슬에 젖은 흙이 형태를 유지하고 있었다.

남궁선은 발자국을 내려다보았다.

```
[추가 분석: 1.0토큰. 발자국의 보폭, 깊이, 방향을 교차 분석하여 용의자를 특정합니다.]
```

남궁선은 잠시 생각했다. 1.0이면 수련 다섯 시경 분이다. 아까 0.5 쓴 것도 아까운데.

'잠깐. 내가 먼저 해볼게.'

```
[무료 의지입니다. 응원합니다.]
```

남궁선은 쪼그려 앉아 발자국을 살폈다. 서른여섯 해를 잔머리로 버텨온 사내다. 관찰력이 없는 건 아니었다.

발자국의 크기. 자기 발보다 약간 크다. 보폭이 넓다. 뛰지는 않았지만 빠르게 걸었다. 그리고 — 오른발이 약간 안쪽으로 꺾여 있었다. 안짱걸음까지는 아니지만, 오른쪽 무릎이 좋지 않은 사람의 걸음이다.

방계 식구들의 얼굴이 머릿속을 스쳤다.

오른쪽 무릎이 안 좋은 장정. 서른여섯의 기억을 더듬었다. 방계에 그런 사람이 있었나?

...있었다.

이름이 뭐였더라. 방계의 하인 중 하나. 금고 근처를 드나들 수 있는 위치에 있던 놈. 이름이 — 기억이 잡힐 듯 말 듯 했다.

```
[기억 노이즈 감지. 자체 복구 시도 중... 실패.]
[정밀 스캔 없이는 복구 불가합니다.]
```

'닥쳐. 돈 안 써.'

남궁선은 일어서서 발자국을 따라갔다. 뒷골목을 따라 오십 보쯤 걸었을 때, 발자국이 왼쪽으로 꺾였다. 방계 하인들의 숙소가 있는 방향이었다.

```
[센서 데이터: 전방 40보 거리에 인물 1명 감지. 심박 정상. 수면 중으로 추정.]
```

남궁선의 눈이 좁아졌다. 새벽에 금고를 털고 돌아와서 자고 있다? 배짱이 두둑하거나, 아니면 들킬 거라고 생각을 안 한 거다.

하인 숙소 앞에 도착했다. 세 칸짜리 낡은 건물. 문이 셋. 태을이 감지한 인물은 가운데 방이었다.

남궁선은 문 앞에 서서 생각했다.

그냥 들이닥칠까? 아니다. 증거가 없다. 발자국이 여기로 이어진다는 것만으로는 부족하다. 이놈이 "나 산책했을 뿐"이라고 버티면 끝이다.

```
[조언: 방 내부를 스캔하면 금고에서 나온 물건의 흔적을 탐지할 수 있습니다.]
[비용: 0.5토큰.]
```

남궁선은 이를 악물었다. 또 값이다. 하지만 이건 써야 했다. 증거 없이 들어가봤자 소용없다.

'...해.'

```
[결제 완료. 0.5토큰 소모. 잔여: 10.9.]
[스캔 실행 중...]
```

눈앞에 푸른 윤곽이 그려졌다. 방 안의 구조가 벽 너머로 투과되듯 드러났다. 사람 하나가 누워 있었다. 그리고 그 옆에 —

```
[감지: 대상의 베개 밑에서 금속 반응. 형태: 소형 주머니.]
[금고의 자물쇠와 동일한 금속 성분 감지. 열쇠로 추정.]
```

남궁선의 입꼬리가 올라갔다.

금고 열쇠를 베개 밑에 숨겨뒀다고? 이건 —

```
[확률 분석: 대상이 범인일 가능성 94%.]
```

남궁선은 속으로 환산했다. 94... 열에 아홉이 넘는다.

"충분하지."

남궁선은 문을 두드리지 않았다. 대신 돌아섰다. 여기서 직접 잡는 것보다 더 좋은 방법이 있었다.

아버지를 데려오면 된다.

방계의 가장이 직접 확인하면 누구도 부정할 수 없다. 그리고 남궁선이 직접 범인을 잡았다는 사실은 — 아버지의 눈에 망나니가 아닌 다른 무언가로 비칠 것이다.

```
[전략적 판단입니다. 효율 점수 8.7/10.]
```

'점수 매기지 마.'

```
[습관입니다.]
```

---

남궁성을 데려온 것은 반 시경 뒤였다.

아버지는 반신반의하는 얼굴이었다. 망나니 아들이 "범인을 찾았다"고 하니, 믿기 어려운 게 당연했다. 하지만 남궁선의 눈이 진지했고, 남궁성은 아들의 눈을 보는 사람이었다.

문을 열었다.

안에서 자고 있던 사내가 벌떡 일어났다. 사십 대 중반, 마른 체격, 오른쪽 다리를 약간 끌며 일어서는 모습.

"가, 가주님?!"

남궁성이 사내를 내려다보았다. 그리고 남궁선이 말한 대로, 베개 밑을 뒤졌다.

작은 주머니가 나왔다. 안에 열쇠 하나.

남궁성의 얼굴이 굳었다. 남궁선은 아버지의 표정을 보았다. 분노보다 — 실망이 먼저 지나갔다. 믿었던 사람에게 배신당한 자의 표정.

"이게... 이게 뭐냐, 조삼."

조삼. 그 이름이 남궁선의 머릿속에서 무언가를 건드렸다. 조삼. 방계의 하인. 금고 출입이 가능했던 자. 서른여섯의 기억에서 이 이름은 —

```
[경고: 데이터 충돌 발생.]
[억눌려 있던 플레이어의 기억 일부를 강제 해금합니다.]
```

눈앞이 번쩍했다.

1초도 안 되는 찰나, 장면 하나가 뇌리를 관통했다. 영화의 한 컷처럼 스치고 지나갔다.

— 불타는 방계 별채. 안에서 뛰쳐나오는 마교 잔당. 그들을 안내하는 뒷모습. 오른쪽 다리를 약간 절뚝이는 뒷모습. 그 남자가 방계의 배치도를 마교에 넘기고 있었다.

남궁선의 전신에 소름이 돋았다.

```
[기억 조각 복구 완료.]
[복구 내용: '조삼'은 회귀 전 마교 잔당에게 방계의 내부 정보를 유출한 인물입니다.]
[신뢰도: 71%. 기억 왜곡 가능성 있음.]
```

남궁선은 조삼을 바라보았다.

지금은 그저 금고나 터는 좀도둑이다. 하지만 서른여섯의 기억 속에서 이 남자는 — 방계가 불탄 그날 밤, 마교에 문을 열어준 자였다.

열에 일곱은 맞는 기억이다.

남궁선의 눈이 서늘하게 가라앉았다.

조삼은 바닥에 엎드려 용서를 빌고 있었다. 남궁성 앞에서 머리를 조아리며, "마, 마귀가 씌었습니다, 가주님!" 하고 울먹이고 있었다.

남궁성은 한숨을 내쉬며 조삼을 끌어 세웠다. "관가에 넘기겠다."

남궁선은 아무 말도 하지 않았다.

관가에 넘기면 끝이다. 금고 도둑으로 처벌받고, 풀려나고, 그리고 — 십칠 년 뒤에 마교의 앞잡이가 된다.

하지만 지금 "이놈이 나중에 마교에 정보를 판다"고 말할 수는 없었다. 아직 일어나지 않은 일이니까.

```
[조언: 지금은 증거가 없습니다. 감시를 권장합니다.]
```

남궁선은 고개를 끄덕였다. 태을의 말이 맞았다. 지금 할 수 있는 건 — 지켜보는 것뿐이다.

남궁성이 조삼을 끌고 나갔다. 남궁선은 빈 방에 홀로 남았다.

```
[사건 해결 보상 정산.]
[데이터 보너스: +3.0토큰.]
[기억 조각: 1개 복구 완료.]
[현재 잔여 토큰: 13.9.]
```

남궁선은 눈앞의 숫자를 보았다. 13.9. 시작했을 때보다 늘었다.

'값이 불었네.'

```
[축하합니다. 투자 대비 수익률 586%입니다.]
```

남궁선은 피식 웃었다.

'수익... 뭐?'

```
[쉽게 말해, 남는 장사였습니다.]
```

'그건 알겠다.'

남궁선은 방을 나섰다. 아침 해가 마당을 비추고 있었다. 아버지가 조삼을 데리고 가는 뒷모습이 보였다. 넓은 등. 살아 있는 등.

이번에는 지킨다.
