# 13화. 약

---

이틀 뒤, 밤.

청심초가 말랐다. 연한 푸른빛이 빠져 은회색이 되었지만, 만지면 미세한 기운이 손끝에 전해졌다. 살아 있는 약초였다.

남궁선은 방문을 닫고 도기 사발에 물을 부었다. 비망록에 적힌 대로, 약초 무게의 열 배. 정확한 저울은 없으니 손으로 가늠했다. 서른여섯 해를 주루에서 술을 따르며 익힌 감각이 여기서 쓰였다. 액체의 양을 눈으로 맞추는 건 — 술꾼의 기본기였다.

```
[액량 추정: 비망록 기준 대비 오차 약 8%.]
[정밀 가공법(0.5토큰) 기준으로는 오차 15%.]
[...술꾼 경력이 도움이 되는 순간이 올 줄은 몰랐습니다.]
```

'인생에 쓸모없는 경험은 없어.'

```
[술을 마시는 것은 쓸모없었습니다.]
```

'따르는 건 쓸모있잖아.'

감초를 잘게 찢어 넣었다. 비망록에는 분량이 "약초의 삼분의 일"이라고만 되어 있었다. 대충이었다. 하지만 대충으로 충분하다.

청심초를 넣었다. 물 위에 떠올랐다가 천천히 가라앉았다.

냉침. 하룻밤.

남궁선은 사발을 방 구석에 놓고 옷으로 덮었다. 어머니가 아침에 방을 들여다봐도 모를 자리.

'내일 아침이면 되는 거지?'

```
[네. 냉침 8시간 이상.]
[아침에 약액만 걸러서 복용하면 됩니다.]
[예상 효과: 폐부 손상 진행 억제. 기침 빈도 감소. 기맥 일부 소통.]
[완치가 아닙니다. 악화를 멈추는 것입니다.]
```

남궁선은 고개를 끄덕였다. 멈추는 것. 시간을 버는 것. 그 사이에 100값을 모으면 된다.

잠이 오지 않았다. 내일 어머니에게 어떻게 먹일지 생각했다. "약이에요"라고 하면 "어디서 났냐"고 물을 것이다. "산에서 따왔어요"라고 하면 "혼자? 위험하게?"라며 걱정할 것이다. 걱정이 기침을 부른다.

잔머리를 굴렸다.

'태을아. 감초물이 폐에 좋다는 건 일반 상식이야?'

```
[이 시대 기준으로는 — 네. 민간요법 수준의 상식입니다.]
```

'그러면 이렇게 하자. 감초물에 청심초를 섞었는데, 겉보기에 감초물과 다른 점이 있어?'

```
[색상: 감초물은 연한 황색, 청심초 배합 후에도 황색 유지. 차이 없음.]
[향: 감초 향이 지배적. 청심초 향은 미미.]
[결론: 구분이 어렵습니다.]
```

남궁선은 웃었다. 감초물이라고 하면 된다. "목이 칼칼해서 감초물 끓였는데 어머니도 드세요." 약이라는 말을 안 하면 걱정도 안 한다.

```
[......감탄할지 한심해할지 모르겠습니다.]
```

'어머니가 걱정 안 하는 게 제일 좋은 약이야.'

```
[의학적 근거는 없지만, 반론할 수 없습니다.]
```

---

다음 날 아침.

남궁선은 약액을 걸렀다. 옷 한 조각으로 찌꺼기를 거르니, 맑은 황색 액체가 남았다. 감초물과 구분이 안 됐다. 계획대로였다.

사발을 들고 부엌으로 갔다.

어머니가 죽을 끓이고 있었다. 오늘은 기침이 벌써 두 번 나왔다. 남궁선은 애써 표정을 바꾸지 않았다.

"어머니. 감초물 끓였는데 한 잔 드세요. 목에 좋대요."

유설화가 돌아보았다. "네가 끓인 거야?"

"예. 잠이 안 와서 이것저것 해봤어요."

어머니가 사발을 받아들었다. 의심은 없었다. 감초물은 흔한 민간 약이었고, 아들이 끓여왔다는 사실 자체에 마음이 갔다. 한 모금 마셨다.

"...쓰다."

"감초가 원래 좀 써요."

유설화가 인상을 찌푸리면서도 사발을 비웠다. 남궁선은 그 모습을 지켜보았다.

```
[약액 복용 완료.]
[체내 흡수 시작. 효과 발현까지 약 2시간.]
[모니터링을 실행합니다. 무료입니다.]
```

남궁선은 아침 죽을 먹으면서 어머니의 기침 횟수를 속으로 셌다. 한 그릇 먹는 동안 — 세 번. 어제는 다섯 번이었다.

아직 효과인지 우연인지 모른다. 하지만 기다리면 안다.

---

두 시경이 지났다.

연무장에서 오전 수련을 마치고 돌아오는 길에, 태을이 말했다.

```
[유설화 모니터링 중간 보고.]
[기침 빈도: 오전 기준 전일 대비 31% 감소.]
[폐부 염증 지표: 미세 하락.]
[결론: 약효 발현 확인. 유효합니다.]
```

남궁선의 걸음이 멈췄다. 잠시 하늘을 올려다보았다.

된다. 약이 듣는다.

'...앞으로 얼마나 먹여야 해?'

```
[비망록 기준 가공으로는 — 매일 한 잔, 최소 열흘.]
[열흘 후 재스캔하여 경과를 확인해야 합니다.]
[청심초 잔여량: 6일분.]
[4일분이 부족합니다.]
```

남궁선은 입술을 깨물었다. 6일분. 한 번에 딴 양이 그만큼이었다.

'금수계곡에 다시 가야겠네.'

```
[그렇습니다.]
[그리고 — 그때 봤던 사람들이 아직 거기 있을 수도 있습니다.]
```

남궁선은 걸으면서 생각했다. 금수계곡의 검은 옷 무인 둘. 정체불명. 창검당 사람이 아니었다. 이 근처에 올 이유가 없는 자들.

서른여섯의 머리가 굴러갔다.

무인 둘이 산속을 돌아다닌다. 말없이. 훈련된 보법으로. 무엇을 찾고 있는 것이다. 아니면 — 무엇을 지키고 있는 것이다.

뭘 찾지? 뭘 지키지?

남궁선은 발걸음을 돌려 연무장이 아닌 정문 쪽으로 갔다.

창검당 정문 앞에는 작은 장터가 있었다. 영지 주변 마을의 농부와 행상이 아침마다 물건을 펼치는 곳. 남궁선은 거기서 과일 장수 노파를 찾았다.

"할머니. 요즘 북쪽 계곡 쪽에 사람이 다니나요?"

"계곡? 가을이면 나무꾼들이 올라가지."

"나무꾼 말고요. 낯선 사람들. 검은 옷 입은."

노파가 고개를 갸웃거렸다. "글쎄... 나는 못 봤는데. 근데 사냥꾼 막동이가 며칠 전에 뭐라 그랬지. 계곡 상류에 사람 발자국이 많다고. 짐승이 다 도망갔다고 투덜거렸어."

남궁선은 과일 두 개를 사면서 고개를 끄덕였다.

발자국이 많다. 둘이 아니라 여럿이 다닌다는 뜻이다.

다음으로 허 노인을 찾아갔다. 장부 정리 건으로 감사 인사를 하는 척 들어갔다.

"어르신, 혹시 창검당 영지 북쪽에 다른 문파 땅이 있습니까?"

허 노인이 안경 너머로 올려다보았다. "없다. 금수계곡 일대는 전부 남궁세가 영지야. 왜?"

"아닙니다. 그냥 궁금해서요."

허 노인은 의아한 눈으로 남궁선을 보았지만, 더 묻지 않았다. 남궁선은 총무실을 나오면서 정보를 정리했다.

남궁세가 영지 안에, 남궁세가 사람이 아닌 무인들이 돌아다니고 있다.

```
[정리하겠습니다.]
[1. 금수계곡 상류에 무장 인원 복수 활동 중.]
[2. 남궁세가 영지 내부.]
[3. 창검당 및 주변 주민에게 알려지지 않은 존재.]
[결론: 은밀하게 무언가를 하고 있습니다.]
[이 정보를 얻는 데 소모된 토큰: 0.]
[과일 두 개 값만 들었습니다.]
```

'과일값은 아깝지 않아. 어머니 드릴 거니까.'

남궁선은 과일을 품에 넣고 서별채로 걸어갔다.

가야 한다. 청심초를 더 따러. 하지만 이번에는 눈을 크게 뜨고 가야 한다. 저 무인들이 뭘 하는지 — 알아내야 한다.

공짜 기억 조각이 걸려 있다. 하지만 그것보다 — 창검당 코앞에서 정체불명의 무인이 활동하는 건 위험한 일이다.

서른여섯의 기억이 말했다. 방계가 불탔던 건, 이런 작은 징후를 아무도 신경 쓰지 않았기 때문이다.

'이번에는 신경 쓴다.'

```
[사건 추적을 시작하시겠습니까?]
```

'아직. 먼저 어머니 약부터.'

```
[우선순위가 정확합니다.]
```
