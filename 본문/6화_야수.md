# 6화. 야수(夜修)

---

해가 졌다.

남궁선은 저녁밥을 먹고 방에 누웠다. 어머니가 끓인 묽은 미음이었다. 두 그릇 다 비웠다.

몸이 쑤셨다. 반 시경 두 차례의 기초검식 수련이 전부였는데, 열아홉의 몸은 이미 비명을 지르고 있었다. 팔이 떨렸고, 허벅지가 당겼고, 손바닥에 목검 자국이 붉게 남아 있었다.

```
[신체 상태: 근육 피로 67%. 손바닥 표피 손상. 내공 — 여전히 없음.]
[총평: 일반인 기준으로도 체력이 하위입니다. 술 때문입니다.]
```

'옛날 얘기야.'

```
[3일 전입니다.]
```

남궁선은 대답하지 않았다. 맞는 말이니까.

방 안이 어두웠다. 창 너머로 달빛이 들어왔다. 창검당은 잠들어가고 있었다. 하인들의 발소리가 멀어지고, 제자들의 숙소에서 불이 꺼졌다.

남궁선은 일어났다.

'태을아.'

```
[네.]
```

'낮에 연무장에서 말했지. 내 검식이 빨래 방망이라고.'

```
[작대기로 승격시켜 드렸습니다.]
```

'그걸 고칠 수 있어?'

```
[질문이 모호합니다. '고친다'의 범위를 지정해주십시오.
자세 교정인지, 초식 최적화인지, 무공 전체의 재설계인지.]
```

남궁선은 생각했다. 서른여섯의 기억을 더듬었다. 남궁세가 기초검식은 정파 세가의 기본기다. 교과서대로 배우면 교과서만큼 강해진다. 그 이상은 안 된다.

'전체.'

```
[전체요?]
```

'그래. 처음부터 다시 깎아봐. 안 되는 건 뭔데?'

잠시 침묵이 흘렀다. 태을의 응답이 평소보다 늦었다.

```
[......재미있는 주문입니다.]
[남궁세가 기초검식 제1초식부터 제8초식까지의 동작 데이터를 수집했습니다.
오늘 연무장에서 교관과 제자 15명의 시연을 관찰한 데이터 기반입니다.]
[분석 결과: 기초검식에는 불필요한 예비 동작이 초식당 평균 2.3개 포함되어 있습니다.]
[이것을 제거하고 최적 경로로 재설계하면 — 같은 위력을 40% 적은 힘으로 낼 수 있습니다.]
```

남궁선의 눈이 커졌다. 열에 넷을 아낀다?

```
[단, 이것은 무료 분석의 한계입니다. 정밀 리팩토링에는 초식당 0.1토큰이 소모됩니다.]
```

'리팩... 뭐?'

```
[리팩토링. 기존 구조를 허물지 않고 내부를 재설계하는 작업입니다.]
```

남궁선은 머릿속에서 말을 굴렸다. 허물지 않고 속을 갈아 끼운다. 그러니까 — 겉은 같은데 알맹이가 다른 검.

'깎는 거지. 검을 깎는다.'

```
[......리팩토링입니다.]
```

'깎기. 됐고, 0.1이면 반 시경 수련분이잖아. 8초식이면 0.8. 아까운데.'

```
[오늘 벌어들인 사건 보상이 3.0이었습니다.
0.8을 투자하면 수련 효율이 40% 상승합니다.
투자 회수까지 예상 기간: 12일.]
[남는 장사입니다.]
```

남궁선은 입을 다물었다. 이놈의 영업이 또 시작이다. 하지만 — 셈이 맞았다. 효율이 올라가면 같은 시간에 더 많이 적립할 수 있다. 12일이면 본전이고, 그 뒤로는 전부 이득이다.

구두쇠의 머리가 빠르게 돌았다.

'...제1초식만 해. 0.1만.'

```
[1초식만요?]
```

'나머지는 내가 써보고 판단할 거야. 첫 술에 배 안 부른다고.'

```
[합리적입니다. 의외로.]
[결제 완료. 0.1토큰 소모. 잔여: 13.9.]
[기초검식 제1초식 리팩토링 실행.]
```

남궁선의 망막에 푸른 선이 떠올랐다. 이번에는 낮의 윤곽선과 달랐다. 사람 형태의 실루엣이 허공에 서 있었고, 그 실루엣이 천천히 제1초식을 시연했다.

정통 제1초식과 비교하면 — 달랐다. 예비 동작 두 개가 사라져 있었다. 검을 들어 올리는 궤적이 짧아졌고, 허리의 회전이 한 박자 줄었다. 하지만 검 끝이 도달하는 지점은 같았다.

같은 곳을 찌르는데, 길이 짧아졌다.

남궁선은 목검을 들었다. 달빛 아래, 서별채 뒤편의 좁은 공터에서. 푸른 실루엣을 따라 검을 휘둘렀다.

한 번. 어색했다.

두 번. 몸이 거부했다. 열아홉 해 동안 — 비록 대충이지만 — 몸에 밴 습관이 남아 있었다. 예비 동작을 빼면 균형이 흔들렸다.

```
[지지발을 반 촌 앞으로.]
```

세 번째. 균형이 잡혔다. 검이 허공을 갈랐다. 짧고, 빠르고, 쓸데없는 동작이 없는 일격.

남궁선은 멈췄다. 손이 떨리고 있었다. 근육 피로 때문만은 아니었다.

이게 되는구나.

```
[제1초식 리팩토링 적용 완료.]
[발동 속도 — 정통 대비 18% 향상.]
[소모 체력 — 정통 대비 23% 절감.]
[외형상 차이 — 교관급 관찰자가 아니면 알아채기 어렵습니다.]
```

남궁선은 다시 휘둘렀다. 네 번, 다섯 번, 여섯 번. 반복할수록 몸이 익어갔다. 서른여섯의 잔머리가 깎인 검의 의미를 이해했고, 열아홉의 몸이 그것을 새겼다.

한 시경이 흘렀다.

```
[수련 적립: +0.2토큰. 잔여: 14.1.]
[참고: 리팩토링된 초식으로 수련 시 적립 효율이 기존 대비 상승합니다.]
[투자 대비 수익 발생 시점이 당초 12일에서 9일로 앞당겨졌습니다.]
```

남궁선은 피식 웃었다. 0.1을 넣었는데 벌써 0.2가 돌아왔다.

'나머지 7초식도 깎아야겠는데.'

```
[영업 성공입니다.]
```

'영업이 아니라 내가 판단한 거야.'

```
[네. 물론입니다.]
```

남궁선은 목검을 내려놓았다. 손바닥이 쓰라렸다. 내일이면 물집이 잡힐 것이다.

서별채로 돌아가는 길에, 본관 쪽에서 불빛이 보였다. 아버지의 집무실이었다. 남궁성은 아직 자지 않고 있었다. 조삼의 처리, 금고의 정리, 관가와의 연락. 당주의 일은 끝이 없다.

남궁선은 불빛을 바라보았다.

넓은 등. 살아 있는 등. 저 등이 칼에 베이는 걸 봤다. 서른여섯의 눈으로, 바로 눈앞에서.

'...반드시.'

남궁선은 방으로 돌아가 누웠다. 눈을 감자마자 잠이 왔다. 피로가 의지를 이기는 나이었다.

달빛이 서별채 지붕 위를 비추고 있었다.
