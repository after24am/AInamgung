# 9화. 기(氣)

---

수련 스무날째 밤.

남궁선은 서별채 뒤편에 앉아 있었다. 목검이 아니라 맨몸이었다. 오늘은 검을 내려놓았다.

```
[내공 순환 가이드를 실행합니다. 비용: 0.3토큰.]
[호흡법은 남궁세가 기본 호흡법 '창궁토납(蒼穹吐納)'을 기반으로 합니다.]
[교관과 제자들의 호흡 데이터 20일치를 수집·분석했습니다.]
[리팩토링된 호흡 경로를 제시합니다.]
```

남궁선은 눈을 감았다. 0.3값. 아까웠지만 필요했다. 검을 아무리 휘둘러도 내공이 없으면 검잡이에 불과했다. 무림에서 내공 없는 검은 날 없는 칼과 같았다.

```
[결제 완료. 0.3토큰 소모. 잔여: 15.0.]
[가이드 시작.]
[들이쉽니다. 코로. 배를 부풀리십시오.]
```

남궁선은 숨을 들이쉬었다.

```
[뱉습니다. 천천히. 배를 집어넣으면서.]
[다시 들이쉽니다.]
[지금 기운이 단전 아래에 모이는 느낌이 있어야 합니다.]
```

없었다.

```
[없습니까?]
```

'없어.'

```
[정상입니다. 처음부터 되면 천재이고, 남궁갑님은 천재가 아닙니다.]
[다시.]
```

남궁선은 이를 악물었다. 다시 들이쉬고, 내쉬고, 들이쉬었다.

서른여섯의 기억을 더듬었다. 내공 순환. 회귀 전에도 결국 해내지 못한 것이었다. 기초는 배웠지만 진지하게 시도한 적이 없었고, 서른이 넘어서는 이미 늦었다고 포기했다.

지금은 열아홉이다. 늦지 않았다. 아직.

```
[호흡 패턴 보정: 들이쉬는 속도를 0.5박 늦추십시오.]
[배의 팽창 타이밍을 흡기 시작 후 1박 뒤로.]
[이렇게 하면 횡격막의 운동이 단전 방향으로 압력을 만듭니다.]
```

남궁선은 따라했다. 느리게 들이쉬고, 한 박 뒤에 배를 부풀렸다.

세 번째 호흡.

다섯 번째 호흡.

일곱 번째 —

배 아래에서 무언가가 움직였다. 뜨겁지도, 차갑지도 않은 것. 물이 흐르는 것 같기도 하고, 바람이 스치는 것 같기도 한 미세한 감각.

남궁선의 눈이 떠졌다.

'이거... 뭐야.'

```
[축하합니다.]
[단전 내 기의 흐름 0.01 감지.]
[첫 내공 순환 성공입니다.]
```

남궁선은 배 위에 손을 얹었다. 아무것도 느껴지지 않았다. 방금 그 감각은 이미 사라져 있었다. 손가락 사이로 빠져나간 물처럼.

'사라졌는데?'

```
[정상입니다. 첫 순환은 찰나에 불과합니다.]
[비유하자면, 마른 수로에 물 한 방울을 떨어뜨린 것과 같습니다.]
[수로가 젖어야 물이 흐릅니다. 반복이 필요합니다.]
```

'몇 번?'

```
[수백 번.]
```

남궁선은 한숨을 내쉬었다. 하지만 — 웃고 있었다.

됐다. 한 방울이라도 흘렀다. 마른 수로에 물이 닿았다. 이제 반복하면 된다.

```
[참고: 방금 순환에서 적립된 토큰은 0.001입니다.]
[기존 검술 수련 적립의 1/100 수준.]
```

'...그래도 0은 아니잖아.'

```
[긍정적 사고입니다. 응원합니다.]
[하지만 이 속도로는 100토큰 적립에 약 27년이 걸립니다.]
```

'내공이 늘면 적립도 빨라진다며.'

```
[맞습니다. 초반이 가장 느립니다.]
[첫 순환 성공 후 1개월이 고비입니다. 이 기간에 수로가 열리면 적립 효율이 급상승합니다.]
[포기하면 수로가 다시 마릅니다.]
```

남궁선은 다시 눈을 감았다. 들이쉬고, 내쉬고.

한 시경을 앉아 있었다. 그 사이에 세 번 더 감각이 왔다. 찰나의 흐름. 잡히지 않는 물줄기. 하지만 횟수가 늘 때마다 감각이 조금씩 또렷해졌다.

```
[수련 종료. 금일 내공 순환 성공 횟수: 4회.]
[적립: +0.25토큰 (검술 0.2 + 내공 0.05). 잔여: 15.25.]
[내공 순환 적립이 검술 적립에 합산되기 시작했습니다.]
[이것이 복리의 시작입니다.]
```

'또 그 말이야.'

```
[중요한 말이니까 반복합니다.]
```

남궁선은 일어섰다. 다리가 저렸다. 한 시경을 가부좌로 앉아 있었으니 당연했다.

달이 기울어져 있었다. 자정이 넘은 시간. 내일도 아침에 연무장이다.

방으로 돌아가려는데 — 발소리가 들렸다.

형이 아니었다. 남궁재의 발소리는 가볍고 날카로웠다. 이 발소리는 무겁고 느렸다.

남궁선이 돌아보았다.

아버지였다.

남궁성이 서별채 모퉁이에 서 있었다. 잠옷 차림이었다. 잠을 자다가 나온 것 같았다. 아니 — 잠을 못 자고 있었던 것이다. 당주의 얼굴에는 피로가 짙게 내려앉아 있었다.

"...아버지."

남궁성은 아들을 바라보았다. 한밤에 땀을 흘리며 앉아 있던 아들을. 목검도 없이, 가부좌를 틀고, 호흡을 하고 있던 아들을.

"내공 순환이냐."

남궁선은 고개를 끄덕였다.

남궁성의 표정이 움직였다. 미세하게. 이 사람은 감정을 잘 드러내지 않았다. 서른여섯의 기억에서도 아버지가 운 건 — 죽기 직전 딱 한 번이었다.

"했느냐?"

"...조금요. 한 방울 정도."

남궁성이 한 발 다가왔다. 그리고 남궁선의 머리 위에 손을 얹었다. 거칠고 두꺼운 손이었다. 검을 삼십 년 넘게 잡은 손이었다.

아무 말이 없었다. 그냥 손을 얹고 서 있었다.

```
[센서 데이터: 남궁성. 심박수 상승. 체온 상승. 손의 미세한 떨림 감지.]
```

남궁선은 아버지의 손 아래서 움직이지 않았다.

"...자거라."

남궁성이 손을 거두고 돌아갔다. 느린 발소리가 멀어졌다. 남궁선은 아버지의 등이 어둠 속으로 사라질 때까지 서 있었다.

```
[무료 조언입니다.]
[자십시오.]
```

'...응.'

남궁선은 방으로 돌아가 누웠다. 눈을 감았지만 잠이 바로 오지 않았다. 아버지의 손이 머리 위에 남아 있는 것 같았다.

단전 아래에서 아까의 감각이 아주 희미하게 — 한 번 더 흘렀다.
