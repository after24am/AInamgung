# 22화. 전야

---

남궁혁 방문까지 3일.

창궁무애검법 제1초식 완성도: 74%.

목표를 넘었다. 남궁선은 달빛 아래에서 검을 거두며 숨을 내쉬었다. 창(蒼)의 검신이 은빛으로 빛나고 있었다.

```
[제1초식 완성도 74%. 목표 초과 달성.]
[곡선 전환점 기 집중 기법 — 안정적.]
[교관급 관찰자 앞에서 시연 가능 수준입니다.]
[잔여 토큰: 18.2.]
```

한 달 동안 토큰이 늘었다. 수련 적립이 꾸준히 쌓인 덕이었다. 내공은 0.9. 검 증폭 포함 1.4. 두 달 반 전의 0과 비교하면 — 하늘과 땅이었다.

하지만 남궁선은 다른 것을 생각하고 있었다.

남궁혁. 본가 무술 사범. 서른여섯의 기억에서 이 사람은 어떤 사람이었나.

기억이 흐릿했다. 하지만 몇 가지는 남아 있었다.

원칙주의자. 정통에 대한 고집이 강하다. 사파 무공이라고 판단하면 가차 없다. 하지만 — 실력을 인정하면 고개를 숙이는 사람이기도 했다.

문제는 "인정"의 기준이 높다는 것이었다.

```
[회귀 기억 기반 남궁혁 프로필.]
[신뢰도: 61%. 부분 왜곡 가능.]
[성격: 원칙주의. 정통 고수. 방계에 대한 편견은 적으나, 정통에서 벗어난 무공에 대한 거부감 강함.]
[약점: 효율적인 것을 보면 — 거부감 속에서도 무의식적으로 인정하는 경향.]
[무료 정보입니다.]
```

효율적인 것을 인정하는 경향. 남궁선은 이 한 줄을 잡았다.

남궁혁은 정통을 고집하지만, 검이 효율적이면 무시하지 못한다. 그러니까 — 남궁혁 앞에서 보여줘야 하는 건 "정통이냐 아니냐"가 아니라 "이 검이 효율적이다"는 사실이다.

'태을아. 무료 질문.'

```
[네.]
```

'남궁혁 앞에서 창궁무애검법을 보여줄 때, 어떤 장면이 가장 효과적이야?'

```
[질문이 모호합니다. 구체적으로?]
```

'남궁혁이 보고 "이건 뭐지?"라고 생각하게 만드는 상황. 의심이 아니라 호기심이 먼저 드는.'

```
[분석합니다. 무료 범위.]
[최적 시나리오: 남궁혁이 직접 교정하려 할 때, 창궁무애검법의 동작이 교정을 거부하는 상황.]
[정통 자세로 고치려 하면 오히려 검이 느려지는 것을 남궁혁 본인이 느끼는 것.]
[무인은 — 자기 손으로 확인한 것을 부정하지 못합니다.]
```

남궁선은 고개를 끄덕였다. 그거다. 남궁혁이 직접 교정해보고, 교정하면 오히려 안 된다는 걸 몸으로 느끼게 하면 — 설명보다 강하다.

하지만 그러려면 남궁혁이 남궁선의 검을 교정하려는 상황이 만들어져야 한다.

어떻게?

서른여섯의 잔머리가 돌았다.

남궁혁은 "점검"을 하러 온다. 위탁생을 본다. 위탁생을 보면서 방계 제자도 볼 것이다. 조 사범이 안내할 테고, 연무장에서 시연을 시킬 것이다.

그때 남궁선이 — 일부러 반쯤만 보여주면?

창궁무애검법의 곡선 동작을 기초검식에 살짝 섞는 것이다. 정통에서 벗어난 동작이 한두 개. 남궁혁의 눈에 띄지 않을 수 없다. 그러면 남궁혁이 다가와서 "이 동작은 뭐냐"고 묻고, 교정하려 할 것이다.

그 순간이 남궁선의 무대다.

"이건 창검당의 검법입니다. 교정하시면 오히려 느려집니다."

말로는 안 된다. 직접 보여줘야 한다.

```
[시나리오 시뮬레이션.]
[1단계: 연무장에서 기초검식에 창궁무애검법 동작을 1~2개 섞음.]
[2단계: 남궁혁이 발견하고 교정 시도.]
[3단계: 교정된 동작과 원래 동작을 비교 시연.]
[4단계: 남궁혁이 차이를 체감.]
[성공 조건: 남궁혁의 반응이 "의심"에서 "관심"으로 전환.]
[실패 조건: 남궁혁이 즉석에서 "사파"로 단정.]
[성공 확률 — 데이터 부족으로 정밀 연산 불가.]
[단순 추정: 55%.]
```

55%. 열에 다섯 반. 동전 던지기보다 약간 나은 정도.

```
[권장: 안전한 경로는 정통만 보여주고 넘어가는 것입니다.]
[위험한 경로는 플레이어가 구상한 시나리오입니다.]
```

남궁선은 검을 바라보았다. 창(蒼)의 낡은 검집.

안전한 길을 가면 — 남궁혁은 "방계에 쓸만한 놈이 하나 있군" 정도로 보고 돌아간다. 그리고 다음에 올 때 리팩토링 검식이 더 발전해 있으면 — 그때는 감출 수 없다. 의심은 미뤄지지만 사라지지 않는다.

위험한 길을 가면 — 지금 판을 벌인다. 성공하면 남궁혁을 아군으로 만들 수 있다. 실패하면 사파 의심.

'...55%면 열에 다섯 반이지.'

```
[그렇습니다.]
```

'열에 다섯 반이면 — 나쁘지 않아. 이 정도 판은 벌여본 적 있거든.'

```
[어디서요?]
```

'도박장에서.'

```
[경험의 출처가 또 처참합니다.]
```

남궁선은 웃었다. 결정했다. 위험한 길.

---

남궁혁 방문 전야.

남궁선은 아버지를 찾아갔다.

"아버지. 내일 남궁혁 사범이 오면 — 제가 창궁무애검법을 보여드려도 되겠습니까?"

남궁성이 아들을 바라보았다.

"보여주겠다고?"

"예. 숨기면 나중에 더 복잡해집니다."

남궁성은 한참 생각했다. 이 사람은 직선의 사람이다. 아들의 잔머리가 정확히 뭘 노리는지는 모르지만 — 숨기는 것보다 보여주는 게 곧은 길이라는 건 이해했다.

"...네 판단대로 해라."

남궁선은 고개를 숙였다. 그리고 한 가지 더.

"아버지. 내일 남궁혁 사범이 제 검을 보고 '이게 뭐냐'고 물으면 — 아버지가 대답해주십시오."

"뭐라고?"

"창검당의 비장 검결을 아들에게 전수했다고."

남궁성의 눈이 미세하게 흔들렸다. 거짓말이 아니었다. 실제로 검결을 줬다. 하지만 아들이 원하는 건 — 당주의 권위로 검법의 정당성을 보증하는 것이었다.

남궁혁이 아무리 정통을 고집해도, 당주가 직접 전수했다고 말하면 "사파"라는 딱지를 붙이기 어렵다.

"...네놈."

"예."

"언제부터 이렇게 머리를 굴리는 놈이 됐느냐."

"술을 끊으니까 머리가 돌아갑니다."

남궁성이 한숨을 내쉬었다. 하지만 입꼬리가 올라간 한숨이었다.

"알겠다. 내일 내가 말하마."

남궁선은 방을 나와 서별채로 돌아갔다.

```
[전략 배치 완료.]
[1. 연무장에서 미끼 노출.]
[2. 남궁혁의 교정 유도.]
[3. 비교 시연으로 효율 입증.]
[4. 당주의 권위로 정당성 보증.]
[실패 시 백업: 당주 보증으로 최소한 "사파" 딱지는 방지.]
```

남궁선은 서별채 뒤편에 서서 창(蒼)을 뽑았다. 달빛 아래서 마지막 수련.

제1초식을 열 번 반복했다. 곡선이 흔들리지 않았다. 74%지만 — 보여줄 수 있는 74%였다.

열한 번째에 검을 거두었다.

내일이다.
