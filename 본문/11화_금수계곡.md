# 11화. 금수계곡

---

열흘이 지났다.

내공은 0.5를 넘었다. 태을이 제시한 최소 기준이었다. 기초검식은 리팩토링 이후 몸에 완전히 배어 있었고, 형과의 대련에서 23초를 버텼다. 아직 이기지는 못하지만, 형의 제4초식까지 보았다.

그리고 어머니의 기침은 — 또 한 차례 심해졌다.

새벽. 해가 뜨기 전에 남궁선은 서별채를 나섰다.

아무에게도 말하지 않았다. 연무장에 빠지는 건 조 사범에게 혼날 일이었지만, 어차피 망나니 전력이 있다. 하루 빠져도 "또 시작이네" 정도로 넘어갈 것이다.

```
[출발 전 체크리스트.]
[내공: 0.52. 최소 기준 충족.]
[장비: 목검 1자루. 도시락(어머니가 싸준 주먹밥 3개). 밧줄 10자.]
[토큰 잔여: 17.1.]
[목표: 금수계곡 상류, 절벽 아래 바위틈. 청심초 채취.]
[기억 신뢰도: 41%.]
[날씨: 맑음. 오후에 흐려질 가능성 있음.]
```

'출발한다.'

```
[행운을 빕니다. 유료 서비스는 아닙니다.]
```

---

금수계곡은 창검당 영지 북쪽 끝에 있었다. 반 시경을 걸었다. 남궁선의 체력은 한 달 반 전과 달랐다. 산길을 걸어도 숨이 크게 차지 않았다.

계곡이 보이기 시작했다. 물소리가 들렸다. 바위 사이로 맑은 물이 흐르고 있었고, 양쪽으로 가파른 절벽이 솟아 있었다.

남궁선은 계곡을 따라 올라갔다. 상류로 갈수록 길이 좁아졌고, 바위가 커졌다.

```
[환경 스캔. 무료 센서 데이터.]
[전방 200보 지점에서 계곡이 두 갈래로 나뉩니다.]
[왼쪽: 완만한 경사. 오른쪽: 급경사, 절벽 지형.]
[기억 데이터 기반 — 오른쪽입니다.]
```

'41%짜리 기억이지.'

```
[그래도 왼쪽보다는 오른쪽입니다. 절벽 아래 바위틈이라는 조건에 부합하니까.]
```

남궁선은 오른쪽으로 갔다. 바위를 타고 올라야 했다. 손으로 바위를 잡고 발을 디뎠다. 이끼가 끼어 미끄러웠다. 내공이 있었다면 가볍게 뛰어올랐을 거리를 — 기어서 올라갔다.

```
[자세 보정: 왼발을 3촌 위 홈에 디디십시오. 거기.]
[오른손은 저 튀어나온 바위를 잡으십시오.]
[무료 가이드입니다. 절벽에서 떨어지면 토큰 수입이 끊기니까.]
```

'걱정해주는 거야 값 걱정이야.'

```
[둘 다입니다.]
```

한 시경을 올라갔다. 팔이 떨리고 손바닥에서 피가 났다. 굳은살 위로 새 상처가 났다. 하지만 멈추지 않았다.

그리고 — 절벽 중턱에 바위틈이 보였다.

사람 하나가 겨우 들어갈 크기의 틈. 안쪽이 그늘져 있었고, 축축한 공기가 흘러나왔다.

```
[환경 분석: 바위틈 내부 습도 높음. 이끼와 초본류 생장 조건 충족.]
[센서 범위 내에서 식물 반응 감지.]
[확인하려면 들어가셔야 합니다.]
```

남궁선은 밧줄을 바위에 묶고 틈 안으로 들어갔다. 어두웠다. 머리 위로 바위가 낮게 내려와 허리를 숙여야 했다. 손바닥의 피가 바위에 묻었다.

3보. 5보. 8보.

틈이 조금 넓어졌다. 그리고 — 빛이 있었다. 바위 틈 사이로 한 줄기 햇빛이 들어오고 있었다. 그 빛 아래, 축축한 바위 표면에 풀이 자라고 있었다.

손바닥만 한 잎. 연한 푸른빛. 이슬이 맺혀 있었다.

```
[식물 스캔 실행. 비용: 0.3토큰.]
```

'해.'

```
[결제 완료. 잔여: 16.8.]
[스캔 중...]
[판별 완료.]
[청심초(淸心草). 일치율 87%.]
[상태: 성장 중기. 채취 가능. 약효 — 유효.]
```

남궁선의 손이 떨렸다. 87%. 열에 아홉 가까이. 이건 진짜다.

41%짜리 기억으로 찾았다.

남궁선은 조심스럽게 청심초에 손을 뻗었다. 뿌리째 뽑으면 안 된다. 서른여섯의 기억 어딘가에서 — 영약은 뿌리를 남겨야 다시 자란다는 지식이 떠올랐다. 대충 살았어도 귀동냥은 했다.

줄기 아래를 잡고 조심스럽게 꺾었다. 잎과 줄기가 손에 왔다. 차가웠다. 그리고 미세하게 — 기운이 느껴졌다. 단전에서 반응이 왔다. 이것이 영약인지는 몰라도, 기를 품은 풀이라는 건 몸이 알았다.

```
[청심초 채취 완료.]
[주의: 직접 복용하지 마십시오. 가공 없이 복용하면 약효의 40%가 손실됩니다.]
[최적 가공법을 안내하려면 0.5토큰이 필요합니다.]
```

'그건 나중에. 일단 가지고 돌아가야지.'

남궁선은 청심초를 주먹밥 보자기에 감쌌다. 빛에 약할 수 있으니까.

바위틈을 빠져나왔다. 절벽 위로 햇빛이 쏟아지고 있었다. 아침에 출발해서 이제 해가 중천이었다.

밧줄을 잡고 내려가려는 순간 — 태을이 말했다.

```
[경고.]
[전방 하단 30보 지점에 인물 2명 감지.]
[심박수 높음. 이동 중. 방향 — 이쪽을 향하고 있습니다.]
```

남궁선의 몸이 굳었다.

'누구야.'

```
[데이터 부족. 창검당 인원이 아닙니다.]
[복장 불명. 무장 여부 불명.]
[30보. 접근 중.]
```

남궁선은 절벽에 매달린 채 아래를 내려다보았다. 계곡 아래, 바위 사이로 두 사람이 걸어오고 있었다. 검은 옷. 얼굴을 가린 건 아니었지만, 이 산속에 올 이유가 있는 사람들이 아니었다.

```
[추가 분석: 보행 패턴 — 무인. 기본 이상의 보법 훈련을 받은 자들.]
[위험도: 현재 플레이어 전투력 기준 — 상대 불가.]
```

남궁선은 숨을 죽였다. 밧줄을 타고 내려가면 눈에 띈다. 바위틈 안으로 다시 들어가면 — 갇힌다.

'어떡하지.'

```
[선택지.]
[A. 바위틈에 숨어서 지나가기를 기다린다. 발각 확률 23%.]
[B. 밧줄로 내려가 반대 방향으로 달린다. 추격당할 확률 71%.]
[C. 동결. 절벽에 매달린 채 움직이지 않는다. 발각 확률 — 대상이 위를 올려다보느냐에 달림.]
```

남궁선은 바위틈을 돌아보았다. 그리고 밧줄을 봤다.

밧줄을 풀었다. 바위틈 안으로 끌어당겨 흔적을 없앴다. 그리고 틈 안으로 몸을 밀어넣었다.

어둠 속에서 숨을 죽였다.

두 사람의 발소리가 아래에서 들렸다. 바위를 밟는 소리. 물을 건너는 소리. 대화는 없었다. 말없이 걷는 두 사람. 훈련된 자들의 침묵이었다.

```
[대상 통과 중. 거리 15보... 10보... 5보...]
[직하방 통과.]
```

남궁선은 숨을 참았다. 심장이 귀 안에서 뛰었다.

```
[통과 완료. 대상 이동 방향 — 상류. 멀어지고 있습니다.]
[발각되지 않았습니다.]
```

남궁선은 한참 뒤에야 숨을 내쉬었다. 등에 땀이 흠뻑 젖어 있었다.

'...누구야 저 사람들.'

```
[불명. 데이터 부족.]
[다만 — 이 계곡에 무장한 자가 둘씩 다니는 것은 정상이 아닙니다.]
[기억 데이터와 교차 검증 시도 중... 일치 항목 없음.]
[새로운 변수입니다.]
```

남궁선은 바위틈에서 나왔다. 조심스럽게 절벽을 내려갔다. 반대 방향으로, 천천히.

보자기 안의 청심초가 품 안에서 차갑게 느껴졌다.

어머니의 약은 찾았다.

하지만 — 이 계곡에 있어서는 안 될 자들이 있었다.

```
[사건 감지. 조사 여부는 플레이어의 판단에 맡깁니다.]
[참고: 사건을 해결하면 기억 조각 보상이 있습니다.]
[무료 정보입니다.]
```

남궁선은 산을 내려가며 생각했다. 기억 조각. 공짜. 하지만 지금은 어머니의 약이 먼저다.

'돌아간다.'

```
[현명한 판단입니다.]
```

남궁선은 걸음을 빨리했다. 창검당을 향해.
