# 29화. 전야(2)

---

비무 첫째 날이 끝났다.

남궁선은 객실로 돌아와 누웠다. 천장을 바라보았다. 몸이 무거웠다. 세 경기. 특히 3회전의 오천전이 몸에 남아 있었다. 손목이 욱신거렸다.

```
[비무 첫째 날 결산.]
[전적: 3전 3승. 4강 진출.]
[총 토큰 소모: 3.8. 잔여: 14.0.]
[신체 상태: 오른손 손목 경미 충격(회복 중). 전신 근피로 중간.]
[내일 가용 토큰: 14.0. 충분합니다.]
[내일 컨디션 예상: 80%. 손목 회복 여부에 따라 변동.]
```

남궁선은 손목을 돌려봤다. 아팠지만 움직였다. 내일 아침이면 괜찮을 것이다. 아마.

방문이 열렸다. 아버지였다.

남궁성이 들어와 남궁선 곁에 앉았다. 말이 없었다. 약상자를 꺼내 남궁선의 손목에 약을 발랐다. 투박한 손이었다. 하지만 약 바르는 손길은 조심스러웠다.

"아프냐."

"괜찮습니다."

"거짓말하지 마라."

"...조금 아픕니다."

남궁성이 약을 다 바르고 헝겊으로 감았다. 일어서며 말했다.

"내일 — 무리하지 마라."

남궁선은 대답하지 않았다. 무리하지 않으면 이길 수 없다. 아버지도 알고 있었다.

남궁성이 문 앞에서 멈췄다. 등을 보이고.

"오늘 잘 싸웠다."

문이 닫혔다.

남궁선은 천장을 보며 웃었다. 아버지가 "잘 싸웠다"라고 한 건 — 기억에 처음이었다.

---

잠이 오지 않았다.

남궁선은 일어나 방을 나왔다. 어젯밤과 같은 행동이었다. 하지만 오늘은 비무대를 보러 가는 게 아니었다.

정보가 필요했다.

남궁천. 내공 3.0 이상. 본가 최고의 천재. 오늘 비무에서 남궁천도 4강에 올랐다. 남궁선은 3회전 뒤에 쉬면서 남궁천의 경기를 봤다.

3경기 전부 — 30초 안에 끝났다. 상대가 약해서가 아니었다. 남궁천이 강해서 끝난 것이었다. 검이 한 번 움직이면 상대가 뒤로 밀려나고, 두 번 움직이면 검이 떨어지고, 세 번 움직이면 끝이었다.

남궁선은 그 세 경기를 전부 봤지만 — 배운 게 없었다.

```
[남궁천 전투 데이터.]
[3경기 분석.]
[검식: 본가 상급 검법. 명칭 불명.]
[특징: 모든 동작이 최적화. 낭비 제로.]
[약점: 감지 안 됨.]
[패턴: 감지 안 됨. 매 경기 다른 방식.]
[승률 재계산: 8% → ...... 8%.]
[변동 없습니다. 추가 데이터로도 약점을 찾지 못했습니다.]
```

8%. 변동 없음. 남궁선은 코웃음을 쳤다.

'세 경기를 봐도 8%?'

```
[아니, 세 경기를 봐서 8%가 유지된 겁니다.]
[못 봤으면 5%로 떨어졌을 겁니다.]
```

남궁선은 복도를 걸었다. 밤공기가 차가웠다.

서른여섯의 잔머리가 돌았다. 8%. 정상적인 방법으로는 안 된다. 정상적이지 않은 방법을 찾아야 한다.

정상적이지 않은 방법. 남궁선은 자기가 가진 것을 세었다.

내공: 0.9. 안 된다.
검식: 리팩토링 8초식 + 창궁무애검법 3초식. 오천한테는 먹혔지만 남궁천한테는 — 수준이 다르다.
토큰: 14.0. 넉넉하다. 하지만 토큰으로 내공이 오르지는 않는다.
정보: 남궁천의 약점 없음.

'없다. 뭘 해도 안 되네.'

```
[정면 대결이 안 되는 건 맞습니다.]
[하지만 — 비무에는 규칙이 있습니다.]
```

남궁선은 멈춰 섰다. 규칙.

'규칙이 뭐였지.'

```
[비무 규칙 — 허 노인 제공 + 본가 공시 기준.]
[유효타 3회 또는 장외 시 패배.]
[무기 이탈 시 패배.]
[항복 가능.]
[시간 제한: 없음.]
```

남궁선은 하나하나 씹었다.

유효타 3회. 남궁천에게 유효타를 3회 넣는 건 — 불가능에 가깝다. 1회도 어렵다.
무기 이탈. 남궁천의 손에서 목검을 떨어뜨리는 것. 내공 3.0의 악력. 불가능.
장외. 남궁천을 비무대 밖으로 밀어내는 것.

남궁선의 눈이 빛났다.

장외.

남궁천을 이기는 게 아니라 — 밀어내는 것.

'태을아. 무료 질문.'

```
[네.]
```

'남궁천을 장외시키는 게 유효타 3회보다 확률이 높아?'

```
[계산합니다.]
[유효타 3회 승리 확률: 3%.]
[장외 승리 확률: ......12%.]
[높습니다. 4배.]
```

12%. 8%보다 올랐다. 3%보다는 훨씬 높다.

'왜 장외가 더 높아?'

```
[유효타는 남궁천의 수비를 뚫어야 합니다. 3번.]
[장외는 남궁천을 한 번만 비무대 가장자리로 몰면 됩니다.]
[실력으로 이기는 게 아니라 — 위치를 이기는 것.]
[비무대는 10보 사방. 좁습니다.]
[좁은 판에서 위치를 다투는 건 — 순수 실력 차이와 다른 게임입니다.]
```

위치를 이기는 것. 남궁선의 서른여섯이 반응했다. 이건 — 도박판의 논리였다.

도박에서 이기는 법. 패가 나쁘면 게임을 바꿔라.

검으로 이길 수 없으면 — 검으로 이기는 게임을 하지 마라. 위치로 이기는 게임으로 바꿔라.

'좋아. 구체적으로 어떻게 해?'

```
[시뮬레이션 — 장외 유도 시나리오. 1.0토큰.]
```

'비싸다.'

```
[내일의 비무를 위한 투자입니다.]
[14.0 중 1.0. 아직 13.0 남습니다.]
```

'...해.'

```
[시뮬레이션 실행.]
[전제: 남궁천은 비무대 중앙을 점거하는 스타일.]
[강한 상대일수록 중앙에서 움직이지 않습니다. 중앙이 유리하니까.]
[남궁천을 중앙에서 끌어내려면 — 미끼가 필요합니다.]
[미끼: 남궁선이 비무대 가장자리에 서는 것.]
[강한 상대가 약한 상대를 가장자리로 몬다 → 당연한 전개.]
[하지만 가장자리에 서면 — 양쪽 모두 가장자리에 가깝습니다.]
[이때 남궁선이 급격하게 방향을 틀어 남궁천의 옆이나 뒤로 빠지면 —]
[남궁천이 관성으로 가장자리를 향하게 됩니다.]
[남궁천이 가장자리에 선 순간, 전력으로 밀어내는 것.]
```

남궁선은 고개를 끄덕였다. 이론은 됐다. 하지만.

'남궁천이 그렇게 쉽게 걸릴까?'

```
[안 걸립니다. 아마.]
[하지만 — 세 번, 네 번 시도하면?]
[남궁천도 사람입니다. 관성이 있습니다.]
[그리고 남궁선이 계속 가장자리에서 싸우면 —]
[남궁천은 "이 놈은 뒤로 밀리는 습관이 있다"고 판단합니다.]
[습관이라고 판단한 순간 — 대비가 느슨해집니다.]
[그때가 진짜 타이밍입니다.]
```

가짜를 세 번 보여주고, 네 번째에 진짜를 넣는다.

남궁선은 웃었다. 이건 도박이 아니라 — 사기에 가까웠다.

'확률이 12%라고 했잖아. 세 번 네 번 시도할 체력이 되나?'

```
[체력은 됩니다. 토큰도 됩니다.]
[하지만 남궁천이 "세 번 네 번 시도"를 허용하는지가 문제입니다.]
[그 전에 유효타 3회로 질 수 있습니다.]
[그래서 12%입니다.]
```

12%. 열에 하나. 남궁천과의 정면 대결 8%보다는 낫다. 하지만 여전히 — 열에 하나.

남궁선은 객실로 돌아가며 생각했다.

열에 하나. 도박판에서 열에 하나는 — 안 나오는 숫자가 아니다. 나올 때가 있다. 문제는 그 "한 번"이 내일이 되느냐는 것이다.

```
[추가 분석 — 승률 향상 요인.]
[1. 남궁천이 남궁선을 얕보는 경우: +3% (총 15%)]
[2. 남궁선이 예상 밖의 패턴을 추가하는 경우: +2% (총 14%)]
[3. 남궁천의 컨디션이 평소보다 낮은 경우: +4% (총 16%)]
[현실적 최대: 약 15%. 기적적 최대: 약 20%.]
```

'남궁천이 날 얕볼까?'

```
[오늘 3회전까지의 경기를 남궁천이 봤다면 — 얕보지 않을 겁니다.]
[하지만 "이기겠다"는 확신은 있을 겁니다.]
[확신과 방심 사이의 간격은 — 좁습니다.]
```

남궁선은 객실에 들어가 누웠다. 눈을 감았다.

내일. 남궁천.

열에 하나를 당기러 간다.

---

잠들기 직전, 복도에서 발소리가 들렸다. 가볍고, 곧고, 흔들림 없는 발소리. 남궁선은 눈을 뜨지 않았다. 귀로만 들었다.

발소리가 남궁선의 방 앞에서 — 잠깐 멈추었다가 지나갔다.

```
[보행 패턴 일치: 남궁천. 신뢰도 91%.]
```

남궁선은 눈을 감은 채 웃었다.

너도 보러 왔구나.
