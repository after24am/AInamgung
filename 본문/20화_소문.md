# 20화. 소문

---

수련 두 달째.

연무장에서 남궁선의 위치가 바뀌어 있었다. 끝자리에서 중간으로. 조 사범이 자리를 옮겨준 게 아니었다. 자연스럽게 밀어 올라간 것이다.

기초검식의 속도가 동기 제자들을 넘어서기 시작했다. 리팩토링된 검식은 정통과 겉이 같았지만, 발동이 한 박자 빠르고 체력 소모가 적었다. 같은 시간 수련해도 남궁선만 덜 지쳤다.

제자들이 눈치채기 시작했다.

"남궁선이 빨라졌다."
"아직도 기초검식인데?"
"기초검식이 저렇게 빨라?"

조 사범도 보았다. 남궁선의 동작을 유심히 지켜보다가, 수련 후에 불렀다.

"남궁선."

"예."

"네 검이 두 달 전과 다르다."

남궁선은 표정을 관리했다. 서른여섯의 경험이 말했다. 여기서 설명하면 안 된다. 설명할수록 꼬리가 길어진다.

"매일 수련하니까 좀 나아진 것 같습니다."

"...그런가."

조 사범은 완전히 납득하지 못한 얼굴이었지만, 더 캐묻지는 않았다. 망나니가 수련을 한다는 것 자체가 기적인데, 실력이 느는 건 당연한 수순으로 보았다.

남궁선은 돌아서면서 속으로 안도했다. 문제는 — 이게 오래가지 않는다는 것이다.

```
[조언: 실력 은폐의 한계점에 도달하고 있습니다.]
[리팩토링된 검식의 발동 속도가 정통 검식의 물리적 한계를 넘기 시작했습니다.]
[교관급 관찰자가 본다면 — '이건 정통이 아니다'라고 판단할 수 있습니다.]
```

'알아. 남궁혁 사범 같은 사람이 보면 바로 들킬 거야.'

```
[남궁혁: 본가 무술 사범. 정통에 대한 고집이 강한 인물.]
[회귀 기억 기반 — 이 사람은 정통에서 벗어난 검을 "사파 무공"이라고 부릅니다.]
[신뢰도: 84%.]
```

남궁선은 걸으면서 계산했다. 지금까지는 창검당 안에서만 놀았다. 조 사범은 교관이지만 눈이 남궁혁만큼 날카롭지는 않다. 하지만 소문이 본가까지 올라가면 — 남궁혁이 직접 보러 올 수 있다.

그때까지 준비해야 한다.

뭘 준비하지?

남궁선의 잔머리가 돌았다.

'리팩토링된 검식을 감추는 건 한계가 있어. 그러면 — 감추지 말고, 설명할 수 있게 만들면 되잖아.'

```
[구체적으로요?]
```

'창궁무애검법.'

```
[......설명해주시겠습니까.]
```

남궁선은 정리했다. 남궁혁이 와서 "이 검식은 정통이 아니다"라고 따지면, 남궁선은 이렇게 답한다. "아버지에게 창궁무애검법을 배우고 있습니다."

창궁무애검법은 창검당의 비장 검결이다. 정통 남궁세가 계열. 사파가 아니다. 이것을 익히는 과정에서 기초검식의 동작이 자연스럽게 변했다고 하면 — 말이 된다.

물론 실제로는 태을의 리팩토링이지만, 겉으로는 상급 검법을 배우면서 기본기가 세련되어진 것으로 보인다.

```
[핑계를 미리 만들어두는 전략입니다.]
[사실이 아닌 것은 아닙니다. 실제로 창궁무애검법을 배우고 있으니까.]
[거짓말이 아니라 — 불완전한 진실.]
```

'세상에 완전한 진실이 어딨어.'

```
[철학적 발언이지만, 맥락상 자기합리화입니다.]
```

'됐어.'

하지만 이 계획이 성립하려면 — 창궁무애검법을 본가 사람 앞에서 보여줄 수 있을 만큼 익혀야 한다. 41%짜리로는 부족하다.

남궁선은 밤수련의 비중을 바꿨다. 기초검식 반복은 줄이고, 창궁무애검법 제1초식에 집중했다.

---

그날 저녁. 허 노인이 남궁선을 불렀다.

"남궁선. 잠깐 오너라."

총무실. 허 노인은 장부를 덮고 남궁선을 마주 보았다. 깐깐한 눈이 평소보다 진지했다.

"본가에서 연락이 왔다."

남궁선의 등이 곧아졌다.

"마교 잔당 건으로?"

"그것도 있고 — 다른 건도."

허 노인이 서신 한 통을 내밀었다. 남궁선은 받아들어 읽었다.

본가 관리 부서에서 보낸 공문이었다. 내용은 두 가지.

첫째, 마교 잔당 포박 건에 대한 치하. "창검당의 공을 치하하며, 위탁비를 이번 분기에 한하여 일할 증액한다."

남궁선은 속으로 환산했다. 일할이면 열에 하나. 적지만 — 있는 것과 없는 것은 다르다. 아까 본가에 빚을 만들어두자는 계산이 벌써 돌아왔다.

둘째가 문제였다.

"본가 정기 점검 일정을 통보한다. 익월 초, 남궁혁 사범이 창검당을 방문하여 위탁생 교육 상황을 점검할 예정."

남궁선의 눈이 멈췄다.

남궁혁. 본가 무술 사범.

한 달 뒤에 온다.

```
[알림: 남궁혁 방문까지 약 30일.]
[남궁혁은 정통 무공에 대한 고집이 강하며, 변형된 검식을 사파로 의심하는 성향.]
[대응 기한: 30일.]
```

남궁선은 서신을 접었다.

한 달. 그 안에 창궁무애검법을 보여줄 수 있을 정도로 익혀야 한다. 남궁혁 앞에서 "이건 창검당의 검법입니다"라고 당당하게 말할 수 있을 정도로.

못 하면 — "사파 무공"이라는 의심이 남궁선뿐 아니라 아버지에게까지 튄다.

```
[30일 안에 창궁무애검법 제1초식을 최소 70% 완성도로 올려야 합니다.]
[현재: 41%.]
[필요 향상: 29%p.]
[리팩토링 없이 순수 수련으로 달성하려면 — 빠듯합니다.]
[리팩토링 적용 시(0.3토큰): 20일 내 달성 가능.]
```

남궁선은 허 노인에게 고개를 숙이고 총무실을 나왔다.

마당에 서서 하늘을 올려다보았다. 달이 떠 있었다. 한 달 뒤에는 저 달이 다시 차 있을 것이다.

'태을아.'

```
[네.]
```

'리팩토링 해. 제1초식. 0.3.'

```
[결제 완료. 0.3토큰 소모. 잔여: 16.65.]
[창궁무애검법 제1초식 리팩토링 실행.]
```

남궁선은 서별채로 걸어가면서 손을 쥐었다 폈다.

30일. 남궁혁이 온다. 그때까지 검을 세워야 한다.
