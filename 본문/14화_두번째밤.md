# 14화. 두 번째 밤

---

이틀 뒤.

청심초가 4일분 남았다. 남궁선은 금수계곡행을 준비했다.

이번에는 달랐다. 처음 갔을 때는 약초만 생각했지만, 지금은 목표가 둘이었다. 청심초 채취와 정체불명 무인들의 정보 수집.

남궁선은 새벽에 일어나 준비를 했다. 밧줄, 주먹밥, 목검. 그리고 — 검은 천 한 장.

```
[검은 천의 용도를 물어봐도 됩니까?]
```

'얼굴 가리려고.'

```
[상대에게 들키면 정체를 숨기기 위해서입니까?]
```

'아니. 내가 들키면 안 되니까.'

```
[같은 말 아닙니까?]
```

'다르지. 들키는 거랑 정체가 탄로나는 건 별개야. 들키더라도 얼굴만 안 보이면 "남궁세가 누군지 모를 사람"이 된다고. 반대로 얼굴이 알려지면 창검당까지 추적당해.'

```
[......리스크 관리 능력이 술꾼답지 않습니다.]
```

'술꾼이니까 아는 거야. 외상 밀려서 도망 다녀본 놈이 추적 회피를 모르겠어?'

```
[경험의 출처가 처참합니다.]
```

남궁선은 서별채를 나섰다. 이번에는 낮이 아니라 밤이었다. 무인들이 활동하는 시간을 확인하려면 밤에 가야 했다. 낮에 돌아다니는 놈들이라면 주민 눈에 띄었을 것이다. 사냥꾼 막동이가 발자국만 봤다는 건 — 주로 밤에 움직인다는 뜻이다.

이것도 잔머리였다.

---

금수계곡. 달이 밝았다.

남궁선은 계곡 하류에서 멈췄다. 상류까지 가기 전에 먼저 살필 것이 있었다.

'태을아. 센서 범위가 어디까지야?'

```
[무료 센서 범위: 반경 50보.]
[유료 확장 스캔: 반경 200보. 비용 0.3토큰.]
```

'50보로 충분해. 여기서 천천히 올라가면서 탐지해줘.'

```
[센서 상시 가동. 무료입니다.]
[인물 감지 시 즉시 알려드리겠습니다.]
```

남궁선은 계곡을 따라 올라갔다. 이번에는 뛰지 않았다. 바위 그림자를 따라, 물소리에 발소리를 묻으며, 천천히.

서른여섯의 기억이 말했다. 숨는 법은 싸우는 법보다 먼저 배워야 한다. 도망 다닌 세월이 이런 데서 빛을 발했다. 외상 주인을 피해 골목을 누비던 기술이, 무인을 피해 계곡을 오르는 기술이 되었다.

한 시경을 올라갔다.

```
[경고. 전방 40보. 인물 1명 감지.]
[정지 상태. 앉아 있는 것으로 추정.]
```

남궁선은 즉시 바위 뒤에 몸을 숨겼다. 숨을 죽이고 고개를 내밀었다.

달빛 아래, 계곡 옆 넓은 바위 위에 사람 하나가 앉아 있었다. 검은 옷. 등을 보이고 있었다. 옆에 검이 놓여 있었다.

보초였다.

남궁선의 머리가 빠르게 돌아갔다.

보초가 있다는 건 지킬 것이 있다는 뜻이다. 그리고 보초가 한 명이라는 건 — 나머지는 안쪽에 있다는 뜻이다.

```
[센서: 대상의 호흡 패턴 — 규칙적. 졸고 있을 가능성 34%.]
[심박수: 안정. 경계 수준 낮음.]
```

경계 수준이 낮다. 남궁선은 그 정보를 씹었다. 경계가 낮다는 건 두 가지 뜻이다. 하나, 위협을 느끼지 않는다. 둘, 오래 있어서 느슨해졌다.

오래 있었다면 — 며칠이 아니라 더 길다.

남궁선은 보초를 우회했다. 계곡의 왼쪽 사면을 타고, 바위와 나무 사이를 기어서. 소리를 내지 않았다. 주루에서 주인 몰래 빠져나가던 기술의 확장판이었다.

보초를 지나 상류로 50보. 계곡이 꺾이는 지점에서 — 불빛이 보였다.

모닥불이었다. 바위 동굴 입구에 불이 피워져 있었고, 주변에 사람이 셋 있었다. 검은 옷. 무장. 그리고 동굴 안쪽에서 망치 소리가 들렸다.

```
[인원 감지: 외부 3명 + 내부 추정 2명 이상. 최소 5명.]
[보초 포함 총 6명 이상.]
[무장 상태: 전원 검 소지. 수준 — 창검당 교관급 이상으로 추정.]
```

남궁선은 바위 뒤에 엎드린 채 관찰했다. 이들의 대화를 들어야 했다.

물소리 때문에 희미했지만, 바람의 방향이 좋았다. 단어가 끊겨 들렸다.

"...아직 안 열려..."

"...주인께서 재촉하신다..."

"...이 밑에 분명히..."

남궁선의 눈이 좁아졌다. '이 밑에 분명히.' 뭔가를 파고 있다. 동굴 안에서 망치 소리가 나는 이유.

뭔가를 찾고 있다.

서른여섯의 기억을 더듬었다. 금수계곡 아래에 뭔가가 묻혀 있었나? 기억이 안 났다. 안개처럼 흐렸다.

```
[기억 조회 시도... 일치 항목 없음.]
[이것은 회귀 전에 일어나지 않았거나, 남궁선이 몰랐던 사건입니다.]
[새로운 변수입니다.]
```

남궁선은 더 다가가지 않았다. 6명 이상, 교관급 이상. 지금의 남궁선으로는 한 명도 상대가 안 된다. 여기서 할 수 있는 건 — 보고 듣는 것뿐이다.

하지만 그것으로 충분했다.

남궁선은 그들의 위치, 인원, 동굴의 위치, 보초의 배치를 머릿속에 새겼다. 그리고 천천히 후퇴했다. 왔던 길을 되짚어, 보초를 다시 우회하고, 하류까지 내려갔다.

돌아가는 길에 절벽에 들러 청심초를 더 땄다. 5일분. 이번에는 요령이 생겨서 반 시경이면 됐다.

---

창검당으로 돌아오는 길. 남궁선은 머릿속을 정리했다.

정체불명의 무인 최소 6명이 금수계곡 동굴에서 뭔가를 파고 있다. "주인"이라는 상위자가 있다. 남궁세가 영지에서 은밀하게 활동 중.

이것을 누구에게 말해야 하나.

아버지? 당주에게 보고하면 아버지가 직접 나설 것이다. 하지만 상대는 최소 6명, 교관급 이상이다. 아버지가 강하지만 혼자 가면 위험하다.

본가? 본가에 알리면 본가가 처리할 것이다. 하지만 방계가 본가에 도움을 요청하면 — 또 한 번 빚이 진다. 거절당한 전력도 있다.

관가? 무림의 일을 관가에 넘기면 체면이 깎인다. 그리고 관가가 움직이는 속도는 느리다.

남궁선은 서른여섯의 경험으로 선택지를 저울질했다.

'바로 알리면 안 돼.'

```
[이유를 말해주시겠습니까?]
```

'정보가 부족해. 누구인지, 뭘 찾는지, 얼마나 강한지. 지금 상태로 아버지한테 "수상한 놈들이 있어요" 하면 아버지가 뭘 할 수 있어? 6명 상대로 창검당 전력을 동원하면 — 이기더라도 피해가 크고, 지면 끝이야.'

```
[그래서요?]
```

'더 알아내야 해. 저들이 뭘 찾는지, "주인"이 누구인지. 그걸 알아야 대응을 정할 수 있어.'

```
[플레이어의 전투력으로는 정찰이 한계입니다.]
[발각되면 사망합니다.]
```

'알아. 그래서 싸우러 가는 게 아니라 보러 가는 거야.'

```
[......위험한 발상이지만, 논리적으로 반박할 수 없습니다.]
[단, 한 가지 확인합니다.]
[이 판단은 기억 조각 보상 때문입니까?]
```

남궁선은 잠시 걸음을 멈췄다.

기억 조각. 공짜 보상. 솔직히 말하면 — 조금은 그것도 있었다. 하지만 전부는 아니었다.

'아니.'

```
[그렇습니까.]
```

'창검당 코앞이야. 어머니가 자고 있는 데서 반나절 거리에 정체불명 무인이 6명이야. 이걸 무시하면 — 서른여섯 때 똑같은 꼴 당해.'

남궁선은 걸었다. 창검당의 불빛이 보이기 시작했다.

'이번에는 놓치지 않는다.'

```
[수련 적립: +0.15토큰 (이동 중 보법 훈련 효과). 잔여: 16.95.]
[사건 추적 상태: 진행 중.]
```

남궁선은 서별채로 돌아갔다. 청심초를 다시 천장에 매달고, 검은 천을 접어 넣고, 잠자리에 누웠다.

내일 아침, 어머니에게 감초물을 한 잔 더 끓여야 한다.
