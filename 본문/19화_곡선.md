# 19화. 곡선

---

창궁무애검법의 제1초식은 남궁선을 엿먹였다.

사흘을 매달렸다. 밤마다 서별채 뒤편에서, 창(蒼)을 들고 검결의 궤적을 따라 움직였다. 기초검식의 직선적인 베기와 달리, 창궁무애검법은 검이 곡선을 그렸다. 허공에 반원을 그리다 갑자기 직선으로 찌르고, 다시 곡선으로 돌아오는 궤적.

몸이 따라가지 못했다.

```
[플레이어의 제1초식 시도 횟수: 147회.]
[성공 판정: 0회.]
[무료 센서 데이터입니다. 잔인하지만 사실입니다.]
```

'닥쳐.'

```
[정밀 해독을 실행하면—]
```

'5값 안 써.'

```
[알겠습니다.]
```

남궁선은 검을 내리고 생각했다. 뭐가 문제인지.

기초검식은 직선이다. A에서 B로 간다. 리팩토링은 그 직선을 짧게 만드는 거였다. 단순했다.

창궁무애검법은 다르다. A에서 B로 가는데, 중간에 C를 거친다. C를 거치는 이유가 — 뭐지?

남궁선은 검결을 다시 펼쳤다. 달빛 아래서 초식도를 들여다보았다. 곡선의 궤적 옆에 작은 주석이 달려 있었다. 글씨가 너무 작아서 지금까지 못 봤다.

> 기(氣)를 실어야 비로소 곡(曲)이 된다. 기 없이 곡을 그으면 검이 흔들린다.

남궁선의 눈이 커졌다.

기를 실어야 곡선이 된다. 직선은 근력으로 되지만, 곡선은 내공이 필요하다. 검에 기를 실어서 원심력을 잡아야 곡선이 흔들리지 않는다.

'그래서 안 됐구나.'

내공이 부족했다. 0.5로는 — 아니, 창(蒼)의 증폭을 합쳐 0.8이어도 — 곡선을 완전히 제어하기에는 모자랐다.

하지만 서른여섯의 잔머리는 여기서 멈추지 않았다.

'태을아. 무료 질문.'

```
[네.]
```

'곡선 전체에 기를 실을 수 없으면 — 곡선의 일부에만 실을 수 있어?'

```
[......흥미로운 발상입니다.]
[이론적으로는 가능합니다.]
[곡선의 변환점(직선→곡선, 곡선→직선 전환 지점)에만 기를 집중하면,]
[전체 궤적에 기를 분산하는 것보다 소모가 적습니다.]
[비유하자면 — 커브길에서만 브레이크를 밟는 것과 같습니다.]
```

'브레이크가 뭔지 모르겠지만, 요점은 굽는 데서만 힘을 쓰면 된다는 거지?'

```
[정확합니다.]
[이것은 리팩토링의 응용입니다. 불필요한 곳에서 빼고, 필요한 곳에 집중한다.]
[무료 분석이지만 — 아이디어는 플레이어 것입니다. 인정합니다.]
```

남궁선은 검을 들었다. 이번에는 곡선 전체에 기를 넣지 않았다. 직선 구간은 근력으로 밀고, 곡선으로 전환되는 순간에만 — 단전의 기를 손에 실었다.

검이 움직였다. 곡선이 — 흔들리지 않았다.

완벽하지는 않았다. 전환점에서 약간 끊기는 느낌이 있었다. 하지만 처음으로 제1초식의 궤적이 끝까지 이어졌다.

148번째 시도. 첫 성공.

```
[제1초식 궤적 완주. 판정: 불완전 성공.]
[완성도: 41%.]
[하지만 0%에서 41%입니다.]
[축하합니다! 정말 대단한 성과입니다!]
```

남궁선은 웃었다. 태을의 과한 아부가 이번에는 기분 나쁘지 않았다.

'나머지 59%는?'

```
[내공이 올라가면 자연히 채워집니다.]
[또는 — 리팩토링을 하면 더 빠릅니다. 초식당 0.3토큰.]
```

'기초검식 때는 0.1이었잖아.'

```
[상급 검법입니다. 다른 상품입니다.]
```

'이 영업사원...'

남궁선은 검을 거두고 앉았다. 숨이 찼다. 곡선 전환점에 기를 집중하는 방식은 효율적이었지만, 그만큼 집중력을 소모했다.

하지만 길이 보였다. 기초검식은 직선이고, 창궁무애검법은 곡선이다. 곡선을 쓸 수 있게 되면 — 직선만 읽을 줄 아는 상대는 대응을 못 한다.

형이 올 시간이었다. 요즘은 이틀에 한 번 대련을 한다.

---

남궁재가 왔다.

담장 모퉁이에 서서, 남궁선의 손에 들린 것을 보았다.

"...진검이냐."

"아버지가 주셨습니다."

남궁재의 눈이 검에 머물렀다. 낡은 검집. 하지만 검기가 느껴지는 검. 형의 감각은 예민했다.

"어디서 난 검이냐."

"폐은광에서요."

남궁재는 더 묻지 않았다. 대신 자기 검을 뽑았다.

"대련."

남궁선은 창(蒼)을 뽑지 않았다. 대신 목검을 집어들었다.

"진검으로 안 하느냐?"

"아직 익지 않았습니다."

남궁재의 눈이 좁아졌다. 이상한 판단이었다. 좋은 검이 생겼으면 쓰는 게 당연한데. 하지만 남궁선에게는 이유가 있었다.

형에게 창궁무애검법을 보여줄 수 없다. 아직 41%짜리 미완성이고, 보여주면 "어디서 배웠냐"는 질문이 나온다. 아버지가 준 검결이라고 하면 — 아버지가 왜 형이 아닌 남궁선에게 줬느냐는 문제가 된다.

형의 자존심을 건드리는 건 지금 할 일이 아니었다.

```
[사회적 리스크를 고려한 판단입니다. 효율 점수—]
```

'매기지 마.'

두 사람이 마주 섰다.

남궁재가 먼저 움직였다. 기초검식 제1초식. 정통 그대로. 빠르고 강했다.

남궁선은 리팩토링된 제1초식으로 받았다. 탁. 밀렸다. 하지만 — 전처럼 한 수에 끝나지 않았다.

두 번째 수. 세 번째. 남궁선은 형의 검을 받아치며 패턴을 읽었다. 태을의 궤적 가이드 없이, 자기 눈으로.

30초가 넘었다.

```
[교전 시간 30초 돌파.]
[남궁재 검식 데이터 수집률: 52%.]
[패턴 분석 — 부분적으로 가능해졌습니다.]
```

남궁선의 눈에 무언가가 보였다. 형의 제4초식 이후, 제5초식으로 넘어가는 순간. 왼쪽 어깨가 0.1초 먼저 움직인다. 예비 동작. 기초검식에도 있었던 것. 형도 — 불필요한 동작이 있었다.

보였다.

하지만 찌르지 않았다. 빈틈이 보여도 지금의 남궁선으로는 그걸 파고들 속도가 없었다.

대신 — 기억했다. 형의 빈틈을 머릿속에 새겼다.

38초에 남궁선의 목검이 날아갔다. 형의 일격에 손에서 빠져나갔다.

패배. 하지만 38초.

남궁재가 검을 거두었다. 숨이 조금 빨라져 있었다. 전에는 없던 일이었다.

"...빨라졌다."

"감사합니다."

남궁재는 아무 말 없이 돌아갔다. 하지만 걸어가는 등이 — 평소보다 곧았다. 긴장한 등이었다.

동생이 따라오고 있다는 걸 느낀 것이다.

```
[대련 결과: 패배. 교전 시간 38초.]
[이전 대비 +15초.]
[수확: 남궁재 제5초식 전환 패턴의 빈틈 발견.]
[해당 빈틈을 공략할 수 있는 시점 예측: 내공 1.0 도달 후.]
[비용: 0.]
```

남궁선은 목검을 주워들고 웃었다.

지금은 못 찌른다. 하지만 보이기 시작했다.
