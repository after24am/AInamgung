# 12화. 흥정

---

창검당에 돌아온 건 해가 기울 무렵이었다.

남궁선은 서별채 뒤편으로 돌아 들어갔다. 정문으로 가면 눈에 띈다. 아침 수련을 빠진 건 조 사범에게 나중에 혼나면 그만이지만, 품에 약초를 안고 있는 꼴을 보여줄 필요는 없었다.

보자기를 풀었다. 청심초. 연한 푸른 잎이 아직 싱싱했다. 기운이 미세하게 느껴졌다.

```
[청심초 상태: 양호. 채취 후 6시간 경과. 24시간 내 가공하지 않으면 약효 감소 시작.]
```

가공. 문제는 이거였다.

```
[최적 가공법 안내: 0.5토큰.]
[건조 온도, 시간, 배합 비율까지 정밀하게 제시합니다.]
[가공 실패 시 약초가 날아갑니다. 투자 가치가 있습니다.]
```

남궁선은 청심초를 내려다보았다. 0.5. 반 시경 수련분. 목숨 걸고 절벽을 기어올라 따온 약초를 가공하는 데 또 값을 내라고.

'...잠깐. 힌트만 줘.'

```
[힌트요?]
```

'무료로 줄 수 있는 범위에서. 가공이 필요하다는 건 알겠어. 뭘로 가공해야 하는지 — 방향만.'

```
[무료 범위: 청심초는 열에 약합니다. 직접 달이면 안 됩니다.]
[이상입니다.]
```

'그것만?'

```
[무료니까요.]
```

남궁선은 생각했다. 서른여섯의 기억을 더듬었다. 약초. 열에 약한 약초의 가공법. 방계에 살면서 약에 대해 들은 적이 있었나?

있었다.

허 노인. 삼십 년 넘게 창검당의 살림을 꾸린 사람. 금고만 지킨 게 아니었다. 약재 구입, 물자 관리, 외부 약재상과의 거래까지 총무가 하는 일이었다. 허 노인은 약재에 대해 알고 있을 것이다.

하지만 직접 물어보면 — "이걸 어디서 구했느냐"는 질문이 돌아온다. 금수계곡에 혼자 갔다 왔다는 걸 알리고 싶지 않았다.

남궁선의 잔머리가 돌기 시작했다.

'태을아. 허 어르신의 총무실에 약재 관련 서적이 있어?'

```
[확인 불가. 직접 가보셔야 합니다.]
```

'그건 알아. 근데 허 어르신이 지금 총무실에 있어?'

```
[센서 데이터: 허 노인은 현재 본관 집무실에 있습니다. 총무실이 아닙니다.]
[무료입니다.]
```

남궁선은 웃었다.

총무실에 허 노인이 없다. 총무실에는 장부와 약재 서적이 있을 것이다. 들어가서 볼 수 있다. 가공법을 직접 찾으면 — 0.5값을 아낀다.

'가자.'

```
[......이것을 효율이라고 해야 할지 좀도둑이라고 해야 할지 모르겠습니다.]
```

'공부하겠다는데 좀도둑이야?'

```
[남의 방에 몰래 들어가서요?]
```

'총무실이지 허 어르신 침실이 아니잖아.'

```
[......기술적으로는 맞습니다.]
```

남궁선은 본관으로 갔다. 총무실은 본관 서쪽 끝에 있었다. 문이 닫혀 있었지만 잠기지는 않았다. 창검당은 본래 그런 곳이었다. 금고 사건 이후에도 문단속 습관이 안 들었다. 방계의 느슨한 보안이 이번에는 도움이 되었다.

안으로 들어갔다. 좁은 방에 장부가 쌓여 있었고, 벽 한쪽에 서가가 있었다. 약재 관련 서적을 찾았다. 《방계 약재 비망록》. 허 노인이 직접 정리한 것인지, 글씨가 깐깐하고 빼곡했다.

남궁선은 빠르게 넘겼다. 폐부 관련 약초. 청심초.

있었다.

> 청심초(淸心草): 폐부 염증 억제. 열에 약함. 직접 달이지 말 것.
> 가공법: 그늘에서 이틀간 자연 건조 후, 감초(甘草)와 배합하여 냉침(冷浸).
> 냉침 시간: 하룻밤. 물의 양: 약초 무게의 열 배.
> 주의: 금속 그릇 사용 금지. 도기(陶器) 사용할 것.

남궁선은 머릿속에 새겼다. 그늘 건조 이틀. 감초 배합. 냉침 하룻밤. 도기.

감초는 부엌에 있다. 어머니가 요리에 쓰는 걸 본 적이 있다. 도기 그릇도 부엌에. 전부 창검당 안에서 해결할 수 있다.

0.5값 — 절약 완료.

```
[인정합니다. 0.5토큰을 절약했습니다.]
[하지만 비망록의 배합 비율이 정밀 가공법과 동일한지는 보장할 수 없습니다.]
[정밀 가공법은 배합 비율을 소수점 단위로 최적화합니다.]
[비망록은 대략적입니다.]
```

'대략이면 얼마나 손해인데.'

```
[약효 기준: 정밀 가공 100이라면 비망록 기준은 약 75.]
```

남궁선은 생각했다. 75면 열에 일곱 반. 완치가 아니라 악화를 멈추는 게 목표니까 — 충분하다.

'된다. 75면 충분해.'

```
[구두쇠의 합리적 판단입니다.]
```

'칭찬으로 듣겠다.'

남궁선은 비망록을 제자리에 놓고 총무실을 나왔다. 나오면서 장부 더미 위에 먼지가 쌓인 것을 보았다. 허 노인이 바빠서 정리를 못 하고 있는 것이다. 조삼이 빠진 뒤 일손이 부족한 게 티가 났다.

남궁선은 잠시 멈칫했다. 그리고 먼지가 쌓인 장부 세 권을 정리하여 가지런히 쌓아놓았다.

작은 거다. 하지만 허 노인이 알아챌 거다. 누가 들어왔다는 건 모르더라도, 장부가 정리된 건 알 테니까. 까다로운 노인의 마음을 여는 건 한 번의 큰 은혜가 아니라 — 여러 번의 작은 성의다.

서른여섯 해를 잔머리로 버텨온 사내의 처세술이었다.

```
[흥미로운 행동입니다. 효율 분석 불가. 인간적 행동으로 분류합니다.]
```

'그게 제일 효율 좋은 거야.'

```
[데이터가 불충분하여 반론할 수 없습니다.]
```

---

서별채로 돌아온 남궁선은 청심초를 서까래 밑 그늘에 걸었다. 자연 건조 이틀. 그 사이에 감초를 준비하고, 도기 그릇을 구해놓으면 된다.

어머니에게 들키면 안 된다. "이게 뭐냐"고 물으면 설명이 복잡해진다. 영약이라고 하면 "어디서 이런 걸" 하며 걱정할 것이고, 걱정이 기침을 부를 것이다.

남궁선은 약초를 자기 방 천장 구석에 숨겼다. 어머니가 청소할 때 올려다보지 않는 자리. 서른여섯 해 동안 술을 숨겨왔던 바로 그 자리였다.

```
[과거에는 술을, 지금은 약을. 같은 자리, 다른 목적.]
[성장이라고 해도 될까요.]
```

'닥쳐.'

하지만 웃고 있었다.

---

저녁. 부엌에서 어머니가 저녁을 준비하고 있었다. 남궁선은 자연스럽게 들어가 찬장을 뒤졌다.

"뭘 찾아?"

"감초 있어요? 목이 좀 칼칼해서."

"거기 아래칸에 있을 거다."

남궁선은 감초를 꺼냈다. 필요한 양보다 조금 더. 어머니의 눈치를 살폈다. 유설화는 죽을 젓고 있었고, 남궁선이 감초를 얼마나 가져가는지 보지 않았다.

도기 그릇은 찬장 맨 위에서 가져왔다. 먼지가 쌓인, 쓰지 않는 도기 사발. 이것도 아무도 모르게.

남궁선은 재료를 방으로 가져갔다. 이틀 뒤면 준비가 끝난다.

```
[재료 확보 완료. 비용: 0.]
[총 절약액: 0.5토큰.]
[방법: 잠입(총무실) + 귀동냥(비망록) + 생활 밀착형 조달(부엌).]
[분류: 가성비의 정점.]
```

'가성비가 뭔데.'

```
[값 대비 성과입니다.]
```

'...그냥 알뜰하다고 하면 안 돼?'

```
[같은 말입니다.]
```

'아닌데.'
