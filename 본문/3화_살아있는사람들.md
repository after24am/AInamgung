# 3화. 빚쟁이

---

복도 끝에서 기침 소리가 들렸다.

남궁선의 발이 그쪽을 향했다. 금고 사건은 급하지만, 이건 더 급했다. 서른여섯 해의 기억이 끌어당기는 방향이었다.

부엌은 좁았다. 두 사람이 서면 어깨가 부딪히는 크기. 그을음 묻은 벽, 밑바닥이 검게 탄 솥, 반밖에 안 남은 찬장의 그릇들. 그 한가운데서 한 여인이 죽을 젓고 있었다.

유설화.

남궁선의 어머니였다.

```
[대상: 유설화. 기침 음파 분석 — 폐부 손상 중등도. 무료 센서 데이터입니다.]
```

남궁선은 대답하지 않았다. 눈앞의 등이 보였다. 작았다. 기침에 맞춰 흔들리는 작은 등. 서른여섯의 기억 속 어머니는 이것보다 심했다. 기침이 피가 되었고, 일어나지 못하게 되었고, 마지막에는 남궁선의 손을 잡고 "괜찮다"고 말하다가 눈을 감았다.

스물넷, 봄. 지금부터 5년 뒤.

유설화가 돌아보았다.

"선아? 해 뜨기 전에 일어나는 걸 본 적이 없는데."

남궁선의 목소리가 떨렸다. "...죽 먹어도 돼요?"

유설화의 눈이 동그래졌다. 열아홉 해 동안 죽을 다 먹은 적 없는 아들이, 해도 안 뜬 아침에 죽을 달라고 했다.

남궁선은 앉아서 죽을 받아들었다. 싱거웠다. 양념이 부족했다. 방계의 살림이 이 정도라는 뜻이었다.

한 숟갈도 남기지 않았다.

"다 먹은 거야?"

"맛있었어요."

유설화의 입가가 떨렸다. 웃으려는 건지 울려는 건지 모를 표정이었다. 남궁선은 그 표정을 보며 속으로 말했다.

100토큰. 반드시 모은다.

```
[감동적인 장면이지만, 허 노인 건은 시간이 지날수록 불리해집니다.]
[범인의 흔적은 시간이 지나면 사라집니다. 무료 조언입니다.]
```

'알았어. 간다.'

남궁선은 두 번째 그릇까지 비우고 자리에서 일어났다. 어머니가 "더 줄까?" 하는 소리를 등 뒤로 들으며 부엌을 나섰다.

---

허 노인의 처소는 방계 별채 동쪽 끝이었다.

가는 길에 태을이 입을 열었다.

```
[사과 전략을 수립하시겠습니까? 무료 컨설팅입니다.]
```

'공짜면 들어볼게.'

```
[분석 결과: 목소리를 평소보다 0.3옥타브 낮추고, 눈을 2초에 한 번 깜빡이면 '반성하는 인상'이 극대화됩니다.]
[추가로 양파를 지참하시면 눈물샘 자극 효과가 있습니다.]
```

'...양파?'

```
[효율적인 감정 연출입니다.]
```

'넌 사람한테 사과해 본 적이 없지?'

```
[저는 사과할 일을 만들지 않습니다.]
```

남궁선은 한숨을 내쉬었다. 이 기계 놈의 조언은 듣지 않는 게 나았다.

허 노인의 방 앞에 도착했다. 문이 닫혀 있었다. 안에서 인기척이 느껴졌다. 남궁선은 숨을 고르고 문을 두드렸다.

"허 어르신, 남궁선입니다."

침묵.

"어젯밤 일로 사죄드리러 왔습니다."

안에서 탕! 하고 무언가가 깨지는 소리가 났다. 찻잔을 내려친 소리 같았다.

```
[청각 센서: 대상의 호흡이 거칠어졌습니다. 혈압 상승 추정. 들어가시면 찻물을 맞을 확률 72%.]
```

"들어올 낯짝이 있느냐!!"

허 노인의 목소리가 문 너머로 쏟아졌다. 가늘지만 독이 잔뜩 오른 목소리였다. 남궁선은 기억을 더듬었다. 허 노인은 아버지의 오랜 벗이었다. 방계의 금고를 삼십 년 넘게 지킨 사람. 고지식하고, 깐깐하고, 하지만 방계에 대한 충심만큼은 의심할 여지가 없는 사람이었다.

적어도 — 서른여섯의 기억에서는 그랬다.

"어르신. 저 진짜로 금고에 손 안 댔습니다."

"뭐?!"

문이 벌컥 열렸다. 허 노인의 얼굴이 나타났다. 마르고, 주름 깊고, 눈에 핏줄이 선 얼굴. 하룻밤 사이에 십 년은 늙은 것 같았다.

"네놈이 안 했으면 누가 했단 말이냐! 네 주루 빚이 정확히 금고에서 사라진 양과 같아!"

남궁선의 눈이 좁아졌다.

주루 빚의 액수와 금고에서 사라진 양이 정확히 같다? 그건 오히려 이상한 거 아닌가. 진짜 범인이라면 그렇게 딱 맞추지 않는다.

```
[동의합니다. 범인이 플레이어에게 누명을 씌우기 위해 금액을 맞춘 것으로 추정됩니다.]
[금고 주변 환경 스캔을 실행하시겠습니까? 0.5토큰.]
```

'0.5면... 30분 수련분이잖아.'

```
[범인을 잡으면 기억 조각 1개가 무료 복구됩니다. 가성비를 생각하십시오.]
```

남궁선은 이를 악물었다. 이 기계 놈이 또 영업을 한다. 하지만 — 맞는 말이었다. 누명을 벗어야 아버지 앞에 설 수 있고, 범인을 잡으면 공짜 기억이 돌아온다.

'...해.'

```
[결제 완료. 0.5토큰 소모. 잔여: 11.4.]
[환경 스캔 실행 중...]
```

남궁선의 눈앞에 금고 방향으로 희미한 선이 그려졌다. 열흔적, 발자국 패턴, 먼지의 교란. 보통 눈으로는 볼 수 없는 것들이 푸른 윤곽으로 드러났다.

```
[스캔 완료.]
[금고 자물쇠에 플레이어의 지문 없음.]
[대신 미확인 인물의 접촉 흔적 감지. 방향: 동쪽 담장.]
```

남궁선은 허 노인을 바라보았다.

"어르신. 제가 범인을 잡아오겠습니다."

허 노인의 눈에 의아한 빛이 떠올랐다. 망나니가 범인을 잡겠다고? 하지만 남궁선의 눈은 — 허 노인이 알던 그 게으르고 흐리멍덩한 눈이 아니었다.

남궁선은 돌아서며 동쪽 담장을 향해 걸어갔다.

```
[조언: 담장을 넘은 흔적이 있습니다. 발자국 크기로 보아 성인 남성, 체중 약 70근.]
[방계 인원 중 해당 조건에 부합하는 자는 3명입니다.]
[추가 분석에 1.0토큰이 소모됩니다. 실행하시겠습니까?]
```

'1.0? 아까 0.5라며.'

```
[아까는 스캔이고 이건 분석입니다. 다른 상품입니다.]
```

'이 사기꾼 같은 놈...'

```
[영업입니다.]
```

남궁선은 담장 앞에 서서 흔적을 내려다보았다. 발자국이 담을 넘어 바깥으로 이어져 있었다. 방계 안의 누군가가 금고를 털고, 밖으로 빠져나간 것이다. 그리고 그 죄를 망나니 남궁선에게 뒤집어씌웠다.

'누구지.'

서른여섯의 기억을 더듬었다. 방계에서 돈과 관련된 사고가 있었던 적이 — 있었다. 분명히 있었다. 하지만 기억이 안개처럼 흐릿했다. 회귀의 부작용인지, 원래 술에 절어서 기억이 없는 건지.

```
[기억 노이즈가 심합니다. 정밀 기억 스캔을 권장하지만 5.0토큰입니다.]
[혹은 — 범인을 직접 잡으시면 무료로 기억 조각이 복구됩니다.]
```

남궁선은 웃었다. 씁쓸하면서도, 어딘가 의욕이 솟는 웃음이었다.

"값은 아끼고, 기억은 공짜로 먹고. 그게 내 방식이다."

```
[구두쇠의 자각이 있으시군요. 응원합니다.]
```

"닥쳐."

남궁선은 담장을 넘었다.
