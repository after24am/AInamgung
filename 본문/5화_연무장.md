# 5화. 연무장

---

조삼이 끌려간 뒤, 창검당은 조용했다.

아침 해가 마당을 비추고, 하인 하나가 사라진 것 외에 달라진 건 없었다. 원래 그랬다는 듯 아침이 흘렀다.

남궁선은 서별채 처마 밑에 앉아 있었다. 등 뒤로 어머니의 기침이 간간이 들렸다.

```
[센서 데이터: 유설화. 기침 빈도 전일 대비 동일. 폐부 손상 진행 중이나 급성 악화 없음.]
```

남궁선은 고개를 끄덕이지 않았다. 대신 주먹을 쥐었다. 스물넷 봄. 5년. 그 안에 100냥... 아니, 100값을 모아야 한다.

```
[정정: 토큰입니다.]
```

'값이라니까.'

```
[의견을 존중합니다. 동의는 하지 않습니다.]
```

남궁선은 일어섰다. 아까 사건으로 3값을 벌었다. 잔여 13.9. 하지만 수련을 해서 적립하지 않으면 어차피 한계가 있다.

서른여섯 해의 기억을 더듬었다. 무공. 나는 무공을 배웠었나?

배웠다. 남궁세가 기초검식. 하지만 진지하게 익힌 적은 한 번도 없었다. 선생이 가르치면 흘려듣고, 연무장에서 졸다가 매를 맞고, 도망치고, 술을 마시고. 그렇게 열일곱 해를 흘려보냈다.

그래서 서른여섯에 죽었다.

검 하나 제대로 못 쥐고.

```
[현 상태: 내공 — 없음. 검술 — 없음. 체력 — 일반인 하위. 솔직히 처참합니다.]
```

'공짜 독설은 필요 없거든.'

```
[사실입니다.]
```

남궁선은 한숨을 쉬고 걸었다. 연무장으로.

---

창검당 연무장은 본관 뒤편에 있었다.

마흔 보 사방의 넓은 마당. 바닥에 모래가 깔려 있고, 가장자리에 목검 걸이가 세워져 있었다. 아침 수련 시간이라 이미 사람들이 모여 있었다.

제자 열다섯. 그 중 열이 방계 아이들이고, 다섯이 본가에서 내려온 위탁생이었다.

남궁선이 연무장 입구에 서자, 시선이 쏟아졌다.

"...남궁선?"

"연무장에? 저 망나니가?"

"해 서쪽에서 뜨겠네."

속삭임이 아니었다. 대놓고였다. 남궁선의 망나니 이력은 창검당에서 전설급이었다. 세 번 연속 수련 땡땡이. 사범에게 "검보다 술이 맛있다"고 말해서 뺨 맞은 사건. 연무장 한가운데서 낮잠 자다 발에 밟힌 사건.

남궁선은 기억이 나지 않았다. 서른여섯의 기억에서 이런 흑역사는 삭제되어 있었다. 하지만 주변의 반응만 봐도 어지간히 해먹은 모양이었다.

```
[기록 말소된 데이터 영역이 비정상적으로 큽니다. 무의식적 자기 방어 삭제로 추정.]
```

'뭐라는 거야.'

```
[쉽게 말해, 흑역사가 너무 많아서 뇌가 알아서 지웠습니다.]
```

'...닥쳐.'

남궁선은 목검 걸이로 걸어갔다. 목검을 하나 집어들었다. 가벼웠다. 손에 굳은살이 없었다. 서른여섯의 기억 속에서 칼자루를 잡은 감각은 흐릿했고, 열아홉의 손은 부드러웠다.

아무것도 없는 상태.

연무장 한쪽에서 교관 한 명이 제자들의 자세를 교정하고 있었다. 오십 대 후반, 넓은 어깨, 성깔 있어 보이는 눈매. 창검당의 교관 조 사범이었다. 서른여섯의 기억에 희미하게 남아 있었다. 매를 자주 때리는 사람.

"남궁선."

조 사범의 목소리가 날아왔다.

"왜 왔느냐."

"수련하러 왔습니다."

연무장이 조용해졌다. 진짜 조용해졌다. 제자들이 검을 내린 채 이쪽을 쳐다보았다.

조 사범의 눈이 가늘어졌다.

"네가 수련을?"

"예."

"어젯밤 술은 안 마셨느냐."

"안 마셨습니다."

조 사범은 남궁선을 위아래로 훑어보았다. 남궁선도 그 시선을 받아냈다. 서른여섯 해를 산 눈으로.

잠시 침묵이 흘렀다.

"...끝자리에 서라."

남궁선은 말없이 제자들의 맨 뒤에 섰다. 열다섯 명 중 열여섯 번째. 가장 끝. 아무도 옆에 서려 하지 않았다.

```
[연무장 환경 스캔. 무료 센서 데이터입니다.]
[전방 15명의 자세를 실시간 수집 중.]
[교관의 교정 패턴 — 표본 3회 수집 시 최적화 가이드가 가능합니다.]
```

남궁선의 눈 한쪽에 희미한 푸른 선이 떠올랐다. 앞줄 제자들의 검을 잡은 각도, 발의 위치, 허리의 기울기가 윤곽선으로 표시되었다. 센서 데이터. 값이 안 드는 정보.

조 사범이 호령했다.

"기초검식 제1초식. 시작!"

열다섯 개의 목검이 일제히 올라갔다. 남궁선도 따라 올렸다.

```
[플레이어의 자세 분석.]
[팔꿈치 각도 — 18도 과다. 손목 회전 — 부족. 지지발 위치 — 2촌 벗어남.]
[총평: 이건 검식이 아니라 빨래 방망이질입니다.]
```

남궁선은 이를 악물었다. 하지만 반박할 수 없었다. 망막에 떠오른 자기 윤곽선과 앞줄 제자들의 윤곽선을 비교하면, 차이가 한눈에 보였다.

'교정해줘.'

```
[무료 교정 가능 범위: 자세의 대략적 방향만 제시.]
[정밀 리팩토링은 0.1토큰입니다. 실행하시겠습니까?]
```

남궁선은 잠시 망설였다. 0.1이면 반 시경 수련분이다. 첫날부터 쓸 건 아니다.

'무료로 해.'

```
[네. 팔꿈치를 내리십시오. 더. 더. 거기.]
[발을 안쪽으로. 아 너무 많이. 다시 밖으로. 거기.]
[......이 정도면 빨래 방망이에서 작대기 정도로 승격입니다.]
```

'칭찬인 거야 욕인 거야.'

```
[팩트입니다.]
```

남궁선은 조 사범의 시연을 보며 검을 휘둘렀다. 어색했다. 열아홉의 몸은 검을 들어본 적이 거의 없었고, 서른여섯의 기억은 검을 잡은 적이 거의 없었다. 이중으로 서투른 셈이었다.

하지만 이상한 점이 있었다. 태을의 무료 교정이 자세를 하나씩 잡아줄 때마다 — 몸이 기억하는 것 같았다. 회귀 전에 억지로 배웠던 검식의 근육 기억이 바닥에 남아 있는 것처럼. 서툴지만, 아예 처음은 아니었다.

```
[잔존 근육 기억 감지. 의외입니다.]
[플레이어가 과거에 수련을 완전히 안 한 건 아닌 모양입니다.]
[정확히는 — 대충 했습니다.]
```

'...그래 대충 했어.'

반 시경이 흘렀다.

조 사범이 정지를 명했다. 제자들이 검을 내렸다. 남궁선은 숨이 턱까지 차올라 있었다. 열다섯 명 중 가장 헐떡거리는 놈이었다.

```
[수련 적립: +0.1토큰. 잔여: 14.0.]
```

남궁선은 눈앞의 숫자를 보았다. 14.0. 반 시경을 죽도록 휘둘러서 겨우 0.1.

'100까지 얼마나 걸리는 거야.'

```
[현재 적립 속도 기준, 약 430일입니다.]
[물론 내공이 성장하면 적립 효율이 상승하므로 실제로는 더 짧을 수 있습니다.]
[혹은 사건 보상으로 단축할 수도 있습니다.]
[아까처럼.]
```

430일. 1년하고도 석 달. 5년 안에 충분히 가능하다. 하지만 — 그건 수련만 했을 때의 이야기다. 전투가 있으면 쓰게 되고, 기연을 찾으면 투자해야 하고.

'...일단 하자.'

남궁선은 목검을 다시 들었다. 다음 반 시경 수련이 시작되기 전에 혼자 한 번 더 휘둘렀다.

앞줄에서 본가 위탁생 하나가 돌아보았다. 열일곱쯤 되어 보이는 소년. 깔끔한 도복, 좋은 목검, 그리고 남궁선을 내려다보는 눈.

"방계 망나니가 검을 잡으니까 어색하네."

남궁선은 대답하지 않았다. 서른여섯의 경험이 말했다. 이런 놈한테는 반응하면 진다.

```
[동의합니다. 상대의 도발 의도 감지. 무시가 최적 대응입니다.]
[추가 정보: 대상의 기초검식은 교과서적이나 손목이 경직됨. 실전 경험 부족으로 추정.]
```

남궁선은 검을 내렸다.

지금은 이길 수 없다. 하지만 볼 수 있다. 태을의 센서가 잡아내는 상대의 자세, 빈틈, 습관. 전부 공짜 정보다.

이건 수련이면서 동시에 — 셈이다.

남궁선은 다시 검을 들었다.
