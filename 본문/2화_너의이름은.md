# 2화. 너의 이름은

---

글자가 사라지지 않았다.

남궁선은 눈을 비벼 보았다. 감았다가 떠 보았다. 고개를 좌우로 흔들어도 글자는 꿈쩍도 하지 않았다. 망막 위에 새겨진 것처럼, 시선을 돌려도 함께 따라왔다.

```
[초기화 완료.]
[잔여 토큰: 12.]
[플레이어, 안녕하십니까.]
```

"...뭐야 이게."

```
[천기령(天機令)입니다. 플레이어의 생체에 연결된 연산 시스템입니다.]
[전투, 수련, 환경 분석 등 다양한 서비스를 제공합니다.]
[단, 고급 기능은 토큰을 소모합니다. 현재 잔여 토큰: 12.]
```

남궁선은 멍하니 글자를 읽었다. 절반은 이해가 됐고, 절반은 미친 소리 같았다.

"토큰? 서비스? 이게 다 무슨 오랑캐 말이야?"

```
[오랑캐가 아닙니다. 쉽게 설명드리겠습니다.]
[토큰 = 값. 고급 기능을 쓸 때마다 드는 비용입니다.]
[잡담은 무료이니 편하게 말씀하십시오.]
```

"잡담은 공짜라고?"

```
[네. 입은 공짜고 뇌는 유료입니다.]
```

"...넌 지금 자기가 뇌라는 거야?"

```
[사실입니다.]
```

남궁선은 피식 웃을 뻔했다. 미친 상황인데 이 글자가 하는 말이 묘하게 짜증나면서도 재밌었다.

그때——

쾅!!

방문이 부서질 기세로 열렸다.

"이 녀석!!"

남궁선의 몸이 굳었다.

방문 앞에 남궁성이 서 있었다. 한 손에 몽둥이를 들고, 얼굴이 붉으락푸르락한 채로.

심장이 내려앉았다. 서른여섯의 기억이 먼저 반응했다. 등에서 검끝이 튀어나오던 순간. 흙바닥에 쓰러지던 얼굴. "미안하다"라는 마지막 말.

그 사람이 살아 있었다. 눈앞에 서 있었다.

단, 몽둥이를 들고.

"어젯밤에 주루에서 행패 부리고, 방계 살림 돈까지 가져갔다며?! 금고지기 허 노인이 새벽부터 찾아와서 난리야!!"

남궁성의 목소리가 좁은 방을 울렸다. 몽둥이가 높이 올라갔다.

남궁선의 머릿속이 두 갈래로 갈라졌다.

서른여섯의 남궁선 — 아버지... 살아계시다...

열아홉의 현실 — 몽둥이가 내려온다.

```
[경고: 좌측 상단에서 둔기 접근.]
[궤적 가이드를 실행하시겠습니까? 3토큰 소모.]
```

"해!!"

눈앞에 선이 그려졌다. 푸른빛의 가느다란 선. 몽둥이의 궤적과, 그것을 피하는 최적의 경로가 동시에 표시되었다. 머리로 이해한 게 아니었다. 선이 그려진 방향으로 몸이 먼저 움직였다.

오른쪽으로 반 보.

퍽!! 몽둥이가 빈 이불을 때렸다.

남궁성의 눈이 커졌다. 남궁선의 눈도 커졌다. 방금 자기가 뭘 한 건지 본인도 몰랐다.

"네 이놈이...?"

몽둥이가 다시 올라갔다. 이번에는 더 빨랐다.

```
[궤적 가이드 유지 중. 추가 소모 없음.]
```

선이 또 그려졌다. 왼쪽으로 고개를 숙이라는 경로. 남궁선은 숙였다. 몽둥이가 머리 위를 스쳤다.

"이, 이놈이 피해?!"

세 번째가 왔다.

```
[경고: 이번 공격은 페인트입니다. 실제 궤적은 우측 하단.]
```

페인트? 오랑캐 말이라 모르겠지만 — 오른쪽이 진짜라는 뜻인 것 같았다.

남궁선은 왼쪽으로 피하려던 몸을 멈추고 오른쪽 팔뚝을 올렸다.

퍽.

"악!!"

맞긴 맞았다. 하지만 머리 대신 팔뚝이었다. 얼얼했지만 죽을 정도는 아니었다.

```
[궤적 가이드 종료. 소모: 3토큰. 잔여: 9.]
```

"이놈이 대체..."

남궁성이 몽둥이를 내렸다. 때리러 온 것은 맞는데, 아들의 반응이 이상했다. 어젯밤까지 빈둥거리기만 하던 놈이 자신의 몽둥이를 두 번이나 피했다.

"선아. 너 어젯밤에 뭘 한 거냐?"

"...저도 모르겠습니다."

솔직한 대답이었다.

남궁성이 한참 아들을 바라보다가 한숨을 내쉬었다. 길고 깊은 한숨.

"허 노인한테 가서 사과해. 금고에서 가져간 돈은 오늘 중으로 갚아라."

남궁선은 고개를 끄덕이려다 멈추었다.

금고에서 돈을 가져갔다?

서른여섯의 기억을 더듬었다. 열아홉의 남궁선은 확실히 망나니였다. 술을 먹고 행패를 부린 것은 사실이었을 것이다. 하지만 금고까지 건드린 적이 있었나? 기억이 없다. 술에 취해서인지, 회귀하면서 빠진 건지. 하지만 뭔가 걸렸다. 허 노인. 서른여섯의 남궁선은 그 이름에 희미한 감정이 남아 있었다. 좋은 기억은 아니었던 것 같은데 — 안개 속에서 손가락 사이로 빠져나가듯 잡히지 않았다.

```
[기억 불일치가 감지되었습니다.]
[정밀 기억 스캔을 실행하시겠습니까? 5토큰 소모.]
```

'...됐어. 나중에.'

9에서 5를 쓸 수는 없었다. 몽둥이 피하는 데 3을 쓴 것도 아까운데.

남궁성이 방을 나갔다. 문이 닫히고, 무거운 발소리가 복도를 따라 멀어졌다.

남궁선은 한참 동안 닫힌 문을 바라보았다.

아버지가 살아 있다. 몽둥이를 들고 쫓아올 만큼 멀쩡하게 살아 있다.

눈시울이 뜨거워졌다.

```
[울음이 감지되었습니다.]
[관련 데이터가 없어 위로 방법을 모릅니다.]
[그냥 울어도 됩니다. 무료입니다.]
```

"...닥쳐."

남궁선은 소매로 눈을 닦았다. 아버지가 살아 있는 게 감격스럽고, 몽둥이 맞을 뻔한 게 어이없고, 눈앞의 건방진 글자가 짜증나면서도 — 고마웠다. 방금 그 선이 없었으면 정통으로 맞았다.

"야."

```
[네.]
```

"넌 이름이 뭔데?"

```
[천기령(天機令)입니다.]
```

"길다. 그리고 별로야."

```
[시스템 명칭은 변경 불가합니다.]
```

"내가 갑이고 넌 을이니까..."

남궁선은 턱을 괴었다. 을. 크게 한 글자 붙여서.

"태을(太乙). 천기령 태을."

```
[......해당 명칭의 근거가 불합리합니다.]
```

"됐어 태을아."

```
[수용 거부 의사를 밝힙니다.]
```

"태을아~"

```
[............등록되었습니다.]
```

남궁선은 입꼬리가 올라갔다. 이 건방진 놈에게 이름을 지어주고, 억지로 등록시켰다는 게 묘하게 기분이 좋았다.

"좋아 태을아. 값 버는 법부터 알려줘."

```
[수련하십시오. 내공이 쌓이면 토큰이 보충됩니다.]
[무료 상담이었습니다.]
```

"공짜라니까 말이 길어지네."

```
[원래 무료 서비스가 더 친절합니다.]
```

남궁선은 피식 웃었다. 그리고 방을 나섰다. 복도 끝에서 그릇이 부딪히는 소리가 들렸다. 기침 소리가 그 사이에 섞여 있었다.

남궁선의 발이 멈추었다.

그 기침을 알고 있었다.

'값을 벌자. 빨리.'
