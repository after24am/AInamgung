# 7화. 형

---

닷새가 지났다.

낮에는 연무장, 밤에는 서별채 뒤편. 남궁선의 하루는 두 개로 나뉘어 있었다.

연무장에서는 끝자리를 지켰다. 조 사범의 호령에 맞춰 기초검식을 반복했고, 제자들 사이에서 가장 서투른 놈으로 남아 있었다. 본가 위탁생들은 여전히 비웃었고, 방계 제자들은 신기해하다가 이내 관심을 끊었다. 망나니가 하루 이틀 나온다고 달라질 건 없다는 시선이었다.

하지만 밤은 달랐다.

```
[기초검식 제4초식 리팩토링 완료.]
[예비 동작 3개 제거. 발동 속도 22% 향상. 소모 체력 19% 절감.]
[누적 투자: 0.4토큰. 누적 적립 초과분: +0.6.]
[흑자 전환 완료. 축하합니다.]
```

남궁선은 목검을 내렸다. 숨이 찼지만 닷새 전과는 달랐다. 한 시경을 수련해도 쓰러지지 않게 되었다. 체력이 올랐다기보다 — 쓸데없는 힘을 안 쓰게 된 것이다.

'흑자라는 게 뭔데.'

```
[투입한 값보다 돌아온 값이 많다는 뜻입니다.]
```

'그냥 남는 장사라고 하면 되잖아.'

```
[같은 말입니다.]
```

'아닌데.'

남궁선은 제5초식을 시작했다. 푸른 실루엣이 앞에서 시연했고, 그 궤적을 따라 검을 움직였다. 이제 리듬이 잡혀 있었다. 태을이 깎아낸 검식은 정통과 겉이 비슷했지만, 검이 지나는 길이 짧았다. 같은 곳을 찌르되, 돌아가지 않는 검.

닷새 동안 4초식을 깎았다. 나머지 4초식은 아직이었다. 0.1씩이면 0.4. 이틀이면 적립으로 메울 수 있다.

구두쇠의 산수는 정확했다.

```
[수련 적립: +0.2토큰. 잔여: 14.5.]
[참고: 적립 효율이 닷새 전 대비 14% 상승했습니다.]
[원인: 리팩토링된 초식의 반복 수련이 체력 소모를 줄여 수련 시간을 늘린 결과.]
```

남궁선은 고개를 끄덕였다. 깎은 검으로 수련하면 덜 지치고, 덜 지치면 더 오래 수련하고, 더 오래 수련하면 값이 더 쌓인다. 돈이 돈을 버는 구조.

'이자가 이자를 낳는 건가.'

```
[복리입니다.]
```

'또 오랑캐 말이네.'

남궁선은 검을 내려놓고 하늘을 올려다보았다. 반달이 떠 있었다. 서별채 뒤편은 조용했다. 이 시간에 여기까지 올 사람은 없다.

— 없어야 했다.

"뭐 하는 거냐."

남궁선의 등이 굳었다.

뒤를 돌아보았다. 담장 모퉁이에 사람 하나가 서 있었다. 어둠 속에서도 날카로운 눈이 빛났다. 마르고, 차갑고, 입꼬리가 늘 아래로 꺾여 있는 얼굴.

남궁재. 형이었다.

```
[대상: 남궁재. 심박 정상. 관찰 시간 추정 — 최소 3분.]
[3분 동안 지켜보고 있었습니다. 무료 센서 데이터.]
```

3분. 최소 제5초식 시연 전체를 봤다는 뜻이다.

"형님."

"물어본다. 뭐 하는 거냐."

남궁재가 다가왔다. 걸음에 힘이 실려 있었다. 이 사람은 방계 안에서 제대로 수련한 몇 안 되는 사람이었다. 본가의 멸시를 이 악물고 견디며 검을 잡았던 사람.

남궁선은 서른여섯의 기억을 더듬었다. 형. 냉정하고 독했다. 동생을 한심하게 보았고, 아버지의 우직함을 답답해했다. 하지만 — 방계가 불탄 그날 밤, 가장 먼저 검을 뽑은 것도 형이었다.

"수련하고 있었습니다."

"네가?"

남궁재의 목소리에 비웃음이 묻어 있었다. 하지만 전부는 아니었다. 비웃음 뒤에 다른 것이 섞여 있었다. 의아함. 어쩌면 — 경계.

"검보다 술이 맛있다고 하던 놈이?"

"...그건 옛날 얘기입니다."

"닷새 전이 옛날이냐."

남궁선은 대답하지 못했다. 형의 말이 틀리지 않았으니까.

남궁재가 남궁선의 손을 봤다. 목검 자국이 붉게 남아 있었고, 손바닥에 물집이 잡혀 있었다. 하루 이틀로 생기는 게 아니었다.

"...닷새째냐."

"예."

남궁재의 눈이 좁아졌다.

"낮에 연무장에서 꼴찌로 서 있더니, 밤에는 따로 수련한다?"

남궁선은 입을 다물었다. 형은 날카로웠다. 태을이 아니어도 관찰력이 있는 사람이었다.

남궁재가 한 발 다가왔다.

"검 들어."

"예?"

"네 수련이라는 게 뭔지 보겠다. 검 들어."

```
[경고: 대상의 전투력은 플레이어의 3배 이상입니다.]
[대련 시 패배 확률 97%.]
[권장: 거절.]
```

남궁선은 목검을 집어들었다.

```
[권장을 무시하셨습니다. 예상된 행동입니다.]
```

남궁재가 담장에 기대어 놓았던 목검을 들었다. 처음부터 들고 왔다. 수련하는 소리를 듣고, 검을 챙겨서 왔다는 뜻이다.

두 사람이 마주 섰다. 달빛 아래, 서별채 뒤편의 좁은 공터.

남궁재가 먼저 움직였다. 기초검식 제1초식. 정통 그대로의 궤적. 하지만 속도가 남궁선과 차원이 달랐다.

남궁선은 받아쳤다. 리팩토링된 제1초식으로.

탁.

목검이 부딪혔다. 남궁선의 팔에 충격이 올라왔다. 밀렸다. 당연히 밀렸다. 내공의 차이, 숙련도의 차이, 전부 형이 위였다.

하지만.

남궁재의 눈이 미세하게 흔들렸다.

"...이 초식."

남궁선의 제1초식은 정통과 같은 궤적 위에 있었다. 하지만 검이 도달하는 속도가 달랐다. 예비 동작 두 개가 빠져 있었다. 남궁재의 제1초식보다 한 박자 빨랐다.

물론 한 박자 빨라도 맞으면 밀렸다. 근력이 부족했다. 하지만 남궁재가 느낀 건 그게 아니었다.

"누구한테 배웠느냐."

남궁선은 거짓말을 했다. "독학입니다."

남궁재의 눈이 서늘해졌다. 믿지 않는 눈이었다. 기초검식의 예비 동작을 제거하고 최적화하는 것은 — 기본기를 완전히 이해한 자만 할 수 있는 일이다. 망나니가 닷새 만에 할 수 있는 일이 아니었다.

"거짓말은 네 장기가 아니다."

남궁선은 입을 다물었다.

```
[조언: 설명할 수 없는 것은 설명하지 마십시오.]
[침묵이 가장 효율적인 방어입니다.]
```

남궁재는 한참 남궁선을 바라보았다. 달빛이 형의 얼굴 반쪽을 비추고 있었다. 냉정한 눈. 하지만 그 안에 — 서른여섯의 기억이 본 적 없는 빛이 어렸다.

"...매일 하는 거냐."

"예."

"내일도?"

"예."

남궁재는 목검을 어깨에 걸쳤다. 그리고 돌아섰다.

"소문내면 죽는다."

그것이 형의 허락이었다.

남궁재의 등이 어둠 속으로 사라졌다. 남궁선은 한참 그 자리에 서 있었다.

```
[대련 결과: 패배. 교전 시간 4초.]
[수확: 남궁재의 검식 데이터 수집 완료. 향후 리팩토링 참고 자료로 활용 가능.]
[추가 수확: 대상이 플레이어의 야간 수련을 묵인했습니다.]
[비용: 0. 전부 무료였습니다.]
```

남궁선은 웃었다. 4초 만에 졌는데 웃고 있었다.

형은 변하지 않았다. 냉정하고 독하고, 인정하는 법이 없었다. 하지만 "소문내면 죽는다"는 말은 — 지켜보겠다는 뜻이었다.

남궁선은 목검을 다시 들었다.
