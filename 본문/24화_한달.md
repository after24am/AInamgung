# 24화. 한 달

---

비무까지 한 달.

남궁선의 하루가 바뀌었다. 잠을 줄였다. 새벽에 일어나 내공 순환, 아침에 연무장, 낮에 체력 단련, 밤에 창궁무애검법. 쉬는 시간은 식사뿐이었다.

```
[수련 강도 경고: 근육 피로 누적 중. 3일에 한 번은 쉬어야 합니다.]
[무시하면 부상 확률이 올라갑니다.]
```

'무료 잔소리야?'

```
[네. 근데 진심입니다.]
```

남궁선은 사흘에 한 번 쉬었다. 쉬는 날에는 다른 일을 했다. 정보 수집.

본가 정기 비무의 규칙을 알아야 했다. 남궁선은 서른여섯의 기억에서 비무에 대한 정보를 찾았다. 흐릿했다. 비무를 본 적은 있지만 참가한 적은 없으니까. 술 마시면서 멀리서 구경했던 기억밖에 없었다.

```
[기억 신뢰도: 28%. 사실상 쓸모없습니다.]
[술에 취한 상태의 기억은 이 시스템도 복구가 어렵습니다.]
```

'...미안.'

```
[사과는 과거의 남궁선에게 하십시오.]
```

기억이 안 되면 물어야 한다. 남궁선은 허 노인을 찾아갔다. 장부 정리를 도와주면서 자연스럽게.

"어르신. 본가 정기 비무가 어떻게 진행되는지 아십니까?"

허 노인이 안경 너머로 남궁선을 보았다. 금고 사건 이후, 허 노인은 남궁선에게 차를 내주는 사이가 되었다. 장부 정리를 도와주는 것도 감점이 아니었다.

"토너먼트다. 16강부터 시작해서 결승까지. 본가 제자 위주지만, 이번에는 네가 들어가니 방계 출신이 하나 있는 거지."

"16명이면 — 4번 이기면 결승이네요."

"그렇지. 1회전부터 3회전까지는 하루에 치르고, 4회전과 결승은 다음 날이다."

남궁선은 계산했다. 하루에 세 경기. 체력 관리가 핵심이다. 초반에 토큰을 다 쓰면 후반에 태을이 먹통이 된다.

'태을아. 비무에서 토큰을 어떻게 배분해야 해?'

```
[비무 시뮬레이션.]
[전제: 4경기. 상대 난이도 점진적 상승.]
[1회전: 궤적 가이드 최소 — 0.5토큰. 관찰 + 기본기로 처리.]
[2회전: 궤적 가이드 표준 — 1.5토큰. 리팩토링 검식 활용.]
[3회전: 궤적 가이드 최대 — 3.0토큰. 전력.]
[4회전(준결승): 잔여 토큰 전부.]
[합계: 최소 5.0토큰 확보 필요.]
[현재 잔여: 18.2. 여유 있지만, 수련 중 소모를 감안하면 비무 당일 15.0 이상 유지를 권장.]
```

남궁선은 고개를 끄덕였다. 15.0 이상. 비무 전까지 토큰을 아껴야 한다. 리팩토링은 이미 필요한 만큼 했으니, 앞으로는 순수 수련으로 적립만 한다.

구두쇠의 셈이 시작되었다.

---

비무까지 20일.

창궁무애검법은 제3초식까지 진행되었다. 제1초식 완성도 81%, 제2초식 63%, 제3초식 42%.

12초식 전부를 한 달 안에 익히는 건 불가능했다. 남궁선은 포기할 줄 아는 사내였다. 전부를 낮은 완성도로 익히는 것보다, 3초식을 높은 완성도로 다듬는 편이 나았다.

```
[동의합니다. 3초식만으로도 조합하면 실전에서 충분한 변화를 만들 수 있습니다.]
[기초검식 8초식(리팩토링 완료) + 창궁무애검법 3초식 = 총 11가지 공격 패턴.]
[본가 비무 수준에서 11패턴이면 — 상대가 다 읽기 어렵습니다.]
```

남궁선은 11이라는 숫자를 머릿속에 넣었다. 11가지 패. 도박판에서 패가 많은 놈이 이긴다.

어머니의 상태도 확인했다. 청심초 복용 한 달.

```
[유설화 상태 보고. 무료.]
[폐부 염증: 복용 전 대비 61% 감소.]
[기맥 소통: 3곳 중 2곳 부분 개통.]
[기침 빈도: 하루 1~2회. 석 달 전의 1/5.]
[경과: 매우 양호.]
[비가역 단계 도달 예측: 기존 3년 10개월 → 5년 이상으로 연장.]
```

5년 이상. 원래 기억보다 더 늘었다. 남궁선은 처마 아래에서 어머니가 빨래를 너는 모습을 보았다. 기침 없이. 등이 예전보다 덜 흔들렸다.

'...됐다. 일단 시간은 벌었어.'

```
[100토큰은 여전히 필요합니다.]
```

'알아. 하지만 급하지는 않게 됐잖아.'

```
[맞습니다. 급하지 않은 것과 필요 없는 것은 다릅니다.]
```

'그건 알아.'

---

비무까지 10일.

형이 찾아왔다. 밤수련 때가 아니라, 한낮에.

남궁재는 서별채 마당에 서서 남궁선을 기다리고 있었다. 팔짱을 끼고, 무표정하게.

"형님?"

"앉아."

두 사람이 서별채 처마 아래에 나란히 앉았다. 형이 먼저 앉는 일은 거의 없었다.

남궁재가 품에서 종이를 꺼냈다. 접힌 종이 세 장.

"비무 대진표가 나왔다."

남궁선의 눈이 빛났다. "어떻게 구하셨습니까?"

"네가 알 필요 없다."

```
[남궁재의 정보 수집 경로 — 추정 불가. 이 사람은 자체 정보망이 있는 것 같습니다.]
```

남궁재가 종이를 펼쳤다. 16명의 이름이 적혀 있었다.

"네 1회전 상대는 본가 하급 제자 마경(馬京). 입문 2년차. 기초검식만 씀. 상대할 수 있다."

남궁선은 고개를 끄덕였다.

"2회전은 — 유광(兪光). 입문 4년차. 기초검식 상급. 힘이 강하지만 보법이 느려. 돌아다니면서 치면 된다."

남궁재의 설명은 간결하고 정확했다. 각 상대의 특징, 약점, 주의할 점을 세 문장 이내로 정리했다. 형은 이 사람들을 본 적이 있었다. 아니, 연구한 적이 있었다.

"3회전은 중급 제자 오천(吳天). 여기부터 강해진다. 내공이 너보다 높고 검식도 다양하다. 정면으로는 안 돼."

"어떻게 하면 됩니까?"

남궁재가 남궁선을 보았다. "오천은 왼손잡이인데 오른손으로 검을 쓴다. 교정했지만 급하면 왼손 습관이 나와. 왼쪽을 노리면 헷갈린다."

왼손잡이가 오른손 검을 쓸 때 생기는 미세한 불균형. 이런 건 직접 보지 않으면 모르는 정보다.

남궁선은 형을 바라보았다.

"형님. 이걸 어떻게 다..."

"닥치고 외워."

남궁재는 네 번째 상대 — 4회전, 즉 준결승 상대는 말하지 않았다. 대진표를 보면 알 수 있었다.

남궁천.

본가 최고의 천재. 4회전까지 올라오면 남궁천과 만난다.

남궁재도 그 이름 위에서 멈추었다.

"남궁천은 — 내 입으로 분석해줄 수준이 아니야."

"강합니까?"

"나보다 강해."

형이 인정한 상대. 남궁재가 자신보다 강하다고 말한 사람. 남궁선의 등에 찬 기운이 스쳤다.

"이기는 법은 — 나도 모른다."

남궁재가 일어섰다. 종이 세 장을 남궁선 앞에 놓고 돌아섰다.

걸어가다가 멈췄다.

"지는 꼴은 보기 싫다."

형의 등이 사라졌다.

남궁선은 종이를 들고 한참 앉아 있었다.

```
[남궁재 제공 정보를 데이터베이스에 등록합니다.]
[1회전 마경: 하급. 정면 돌파 가능.]
[2회전 유광: 중하급. 보법 약점. 기동전.]
[3회전 오천: 중급. 왼손 습관. 왼쪽 공략.]
[4회전 남궁천: 상급. 데이터 없음.]
[비용: 0. 형님이 전부 공짜로 주셨습니다.]
```

남궁선은 웃었다. 이 세상에서 가장 비싼 무료 정보였다.
