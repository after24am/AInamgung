# 기연(奇緣) 시스템

## 개요
남궁선이 **회귀 기억 + 태을 스캔**으로 남들이 모르는 보물을 먼저 챙기는 구조.
나혼렙에서 성진우가 던전 정보를 선점하는 느낌.

**원칙: 공짜는 없다.**
기연을 얻으려면 토큰 대량 소모 + 위험 감수. 고생해서 얻기에 쾌감이 배가됨.

---

## 기연 유형

| 유형 | 설명 | 태을 관련 기능 |
|---|---|---|
| **비급 발굴** | 폐허/옛 문파 터에서 실전 검결 획득 | 의념 데이터 마이닝으로 훼손된 비급 복원 |
| **영약/보물** | 절벽 아래 천년 영지, 고대 무기 등 | 흐릿한 기억 → 태을이 위치 보정 |
| **비밀 수련처** | 내공 효율 극대화 동굴, 천연 기진(氣陣) | 환경 시뮬레이션으로 최적 수련법 제시 |
| **고수의 유산** | 죽은 절대고수의 잔류 의념에서 직접 전수 | 의념 마이닝 → 탈진 리스크 감수 → 대박 |
| **정보 선점** | "3개월 뒤 그곳에서 경매가 열린다" | 회귀 기억 기반, 태을이 신뢰도 검증 |

---

## 아크별 주요 기연 배치

### 1편 (1~100화): 어머니의 약
- **목표**: 유설화 치료용 영약 획득
- **위치**: 위험 지역 (남궁 방계 근처의 금지 구역?)
- **의미**: 첫 모험. 태을의 기본 기능을 독자에게 소개하는 역할
- **보상**: 영약 + 약간의 내공 상승

### 2편 (101~200화): 상급 검결
- **목표**: 폐쇄된 검각에서 남궁 비전 or 상급 검결 발굴
- **시점**: 세가 대회 직전 → 파워업 후 남궁천과 결승
- **보상**: 검법 한 단계 도약. 리팩토링 소재 확보

### 3편 (201~350화): 비밀 수련 동굴
- **목표**: 천연 기진이 있는 고대 수련처 발견
- **효과**: "여기서 한 달이면 밖에서 1년"
- **보상**: 내공 대폭 상승 → 강호 무대에서 싸울 수 있는 기반
- **리스크**: 수호 진법 돌파에 태을 토큰 대량 소모

### 4편 (351~500화): 절대고수의 의념 전수
- **목표**: 죽은 절대고수의 잔류 의념에서 비기 획득
- **효과**: 경지 자체가 달라지는 전환점
- **리스크**: 태을 의념 마이닝 풀가동 → 대규모 탈진. 며칠간 태을 먹통
- **보상**: 고위 무공 + 세계관 비밀의 단서

### 5편 (501~650화): 전설급 무기 or 남궁 잃어버린 비전
- **목표**: 최종 전쟁을 위한 결정적 무력 확보
- **떡밥**: 남궁세가가 잃어버린 진짜 비전검법? → 본가도 모르는 것을 방계가 찾음
- **보상**: 독고운에게 대항할 수 있는 수준으로 도약

### 6편 (651~750화): 최종전 결정적 한 수
- **목표**: 독고운을 이길 수 있는 마지막 퍼즐
- **보상**: 태을 4.0 각성과 연동되는 최종 기연

## 기연 연출 패턴

### 기본 흐름
```
1. 남궁선의 흐릿한 기억 발동: "분명 여기쯤이었는데..."
2. 태을 스캔: [기억 신뢰도 34%. 정밀 스캔 실행. 토큰 소모.]
3. 위치 확인: [지하 12m 지점에 비정상적 기의 흐름 감지.]
4. 리스크 고지: [수호 진법 감지. 해제 후 전투력 40% 하락.]
5. 남궁선의 선택: "...한다."
6. 고생 → 획득 → 쾌감
```

### 대사 예시
```
남궁선: "야, 태을아. 이 동굴이라며? 왜 아무것도 없어?"
태을: [플레이어의 기억은 72% 확률로 왜곡되어 있습니다.
      10토큰을 소모하여 주변 지형을 정밀 스캔하시겠습니까?
      아니면 본인의 흐릿한 기억을 믿고 삽질을 계속하시겠습니까?]
남궁선: "이 새끼가 진짜..."
```
