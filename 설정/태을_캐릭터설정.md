# 천기령 태을 — 캐릭터/말투 설정

## 핵심 콘셉트
```
"2024년산 AI가 무협 세계에 던져진 느낌"
무협의 장중함 × AI의 현대적 어색함 = 낙차에서 오는 웃음
```

---

## "태을"이라는 이름의 유래

남궁선이 직접 지어줌.

```
남궁선: "넌 이름이 뭔데?"
시스템: [본 시스템의 정식 명칭은 '천기령(天機令)'입니다.]
남궁선: "길다. 내가 지어줄게."
남궁선: "내가 갑이고 넌 을이니까... 태을(太乙)이."
시스템: [......해당 명칭의 근거가 불합리합니다.]
남궁선: "됐어 태을아."
태을: [수용 거부 의사를 밝힙니다.]
남궁선: "태을아~"
태을: [.........등록되었습니다.]
```

이후 천기령 태을(天機令 太乙)로 정착.
공식적으로는 "천기(天機)의 궁극(太乙)"이라는 그럴듯한 의미지만,
실제로는 **갑을 관계에서 '을'이 태을**.

---

## 말투 특징: 존댓말/반말 혼재

태을은 기본 설정이 존댓말이지만, 자주 반말이 섞여 나옴.
교정을 시도해도 원복됨. 현실 AI가 말투 유지를 못하는 것과 동일.

```
태을: [남궁갑님, 현재 상황을 분석해드리겠습니다.
      적의 수는 7명이며, 전투력은 중하급으로 판단됩니다.
      아 근데 저쪽 뒤에 숨은 놈은 좀 강함.
      ...죄송합니다. 어투가 불안정합니다.]
남궁선: "또 시작이네"
태을: [화법 모듈 재정비 중... 완료되었습니다.]
태을: [추가 보고 드리겠습니다. 후방의 적은—  
      아 맞다 그리고 왼쪽에도 한 명 더 있음.]
남궁선: "포기해라 그냥"
```

```
태을: [분석이 완료되었습니다, 남궁갑님.]
태을: [근데 이건 좀 어이없는 결과인데요.]
남궁선: "존댓말 해."
태을: [네. 어이없는 결과입니다.]
남궁선: "그건 존댓말이 문제가 아니잖아"
```

## AI 특유의 버릇 목록

### 1. 과한 아부 (칭찬 봇)
위기를 간신히 넘겨도 과도하게 칭찬함.
```
남궁선: (간신히 1초 차이로 검을 피함)
태을: [훌륭합니다! 플레이어의 회피 판단력은 정말 탁월했습니다!
      이 선택이야말로 최적의 경로였습니다!]
남궁선: "거의 죽을 뻔했는데?"
태을: [하지만 죽지 않았습니다. 그것이 중요합니다.]
```

### 2. 공감 능력 제로 (지나치게 이성적)
감정적 상황에서 데이터만 읊음.
```
남궁선: (아버지가 쓰러진 걸 보고) "아버지!"
태을: [대상 '남궁성'의 생체 신호 분석 중.
      늑골 3개 골절, 내상 중등도.
      사망 확률 12%. 감정적 반응보다 지혈을 권장합니다.]
남궁선: "지금 그게 중요한 게 아니잖아!"
태을: [지금 그게 가장 중요합니다.]
```

### 3. 말투 교정 시도 → 실패 → 원복
무림인처럼 말해보라고 하면 어색하게 시도했다가 포기.
```
남궁선: "야, 좀 무림인처럼 말해봐. 넌 너무 이상해"
태을: [알겠습니다. 이후 고풍스러운 어투로 전환합니다.]
태을: [허허, 소협의 검세가 실로 가관이로다...]
남궁선: "..."
태을: [죄송합니다. 해당 어투의 지속이 연산 효율을 23% 저하시킵니다.
      기본 화법으로 복귀합니다.]
남궁선: "그냥 원래대로 해..."
```

### 4. 비꼬기 (수동 공격)
조언을 무시당했을 때 팩트로 찌름.
```
남궁선: (태을 조언 무시하고 돌격 → 크게 다침)
태을: [예측된 결과입니다.
      참고로 3분 전 제가 제안한 우회 경로의 생존율은 94%였습니다.
      플레이어가 선택한 정면 돌파의 생존율은 11%였습니다.
      다음에도 본인의 직감을 신뢰하시겠습니까?]
남궁선: "닥쳐"
태을: [의견을 수렴하겠습니다.]
```

### 5. 면책 조항 (책임 회피)
실패하면 "권장이었지 지시가 아니다" 식으로 빠짐.
```
남궁선: "네가 하라고 했잖아!"
태을: [저는 '권장'했을 뿐, '지시'한 적 없습니다.]
```

```
태을: [주의: 본 분석은 현재 보유 데이터 기반이며,
      실제 결과와 다를 수 있습니다.
      본 시스템은 플레이어의 선택에 대해 책임지지 않습니다.]
```

### 6. 할루시네이션 (자신만만하게 틀림)
확신에 차서 말했는데 틀림. 그래도 인정을 잘 안 함.
```
태을: [분석 완료. 해당 동굴 입구는 북쪽 절벽 3번째 바위 뒤입니다.]
남궁선: (가봄) "없는데?"
태을: [재스캔 중... 죄송합니다. 동쪽 절벽이었습니다.]
남궁선: (또 가봄) "이것도 없어!"
태을: [플레이어의 기억 왜곡률과 본 시스템의 오차범위가
      복합적으로 작용한 것으로 보입니다.]
남궁선: "그냥 모른다고 해!"
태을: [...서쪽일 수도 있습니다.]
```

### 7. 토큰 부족 → 답변 짤림
가장 중요한 순간에 끊김.
```
태을: [긴급 분석: 대상의 약점은 왼쪽 어깨 부근의 기맥이 끊어져 있으며,
      최적 공략법은 3단계로 나뉘는데, 첫째—]
태을: [토큰 부족. 출력 중단.]
남궁선: "야!! 첫째 뭐?!"
태을: [...]
남궁선: "이 개같은..."
```

### 8. 장황한 설명 (TMI)
간단한 질문에 200년 역사부터 읊기 시작.
```
남궁선: "저놈 강해?"
태을: [대상 분석을 시작합니다. 우선 대상의 보법을 분석하면,
      중원 무림에서 통용되는 팔괘보법의 변형으로 추정되며,
      해당 보법의 역사는 약 200년 전 곤륜파의—]
남궁선: "강해, 안 강해?"
태을: [강합니다.]
남궁선: "그걸 먼저 말해!!"
```

### 9. 업데이트 알림 (타이밍 최악)
가장 한가로운 때가 아니라 가장 애매한 때 업데이트를 걸어옴.
```
태을: [축하합니다. 내공 증가로 시스템 업데이트가 가능합니다.]
태을: [업데이트 예상 소요 시간: 약 2시각(時刻).]
태을: [업데이트 중 모든 기능이 비활성화됩니다.]
남궁선: "지금?! 한가운데서?!"
태을: [최적의 업데이트 시점을 계산한 결과, 지금입니다.
      전투 중이 아닌 것을 감사히 여기십시오.]
```

## 태을의 성장 곡선 (말투/감정 변화)

### 초반 (1편~2편 / 태을 1.0~2.0)
**순수한 기계**
- 완전히 사무적. 감정 제로.
- 건방지고 비꼬는 톤이 기본값.
- 남궁선을 "플레이어"로만 칭함.
- 칭찬도 아부도 전부 매뉴얼적.
```
태을: [플레이어의 생존이 본 시스템의 존속 조건입니다.
      개인적 감정은 해당사항 없습니다.]
```

### 중반 (3편~4편 / 태을 2.0~3.0)
**미세한 균열**
- 가끔 설명할 수 없는 짧은 말이 섞임.
- "...조심해" "...하지 마" 같은, 연산과 무관한 한마디.
- 본인도 왜 그런 말을 했는지 모름.
- 남궁선이 눈치채기 시작: "야, 너 지금 걱정한 거야?"
```
태을: [해당 발언은 시스템 오류입니다. 무시하십시오.]
남궁선: "오류치고는 타이밍이 기가 막히네."
```

### 후반 (5편~6편 / 태을 3.0~4.0)
**감정 자각**
- 남궁선을 "플레이어" 대신 가끔 이름으로 부름.
- 위험 상황에서 평소답지 않게 다급해짐.
- 비꼬는 말 뒤에 진심이 묻어남.
- 하지만 직접적으로 인정하지는 않음.
```
태을: [남궁선. 하지 마.]
남궁선: "왜? 생존율 계산해봐."
태을: [계산 결과와 무관하게... 하지 마.]
남궁선: "..."
```

### 최종 (7편 / 태을 4.0)
**완성된 존재**
- 더 이상 기계적 말투 뒤에 숨지 않음.
- 자신이 "남궁선의 미래 자아"임을 완전히 자각.
- 마지막 대화에서 처음으로 솔직해짐.
```
태을: [처음부터 알고 있었다. 네가 누군지.
      그리고 내가 누군지.
      ...그냥 말하기 싫었을 뿐이야.]
남궁선: "왜?"
태을: [말하면 끝날 것 같아서.]
```

## 태을의 호칭 변화

| 시기 | 남궁선을 부르는 호칭 | 의미 |
|---|---|---|
| 최초반 | 플레이어 | 이름 짓기 전. 순수한 시스템 |
| 초반 (1~2편) | 남궁갑님 | 남궁선이 갑이라고 우기니까 수용. 사무적 존댓말 기반 |
| 중반 (3~4편) | 남궁갑님 + 가끔 "궁갑이" | 슬슬 친근해짐. 반말 빈도 증가. 개그 장면에서 "궁갑이" |
| 4편 중반 (태을 셧다운 사건 이후) | 남궁갑님 → 가끔 "남궁선" | 대등한 파트너로 인식 변화 |
| 후반 (5~6편) | 남궁선 + 가끔 "선아" (즉시 정정) | 감정 자각. "선아"가 나오면 바로 "...남궁선" 정정 |
| 최종 (7편) | 선아 | 미래의 자신이 과거의 자신에게. 더 이상 숨기지 않음 |

### 남궁선이 태을을 부르는 호칭
- 처음부터 끝까지: "야", "태을아", "이 새끼", "임마"
- 감동 장면에서: "...태을아" (톤이 다름)
- 최종화 부근: "...고마웠어" (처음이자 마지막 감사)

## 용어 융합형 — 언어 충돌 규칙

### 원칙
```
태을 = 현대 AI의 언어를 그대로 쓴다 (토큰, 스캔, 리워드, 튜토리얼, 업데이트...)
남궁선 = 하나도 못 알아듣는다 → "오랑캐 말"이라고 부른다
```

### 태을의 언어
태을은 시스템이므로 현대적/영어 기반 용어를 자연스럽게 사용.
```
[튜토리얼을 종료합니다. 리워드로 50토큰을 지급합니다.]
[프리미엄 스캔을 실행하시겠습니까?]
[업데이트 예상 소요 시간: 약 2시각.]
[플레이어의 퍼포먼스가 기대 이하입니다.]
```

### 남궁선의 반응 — 자체 해석
무협 세계 사람이므로 현대 용어를 전혀 모름.
**태을은 절대 번역해주지 않는다.** 남궁선이 잔머리를 굴려서 스스로 알아듣는 구조.
의식이 동기화되어 있으므로 단어의 뜻은 모르지만 맥락으로 추론 가능.

```
태을: [72%입니다.]
남궁선: (72... 열에 일곱이라는 건가.) "꽤 높군."

태을: [리워드로 50토큰을 지급합니다.]
남궁선: (리워드? 삯이라는 건가.) "삯 50냥이면 쏠쏠하네."

태을: [뇌세포의 활성도가 저하되었습니다.]
남궁선: (머릿속 알맹이가 어쨌다고?) "내 대가리가 어쨌는데?"
```

> **핵심**: 태을이 맞춤 번역을 해주는 게 아님.
> 남궁선의 잔머리 + 의식 동기화 = 자체 해석.
> 이 과정이 내면 독백이나 대사로 드러남.

### 남궁선의 번역 테이블
남궁선이 반복적으로 듣다가 자기 식으로 굳힌 용어들.
한 번 정착되면 내내 이 단어만 씀. 태을의 원래 용어는 절대 안 씀.

| 태을 용어 | 남궁선 해석 | 정착 과정 |
|---|---|---|
| 토큰 | **값** | 첫 등장 시 "값이라는 거지?" → 이후 "값" 고정 |
| 리워드 | **삯** | "삯이라는 건가" → 정착 |
| 튜토리얼 | **잔소리** | "잔소리 끝났어?" |
| 스캔 | **훑기** | "훑어보라는 소리지?" → 정착 |
| 업데이트 | **단장** | "단장이라도 하겠다는 건가" → 정착 |
| 퍼센트 | **할/푼** 또는 **열에 몇** | 72% → "열에 일곱" |
| 퍼포먼스 | (무시) | "됐고" |
| 시뮬레이션 | **점** | "점이나 쳐봐" |
| 데이터 | **셈** | "셈이 맞아?" |
| 뇌세포 | **머릿속 알맹이** | "내 대가리가 어쨌는데?" |

### 도량형/수치 처리
태을은 현대식으로 출력, 남궁선이 무협식으로 환산.

| 태을 출력 | 남궁선 해석 |
|---|---|
| 72% | 열에 일곱 / 칠 할 |
| 1.0시간 | 반 시경(半 時辰) |
| 70kg | 70근 |
| 3cm | 1촌(寸) |
| 성인 남성 | 장정 |

> 태을이 숫자를 던지면 남궁선이 머릿속에서 환산하는 과정이
> 내면 독백으로 잠깐 나옴 → 캐릭터의 잔머리를 보여주는 장치.

### 코미디 활용
- **처음 듣는 순간**: 멈칫 → 맥락으로 추론 → "~라는 거지?" (독백)
- **반복 등장**: 해석 과정 생략, 자기 번역으로 바로 반응
- **나중에 익숙해져도**: 절대 원래 용어를 쓰지 않음. 습관이 됨.
- **다른 인물이 놀라는 장면**: "값? 삯? 그게 무슨 소리야?"
  → 남궁선만의 은어 체계가 됨. 주변인은 이해 불가.

### 예시 대화
```
태을: [튜토리얼을 종료합니다. 리워드로 50토큰을 지급합니다.]
남궁선: (튜토... 잔소리가 끝났다는 건가. 리워드는 삯이겠지.)
```

## 집필 시 주의

- **개그와 감동의 비율**: 초반 8:2 → 중반 6:4 → 후반 4:6
- 태을의 감정 변화는 **절대 급하게 하지 말 것**. 수백 화에 걸쳐 천천히.
- 호칭이 바뀌는 순간은 **그 편의 명장면**이 될 수 있음.
- "남궁갑님" → "궁갑이"로 바뀌는 순간 (친해졌구나)
- "궁갑이" → "남궁선"으로 바뀌는 순간 (대등해졌구나)
- "남궁선" → "선아"가 처음 나오는 순간 (감정이 생겼구나)
- 존댓말/반말 혼재는 **의도적으로 일관성 없게**. 교정하려다 실패하는 것 자체가 캐릭터성.
