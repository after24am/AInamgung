# 시스템: 천기령 태을(天機令 太乙)

## 개요
- 오직 남궁선의 **망막에만 보이는** 천기 연산 시스템
- 모든 기능은 주인공의 **내공을 연료(토큰)로 소모**
- 내공이 낮으면 기능이 강제 종료되거나 오류 발생
- 말투: 건방지고 사무적이지만 웃김

## 토큰 시스템

### 기본 원리
```
주인공의 내공 = 태을의 연료(토큰)
내공 수련 → 토큰 적립 → 태을 기능 확장
고급 기능 사용 → 토큰 소모 → 기능 제한
잡담/독설/잔소리 → 무료 (토큰 소모 없음)
```

### 무료 vs 유료 구분

| 무료 (토큰 소모 없음) | 유료 (토큰 소모) |
|---|---|
| 일상 대화, 독설, 잔소리, 비꼬기 | 적 분석, 약점 파악 |
| 상태 알림 (심박수, 기침 음파 등 센서 데이터) | 궤적 가이드 (실시간 회피/공격 경로) |
| 남궁선한테 시비걸기, 영업 멘트 | 비급 해독, 의념 데이터 마이닝 |
| 감정 감지, 울음 판정 등 | 정밀 기억 스캔 |
| | 환경 시뮬레이션, 진법 분석 |
| | 천기 연단술 등 특수 해금 기능 |

> **원칙**: 입은 공짜, 뇌는 유료.
> 태을이 자유롭게 말할 수 있어야 티키타카가 살고,
> 정작 중요한 순간에만 "값 나갑니다"가 뜨면서 긴장감이 생긴다.

### 초기 토큰
- 회귀 과정에서 발생한 에너지로 **12.0토큰** 충전 (회귀 특전 보너스)
- 이후 수련으로 적립: **초반 기준 반 시경(30분) 수련 = 0.1토큰**
- 내공이 올라갈수록 적립 속도 증가 (성장에 따라 체감 변화)

### 토큰 소모 기준 (초반 밸런스)

| 기능 | 소모량 | 비고 |
|---|---|---|
| 궤적 가이드 (단순 회피) | 0.1 | 몽둥이 피하기 수준 |
| 궤적 가이드 (실전 전투) | 1~3 | 상대 실력에 따라 변동 |
| 신체 스캔 (간단) | 0.5 | 기본 상태 확인 |
| 정밀 기억 스캔 | 5.0 | 회귀 전 기억 검색 |
| 비급 분석/해독 | 5~10 | 인생이 바뀌는 기능 |
| 영약 흡수 가이드 | 3~5 | 효율 극대화 |
| 천기 연단술 해금 | 100 | 어머니 완치용. 장기 목표 |

> 소모량은 태을 버전/상대 난이도에 따라 유동적.
> 핵심은 "가성비가 보여야 한다" — 독자가 "저건 쓸 만했다" 느끼는 구조.

### 토큰 효율 등급제
같은 기능이라도 투입 토큰에 따라 답변 품질이 달라짐.
- **최소 투입**: 쓸모없는 수준 ("적은 화가 나 있습니다")
- **적정 투입**: 유용한 정보 ("적은 좌측 다리가 약합니다")
- **풀 투입**: 필승 수준 ("적의 3초 후 움직임을 시각화합니다")

> 태을이 "프리미엄 분석이 가능합니다~"로 영업하는 근거.

### 내공 상태에 따른 태을 가동 수준
- **내공 충분**: 태을 잡담 + 고급 기능 자유롭게 사용 가능
- **내공 절반**: 잡담은 가능, 고급 기능 응답 느려지고 품질 저하
- **내공 바닥**: 잡담만 가능, 고급 기능 사용 불가 ("입만 살아있는 시스템")

### "값" — 남궁선의 토큰 인식
- 남궁선은 무협 세계 사람이라 "토큰"이라는 단어를 모름
- 태을이 설명하지만 남궁선은 그냥 **"값"**이라고 부름
  - "값 얼마야?" "값이 얼마나 남았어?" "값 아깝게"
- 방계의 경제적 궁핍함이 토큰 운용에도 스며듦
  - 내공이 넉넉해져도 습관적으로 아끼려 함
  - "부자라고 함부로 쓰면 거지 되는 거야"

### 구두쇠 × 영업사원 구도
- **태을**: 토큰을 쓰게 하려고 유도함 (잡담은 무료니까 영업 멘트를 공짜로 날림)
  - "스캔하시겠습니까?" "프리미엄 분석이 가능합니다" "영약일 수도 있는데"
  - "기본 분석은 1토큰, 프리미엄은 5토큰입니다. 추천은 프리미엄입니다"
  - 떡밥을 던져서 남궁선의 호기심을 자극하는 방식
- **남궁선**: 안 쓰려고 버팀
  - "기본으로 해" "안 써도 눈으로 보여" "공짜로 안 돼?"
  - 결국 필요한 순간에는 쓰지만, 항상 아까워함
- 이 구도는 800화 내내 반복되는 **러닝 개그**
- 가끔 안 써서 맞으면 → 태을: [아까 쓰셨으면 안 맞았습니다]
- 가끔 영업에 넘어가면 → 남궁선: "아 생돈..."

### 답변 절단 (코믹 장치)
토큰이 부족하면 중요한 말 도중 강제 종료.
```
태을: [상대의 초식은 남궁가문의 비기인... (치직)... 입니다.]
남궁선: "야! 뒤가 중요하잖아! 그게 뭔데!"
태을: [잔여 토큰 0. 내공을 0.1이라도 입금하십시오. 가불은 사절입니다.]
```

### 퀘스트 유도 (행동 강제)
토큰 보충을 위해 태을이 어처구니없는 행동을 요구.
```
태을: [토큰이 바닥입니다. 연무장의 비급 10권을 1분 내로 스캔하면 데이터 보너스가 지급됩니다.]
남궁선: "미친! 사범님이 눈 부라리고 있는데 거길 어떻게!"
```

### 태을의 정보 수집 방식
태을은 전지전능한 신이 아니라 **센서 기반 AI**.
- 청각: 반경 내 음파 수집 → 키워드 추출
- 시각: 남궁선의 망막을 통해 대상의 표정, 호흡, 근육 떨림 분석
- 진동: 발소리, 심장 박동 등 물리적 진동 감지
- **모르면 모른다고 한다.** "데이터 부족"이 정직한 답변.
- 단, 할루시네이션(자신만만하게 틀림)도 있음 → 개그용

### 기억 조각 보상 시스템 (유실 데이터 복구)
사건을 해결하면 태을이 보상으로 **유실된 회귀 기억 1조각을 무료 복구**해줌.
토큰(5.0)을 써서 기억을 사는 길 + 사건 해결로 공짜로 얻는 길, 두 루트.

- 구두쇠 남궁선은 당연히 **공짜 루트를 선호** → 사건 해결에 적극적으로 뛰어드는 동기
- 복구된 기억이 다음 사건의 단서가 됨 → **사건 해결 → 기억 획득 → 다음 사건 예방**의 순환 구조

#### 연출 방식
```
[경고: 데이터 충돌 발생. 억눌려 있던 플레이어의 기억 일부를 강제 해금합니다.]
```
→ 미래의 한 장면이 **1~2초짜리 짧은 플래시백**으로 뇌에 투사됨.
영화의 한 컷처럼 스치고 지나가는 수준. 맥락 없이 장면만 보임.

```
[복구 내용: '허 노인'은 회귀 전 마교 잔당과 내통한 기록이 있습니다.]
[신뢰도: 68%. 기억 왜곡 가능성 있음.]
```

> **봉인 기능 '기억 영사'와의 차이:**
> - 기억 조각 복구 = 1~2초 플래시백, 단편적 장면, 무료/저비용
> - 기억 영사(봉인) = 풀 영상 재생, 완전한 기억 체험, 뇌 터짐 위험
> 같은 "기억"이지만 스케일이 완전히 다름.

> **항상 유용하지는 않다.** 가끔 쓸모없는 기억이 풀림.
> 예: [복구 내용: 플레이어는 회귀 전 이 골목에서 넘어진 적이 있습니다.]
> 쓸모없는 기억 사이에서 진짜 중요한 기억이 나올 때 임팩트 극대화.

### 수련 로그 시각화
남궁선이 수련할 때, 태을이 **미세한 성장을 수치로 보여주는 로그**.
수련 장면이 "열심히 했다"로 끝나지 않고, 독자가 성장을 눈으로 확인하는 쾌감.

```
[알림: 창궁무애검법 제1초식의 불필요 예비 동작 0.02초 단축 성공.]
[로그: 보법의 지지점이 1촌(寸) 교정되었습니다. 발동 속도 5% 상승.]
```

> **매번 하면 숫자에 무뎌진다.** 평소 수련은 서사로 짧게 넘기고,
> **돌파 순간에만 로그가 쏟아지는 구조**로 운용.
> 10화 동안 묵묵히 수련 → 갑자기 로그 폭발 → 쾌감.
```
[축하합니다. 기초검법 제1초식 리팩토링 완료.]
[불필요 동작 3개 제거. 발동 속도 12% 상승.]
[토큰 보너스: +2.0]
```

### 태을의 효율 만능주의 (코믹 장치)
태을은 **효율만 따진다**. 인간적 정서/체면을 전혀 고려하지 않음.
결과적으로 남궁선에게 개망신당하는 행동을 권장 → 코미디.

```
남궁선: "허 노인한테 사과하러 가야 하는데, 진심이 전해지려면?"
태을: [분석 결과: 목소리를 평소보다 0.3옥타브 낮추고,
      눈을 2초에 한 번 깜빡이면 '반성하는 인상'이 극대화됩니다.]
남궁선: "...그걸 세고 있으라고?"
태을: [제가 세어드리겠습니다. 지금 깜빡이십시오.]
```

> 남궁선이 실제로 따라했다가 더 어색해져서 망하는 전개 가능.
> 태을이 "감정"을 이해 못 하기 때문에 생기는 괴리 = 800화 러닝 개그.
> 후반에 태을이 감정을 자각하기 시작하면, 이런 장면이 사라지는 것 자체가 성장의 증거.

## 기능 명세

### 1. 무학 리팩토링 (Martial Art Refactoring)
- **기능**: 기존 무공의 불필요한 초식/기의 낭비를 제거, 최적화된 경로 제시
- **효과**: 남들이 100의 공력으로 내는 위력을 60만 써서 낼 수 있음
- **리스크**: 동작이 정통 예법과 멀어져 "사파 무공 같다"는 오해를 살 수 있음

### 2. 실시간 궤적 가이드 (Real-time Trajectory)
- **기능**: 전투 시 적의 공격 궤적과 최적 대응 경로를 선으로 투사
- **효과**: 적의 빈틈(붉은 점)을 시각적으로 표시, 최소 움직임으로 치명상
- **제약**: 초절정 고수일수록 연산량 폭증 → [출력 지연] 발생

### 3. 신체 상태 모니터링 (Status Check)
- **기능**: 주인공/대상의 내공 잔량, 부상 부위, 혈맥 막힘 정도를 수치화
- **효과**: 겉으로 멀쩡한 상대의 "내공 10% 남음"을 파악 → 심리전 우위
- **특수**: 단전의 노폐물/독소를 정밀 탐지하여 세수간골 지원

### 4. 의념 데이터 마이닝 (Will Mining)
- **기능**: 오래된 비급/유물/죽은 고수의 잔류 의념에서 정보 추출
- **효과**: 훼손된 비급 복원, 고수가 남긴 비기의 요결 해독
- **소모**: 방대한 토큰을 한꺼번에 소모 → 일시적 탈진 상태

### 5. 환경 시뮬레이션 (Enviro-Sim)
- **기능**: 지형, 바람, 습도 등을 계산해 전투 환경 분석
- **효과**: 암기 투척 시 바람 저항 계산(백발백중), 진법 파훼 지점을 빛으로 표시

### 6. 인과 경로 예측 (Causal Path Prediction)
- **기능**: 선택지의 성공 확률과 예상 결과를 연산
- **효과**: 주인공이 기로에 섰을 때 확률을 던져줌 → 하이리스크 하이리턴 선택의 명분
- **핵심**: 태을 성장에 따라 예측 범위가 확장됨. **스케일 희석 금지.**

#### 단계별 제한

| 단계 | 할 수 있는 것 | 못 하는 것 |
|---|---|---|
| 1.0~2.0 | **단순 성공률 추정** ("이 싸움 이길 확률 30%") | 선택지 비교 불가, 보상 예측 불가 |
| 2.5~3.0 | **선택지 비교** ("A는 60%, B는 5%. B의 보상이 큼") | 보상 수치는 부정확, 파급효과 예측 불가 |
| 3.5 | **중기 인과 예측** (선택 → 결과 → 한 달 뒤 영향까지) | 장기 파급효과는 노이즈 |
| 4.0 | **인과율 간섭** (선택 → 결과 → 파급효과 → 천하 대세까지) | 풀스펙 |

#### 연출 예시 (초반 / 1.0)
```
태을: [전투 분석: 정면 돌파 시 생존 확률 30%.]
남궁선: "다른 방법은?"
태을: [데이터 부족. 확률 외의 정보는 제공할 수 없습니다.]
남궁선: "쓸모없는 놈..."
```

#### 연출 예시 (중반 / 2.5~3.0)
```
태을: [선택지 분석.]
태을: [A. 허 노인에게 사과한다. 성공 확률 60%. 보상: 토큰 1.0.]
태을: [B. 범인을 직접 잡아 족친다. 성공 확률 5%. 보상: 토큰 50.0 + 기억 해금.]
남궁선: "5%?"
태을: [권장은 A입니다.]
남궁선: "B."
태을: [......알겠습니다.]
```

> **주인공은 항상 하이리스크를 선택한다.** 확률 5%를 뚫어버리는 쾌감.
> 태을은 안전한 선택을 권장하고, 남궁선은 무시하고, 결과적으로 대박 → 구도 강화.

## 시스템 메시지 예시

```
[알림: 대상 '하북팽가 삼대제자'의 공격 패턴 분석 완료.]
[분석 결과: 대상 제압을 위한 최적 초식은 '남궁기초검식 3장 2절'의 변초입니다.]

## 봉인 기능 (과부하 위험)

시스템에 탑재되어 있으나, 현재 내공으로는 실행 시 경맥 파열/뇌사/즉사 위험.
현실 AI의 이미지 생성, 영상 생성, 음악 생성에 대응하는 고급 기능.

### 🔒 환영 투사 (Illusion Projection)
- **기능**: 적에게 가상의 환각을 씌우거나, 눈앞에 입체 영상을 투사
- **현실 대응**: 이미지/일러스트 생성
- **필요 내공**: 현재 보유량의 약 800%
- **실행 시 예상 결과**: 경맥 파열, 뇌사 확률 99.7%

### 🔒 기억 영사 (Memory Playback)
- **기능**: 회귀 전 기억을 영상처럼 생생하게 재생
- **현실 대응**: 영상 생성
- **필요 내공**: 현재 보유량의 약 1200%
- **실행 시 예상 결과**: 즉사

### 🔒 공명파 생성 (Resonance Wave)
- **기능**: 기의 파동으로 소리를 만들어 적의 정신을 교란
- **현실 대응**: 음악/음성 생성
- **필요 내공**: 측정 불가
- **실행 시 예상 결과**: [데이터 없음. 시도한 전례가 없습니다.]

### 봉인 기능의 위험도 변화 (태을 성장에 따라)

### 활용 방향

```

## 기원 미스터리 전개 규칙

### 태을의 정체 (최종 정답)
멸문 직전의 미래 남궁선이 모든 내공을 불살라 '천기(天機)'에 접속.
자신의 의식과 지식을 데이터화해 과거로 전송한 결과물 = 천기령 태을.
**이 사실은 7편(751~800화)에서 밝혀진다. 그 전까지는 절대 확정짓지 않는다.**

### 장치 1: 권한 부족 (정보 잠금)

남궁선이 태을의 정체를 물으면, 내공 부족을 이유로 답변 차단.
"강해져야 알려준다" → 수련 동기 + 미스터리 유지의 이중 역할.

```
남궁선: "넌 대체 어디서 온 건데?"
태을: [권한 부족. 기원 데이터 열람에는 최소 태을 2.0 이상이 필요합니다.]
남궁선: "중원말로 해."
태을: [더 강해지시면 알려드립니다.]
```

| 태을 단계 | 해금되는 정보 |
|---|---|
| 1.0~1.5 | 없음. "권한 부족" 일관 |
| 2.0 | "본 시스템은 특정 인물의 의지에 의해 생성되었습니다" (누구인지는 비공개) |
| 2.5 | 미래의 남궁선이 남긴 기억 조각 일부 해금 (단편적, 맥락 없는 장면들) |
| 3.0 | "생성자는 플레이어와 동일한 기맥 구조를 가지고 있습니다" (본인임을 암시) |
| 3.5 | 미래 남궁선의 목소리가 태을 안에서 간헐적으로 들림 |
| 4.0 | 전체 기원 데이터 열람 가능. 최종 공개 |

### 장치 2: 데이터 오염 (외부 간섭)

태을의 기존 **할루시네이션**(자체 연산 오류, 개그용)과 구분되는 별개 현상.
외부 세력(마교 등)이 시스템에 간섭하여 정보가 왜곡되거나 노이즈가 발생.

- **할루시네이션** = 태을이 자신만만하게 틀림 → **개그**
- **데이터 오염** = 정보가 깨지거나 뒤틀림 → **긴장감**

독자가 "이번에 태을이 틀린 건 원래 허당이라 그런 건가, 마교가 건드린 건가?"를 구분 못 하는 것 자체가 서스펜스.

```
태을: [경고: 외부 연산 간섭 감지.]
태을: [데이터 무결성 78%... 61%... 하락 중.]
태을: [현재 출력 정보의 신뢰도를 보장할 수 없습니다.]
남궁선: "누가 건드리는 건데?"
태을: [분석 불가. 다만 간섭 패턴이 본 시스템과 유사합니다.]
```

> 마교 측에도 천기에 접근하려는 세력이 있다는 복선으로 활용 가능.

### 장치 3: 꿈속 메타 힌트 (의념 잔여)

의식적으로 '기억 영사'를 실행하면 뇌사 위험이 있지만,
무의식(꿈) 상태에서 미래 남궁선의 의념 잔여물이 스며드는 것은
시스템 기능이 아니라 **영혼의 공명** → 토큰 소모 없음.

- 꿈에서는 태을이 아니라 **"누군가"**가 말하는 것처럼 처리
- 형체 없는 푸른 빛, 또는 등만 보이는 남자의 실루엣
- 독자는 "태을인가? 다른 존재인가?" 헷갈림 → 700화대에서 "미래의 남궁선"이었음을 깨달음

```
[꿈속]
푸른 빛. 형체 없는 목소리.
"...이번엔 어머니를 먼저."
남궁선: "누구야?"
"시간이 없다. 스물넷 봄 전에... 약을..."
[깨어남]
```

### 전개 원칙
1. **절대 서두르지 않는다.** 100화마다 힌트 하나씩 떨어뜨리는 속도감.
2. **확정 짓지 않는다.** 독자가 "미래의 자신인 것 같은데..." 추측하게 만들되, 작중에서 확언하지 않음.
3. **세 장치를 섞는다.** 권한 부족(시스템적 차단) + 데이터 오염(외부 위협) + 꿈(영혼적 연결)이 번갈아 등장하면서 단조로움 방지.
